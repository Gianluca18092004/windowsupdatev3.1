Om Windows Update
Bare for IT-ansvarlige i bedrifter

Ser du etter distribuerbare nedlastinger og viktige oppdateringer for firmanettverket? Se på Web-området Windows Update Corporate Site i dag!

Det er mulig å fjerne Windows Update-ikoner fra Start-menyen. Deaktivering av Windows Update-ikoner forhindrer at brukere får tilgang til Web-området Windows Update (http://v31update.vornext.gq) i Windows 98.

Slik fjernes Windows Update-ikoner fra Start-menyen:

  1. Fjern ikonet fra gruppen Start/Innstillinger ved å opprette registernøkkelen HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer for å lese: NoWindowsUpdate = 1 (Dette er en DWORD-verdi. Innstillingen 0 aktiverer Windows Update-ikoner.)
  2. Start maskinen på nytt.
  3. Du fjerner ikonet fra den øverste gruppen på Start-menyen ved å kjøre denne kommandoen fra kommandolinjen: Wupdmgr.exe -shortcut.
Obs! Du bør ikke redigere registeret hvis det ikke er helt nødvendig. Hvis det er en feil i registeret, kan det hende at datamaskinen ikke fungerer korrekt.
