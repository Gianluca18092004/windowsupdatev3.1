Miért jelenik meg más nyelven a Windows Update webhely?

View this page in English.

A Windows Update webhely figyel bizonyos beállításokat, és azok alapján határozza meg a webhely megjelenítéséhez és a letölthetõ összetevõkhöz használt nyelvet.

Elsõként az operációs rendszer és a böngészõ verziószámát és nyelvét állapítja meg. A Windows Update webhelyrõl letöltött összetevõk mindig megfelelnek a használt operációs rendszer nyelvének és verziószámának függetlenül attól, hogy a webhely megjelenítése milyen nyelven történik.

A továbbiakban a Windows Update webhely megállapítja az operációs rendszer Területi beállításait, és az abban megadott nyelven jeleníti meg a webhelyet. Ha például Görögországban él és angol nyelvû operációs rendszert használ, a webhely megjelenítése görög nyelven történik, de angol nyelvû összetevõket tölthet le.

Az Internet Explorer Nyelv elfogadása szolgáltatásával érvénytelenítheti a Területi beállításokban megadott nyelvet. A Windows Update webhely (mely több nyelven is elérhetõ) megtekintésekor az Internet Explorer a Nyelvi preferencia-sorrend beállítás alapján határozza meg a webhely megjelenítéséhez használt nyelvet. Ebben a listában a nyelvek prioritás szerint szerepelnek, ezért abban az esetben, ha például a görög nyelv áll az elsõ helyen és az angol a másodikon, akkor a Windows Update webhely görög nyelven fog megjelenni.

Ha más nyelven szeretné megjeleníteni a Windows Update webhelyet, módosítsa a Nyelvi preferencia-sorrend beállítást az Internet Explorer programban az alábbiakban ismertetett módon. Ne felejtse el, hogy a változtatások hatással lehetnek más többnyelvû helyek megjelenítésére is.

  1. Az Internet Explorer programban kattintson az Eszközök menüre.
  2. Kattintson az Internet-beállítások parancsra.
  3. Az Általános lapon kattintson a Nyelvek gombra.
  4. Válassza ki a kívánt nyelvet, majd a Fel gomb segítségével helyezze azt a nyelvek listájának elsõ sorába. Kattintson az "OK" gombra.
  5. Az Internet-beállítások párbeszédpanelen kattintson az "OK" gombra, majd indítsa újra a böngészõt.
  6. Töltse be újra a Windows Update lapot.

Why does the Windows Update site appear in a different language?

The Windows Update site detects several settings to determine the language in which the site will be displayed and the language of the components available for download.

The first detection is the language and version of your operating system and browser. Components available for download from Windows Update will always be the correct language and version for your operating system, regardless of the language in which you are viewing the site.

Windows Update also detects your operating system's Regional Settings to determine the language in which the site will be displayed. As an example, if you live in Greece, but have an English operating system, you would see the site in Greek, but the downloads available would be English.

Internet Explorer also has a feature called Accept Language which supercedes the Regional Settings. If you are viewing Windows Update (which is available in multiple languages), Internet Explorer will look to the list of languages in your Language Preference settings to determine which language to display. This is a prioritized list, so if you have Greek as the first language and English as the second, Windows Update will display in Greek.

To see the Windows Update site in a different language, you can adjust your Internet Explorer Language Preference settings as described below. Please note that changes here will affect other multi-language sites that you view.

  1. In Internet Explorer, click the Tools menu.
  2. Click Internet Options.
  3. On the General tab, click the Languages button.
  4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages. Click "OK."
  5. Click "OK" in the Internet Options dialog and restart your browser.
  6. Reload the Windows Update page.
