Om Windows Update
Kun til de IT-ansvarlige

Søger du efter distribuerbare overførsler og vigtige opdateringer til det fælles netværk? Besøg webstedet Windows Update Corporate i dag!

Du kan fjerne de Windows Update-ikoner, der vises i menuen Start. Hvis du deaktiverer Windows Update-ikonerne, kan brugere ikke få adgang til Windows Update-webstedet (http://v31update.vornext.gq) fra Windows 98-produktet.

Du kan fjerne Windows Update-ikonerne fra menuen Start ved at gøre følgende:

  1. Fjern ikonet fra gruppen Start/Indstillinger ved at ændre registreringsnøglen HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer til NoWindowsUpdate = 1 (Dette er en DWORD-værdi. Hvis den angives til 0, aktiveres Windows Update-ikoner).
  2. Genstart computeren.
  3. Du kan fjerne ikonet fra den øverste gruppe i menuen Start ved at køre følgende kommandolinje: Wupdmgr.exe -shortcut.
Bemærk! Du bør ikke redigere registreringsdatabasen, medmindre det er absolut nødvendigt. Hvis der er en fejl i registreringsdatabasen, fungerer computeren måske ikke korrekt.
