Siden kan ikke vises
Prøv igen

Du kan være på denne side af én af følgende grunde:
  1. Opdateringer til styringen af og/eller arkitekturen til Windows Update-webstedet kan have forårsaget denne fejlmeddelelse. Yderligere oplysninger finder du i Microsoft Knowledge Base-artiklen Q243787.
  2. Du har valgt ikke at hente softwarekontrolelementer, eller der opstod et problem under overførsel kontrolelementerne, der forårsagede, at Windows Update-webstedet ikke er tilgængeligt. Hvis du vil hente kontrolelementerne nu, skal du klikke ovenfor på Prøv igen.
  3. Du har ikke administratorrettigheder til denne computer. Hvis du vil bruge Windows Update, skal du kontakte systemadministratoren for at opdatere softwaren.
  4. Du skal måske tømme browserens cachehukommelse. Yderligere oplysninger finder du i Microsoft Knowledge Base-artiklen Q193385.
  5. Du skal måske fjerne Windows Update-mappen. Yderligere oplysninger finder du i Microsoft Knowledge Base-artiklen Q243787.
  6. Opdateringer til Windows NT® 4.0 Terminal Server Edition er ikke tilgængelige på webstedet for Windows Update.Yderligere oplysninger finder du i artikel Q323564 i Microsoft Knowledge Base (KB).
