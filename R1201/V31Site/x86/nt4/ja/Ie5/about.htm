Windows Update について
 Windows Update は、Windows のオンライン拡張機能です。現在、Windows Update は、製品の更新カタログとサポート情報の 2 つのエリアから構成されています。これらのエリアを表示するには、画面の左側にあるナビゲーション バーを使用します。

[製品の更新] について
[製品の更新] は、Windows および Windows と共に動作する多くのプログラムの修正プログラム、アップデート、および追加コンポーネントのカタログになっています。このカタログを使用して、コンピュータを使用しやすくするコンポーネントを入手できます。

カタログを見て、好きなだけコンポーネントを選択し、そのコンポーネントを Web からコンピュータに直接ダウンロードおよびインストールすることができます。まだ Web からダウンロードしたことがない場合は、下の「どのような操作が必要ですか?」をお読みください。このカタログには、コンピュータにインストールされているソフトウェアを調べ、まだインストールされていないアップデートや追加コンポーネントを判断する機能があり、必要に応じてこの機能を利用することができます。

どのようなしくみになっていますか?
初めて [製品の更新] ページを訪問すると、Windows Update によってシステムに既にインストールされているコンポーネントがスキャンされ、インストールした方がよいコンポーネントの一覧が表示されます。システムを調査する際、お客様のシステム情報の安全とプライバシーは確保されています。この情報が、弊社またはインターネットに送信されることはありません。 システムを調べることによって、このサイトからダウンロードするコンポーネントの最新の正確なバージョンを確認することができます。さらに、既にインストールされているコンポーネントを、誤ってもう一度ダウンロードするのを避けることができます。

コンポーネントの一覧がページに表示されたら、その一覧を見て、コンピュータにダウンロードするコンポーネントを決めることができます。
お使いのシステムにダウンロードすることをお勧めしている重要な更新はすべてダウンロードしてください。重要なアップデートによって、セキュリティ問題などの既知の問題が解決されます。

どのような操作が必要ですか?
[製品の更新] を使用すると、コンピュータのソフトウェアに対応するアップデートや追加コンポーネントを簡単にダウンロードすることができます。
  1. まず、カタログを見ます。
  2. ダウンロードするコンポーネントを決め、コンポーネントの左のボックスをクリックします。ボックスにチェック マークが表示されます。コンポーネントをダウンロードする前に、"最初にお読みください" というリンクをクリックします。次のような重要な情報が含まれているページが表示されます。
    • コンポーネントの詳細な説明
    • コンポーネントの起動方法
    • アンインストールに関する説明
    • 製品サポート情報
  3. 選択した製品を取り消す場合は、ボックスをもう一度クリックして、チェック マークを消します。
  4. 希望のコンポーネントをすべて選択した後に [ダウンロード] ボタンをクリックします。
  5. ページが表示されます。一覧にダウンロードするコンポーネントがすべて表示されているかどうかを確認します。変更する場合は、[戻る] をクリックします。
  6. [ダウンロード開始] をクリックします。選択したすべてのコンポーネントがダウンロードされ、インストールされます。インストール中に、追加の情報を入力する必要があるコンポーネントもあります。
インストールが完了すると、"インストールが完了しました" というメッセージが表示されます。コンポーネントをすべてインストールした後に、コンピュータをシャットダウンし、再起動する必要があるコンポーネントもあります。

コンポーネントは [製品の更新] から簡単にアンインストールできます。以下の方法でアンインストールします。

  1. [製品の更新] ページに移動します。ダイアログ ボックスが表示されたら [はい] をクリックします。
  2. [製品の更新] ページが表示されたら、ページ最上部の [すべて表示] ボタンをクリックします。
  3. アンインストールしたいコンポーネントの右にある [アンインストール] ボタンをクリックします。

[サポート情報] について
Windows Update の情報センターです。Windows Update のサポート情報があります。よく寄せられる質問や確認されている問題を読んだり、弊社のサポート センターに問い合わせることができます。
