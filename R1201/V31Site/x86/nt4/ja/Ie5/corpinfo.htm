Windows Update について
企業の IT 管理者向け

企業ネットワーク上で配布できるダウンロードおよび重要な更新をお探しですか? Windows Update の企業用サイト を参照してください。

[スタート] メニューに表示される Windows Update アイコンを削除できます。Windows Update アイコンを無効にすることによって、Windows 98 製品から Windows Update サイト (http://windowsupdate.microsoft.com) を利用できないようにします。

[スタート] メニューから Windows Update アイコンを削除するには、

  1. HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer にレジストリ キー NoWindowsUpdate = 1 を作成して、[スタート] メニューの [設定] グループからアイコンを削除します。 (レジストリ キーの値は DWORD です。値に 0 を設定すると Windows Update アイコンが表示されます)
  2. コンピュータを再起動します。
  3. [スタート] メニューの一番上のグループからアイコンを削除するには、コマンド ラインから次のコマンドを実行してください。 Wupdmgr.exe -shortcut
注意:必要のない限り、レジストリを編集しないようにしてください。 レジストリにエラーがあると、コンピュータが正しく機能しなくなる可能性があります。
