Tente novamente

Você chegou a esta página por um dos seguintes motivos:
  1. As atualizações dos controles e/ou arquitetura do site Windows Update podem ter causado a exibição dessa mensagem de erro. Para obter mais informações, leia o artigo Q243787 do Microsoft Knowledge Base (KB).
  2. Você optou por não fazer o download dos controles de software ou ocorreu um problema durante o download dos controles. Nesse caso, uma grande parte do site Windows Update não estará disponível para você. Se desejar fazer o download dos controles agora, clique no link Tente novamente, acima.
  3. Você não tem privilégios administrativos para este computador. Para usar o Windows Update, contate o administrador do sistema para atualizar o software.
  4. Talvez você não tenha limpado o cache do navegador. Para obter mais informações, leia o artigo Q193385 do Microsoft KB.
  5. Talvez você não tenha removido o diretório do Windows Update. Para obter mais informações, leia o artigo a href="http://support.microsoft.com/support/kb/articles/q243/7/87.asp" target="_new">Q243787 do Microsoft KB.
  6. Atualizações para o Windows NT® 4.0 Terminal Server Edition não estão disponíveis no site Windows Update. Para obter mais informações, leia o artigo da Microsoft Knowledge Base (KB) Q323564.
