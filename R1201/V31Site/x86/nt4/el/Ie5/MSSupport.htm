Πληροφορίες υποστήριξης
 Εάν αντιμετωπίζετε δυσκολίες με την τοποθεσία του Windows Update στο Web, υπάρχει διαθέσιμη ηλεκτρονική υποστήριξη, για να σας βοηθήσει. Απλώς κάντε κλικ σε μία από τις επιλογές που παρατίθενται παρακάτω.

Όταν κάνετε κλικ σε ορισμένες από αυτές τις επιλογές υποστήριξης, ίσως να σας ζητηθεί να δημιουργήσετε ένα "Ηλεκτρονικό προφίλ" της Microsoft. Το ηλεκτρονικό προφίλ θα αποτελεί το εισιτήριό σας για τη λήψη υποστήριξης μέσω Internet από τη Microsoft.


Υποστήριξη για ενημερωμένες εκδόσεις λογισμικού: Προβάλετε τις πληροφορίες υποστήριξης για τις ενημερωμένες εκδόσεις λογισμικού που παρέχει το Windows Update.

Υποστήριξη για προγράμματα οδήγησης συσκευών: Προβάλετε τις πληροφορίες υποστήριξης για τα προγράμματα οδήγησης συσκευών που προσφέρει το Windows Update.

Συνήθεις ερωτήσεις: Εάν έχετε κάποια απορία σχετικά με το Windows Update, μπορείτε πρώτα να ελέγξετε τη λίστα με τις απαντήσεις μας.

Γνωστά θέματα: Μπορείτε επίσης να διαβάσετε τη λίστα των γνωστών θεμάτων που υπάρχουν αυτήν τη στιγμή στην τοποθεσία του Windows Update.

Επικοινωνία: Η φόρμα γρήγορης έρευνας παρέχει χώρο για τα σχόλια και τις προτάσεις σας.

Τρόπος κατάργησης εγκατάστασης των ενημερωμένων εκδόσεων λογισμικού
Η κατάργηση της εγκατάστασης ορισμένων ενημερωμένων εκδόσεων λογισμικού μπορεί να γίνει με χρήση της σελίδας "Ενημερωμένες εκδόσεις προϊόντων" της τοποθεσίας του Windows Update:
  1. Μεταβείτε στη σελίδα "Ενημερωμένες εκδόσεις προϊόντων".
  2. Εάν εμφανιστεί ένα μήνυμα, κάντε κλικ στο κουμπί Ναι.
  3. Όταν εμφανιστεί η σελίδα "Ενημερωμένες εκδόσεις προϊόντων", κάντε κλικ στο κουμπί Εμφάνιση εγκατεστημένων ενημερώσεων που βρίσκεται στο επάνω μέρος της σελίδας.
  4. Εάν είναι δυνατή η κατάργηση της εγκατάστασης μιας ενημερωμένης έκδοσης, εμφανίζεται το κουμπί Κατάργηση εγκατάστασης δεξιά από την περιγραφή της ενημερωμένης έκδοσης.
  5. Κάντε κλικ στο κουμπί Κατάργηση εγκατάστασης που βρίσκεται δίπλα στην περιγραφή της ενημερωμένης έκδοσης, της οποίας την εγκατάσταση θέλετε να καταργήσετε.
Η κατάργηση της εγκατάστασης των ενημερωμένων εκδόσεων, η οποία δεν είναι δυνατό να γίνει από τη σελίδα "Ενημερωμένες εκδόσεις προϊόντων", μπορεί να γίνει με τη χρήση άλλων μεθόδων. Ανατρέξτε στη σελίδα Βασικές πληροφορίες κάθε ενημερωμένης έκδοσης για επιπλέον πληροφορίες σχετικά με την κατάργηση της εγκατάστασης.

ΥΠΟΣΤΗΡΙΞΗ ΠΡΟΓΡΑΜΜΑΤΩΝ ΟΔΗΓΗΣΗΣ ΣΥΣΚΕΥΩΝ
Η δωρεάν υποστήριξη εγκατάστασης/κατάργησης της εγκατάστασης είναι διαθέσιμη μέσω ηλεκτρονικού ταχυδρομείου στην τοποθεσία Υποστήριξη τοποθεσίας του Windows Update. Για θέματα εκτός της εγκατάστασης/κατάργησης εγκατάστασης προγραμμάτων οδήγησης, μπορείτε να βρείτε μια πλήρη λίστα επιλογών υποστήριξης στη διεύθυνση http://support.microsoft.com/directory. Σε ορισμένες περιπτώσεις ενδέχεται να ισχύει χρέωση υποστήριξης. Επισκεφτείτε την τοποθεσία για λεπτομέρειες.

Για πληροφορίες σχετικά με θέματα υποστήριξης, επικοινωνήστε με την τοπική θυγατρική. Για την πιο πρόσφατη λίστα, επισκεφτείτε την τοποθεσία Ηλεκτρονική υποστήριξη στο Web και ενεργοποιήστε την επιλογή που αφορά την υποστήριξη σε όλο τον κόσμο. Εάν δεν υπάρχει γραφείο θυγατρικής εταιρείας της Microsoft στη χώρα σας, επικοινωνήστε με το κατάστημα από το οποίο προμηθευτήκατε το προϊόν της Microsoft.

Οι υπηρεσίες υποστήριξης της Microsoft υπόκεινται στις εκάστοτε τιμές, όρους και προϋποθέσεις της Microsoft, και μπορεί να αλλάξουν χωρίς προειδοποίηση.

Τρόπος κατάργησης εγκατάστασης των ενημερωμένων εκδόσεων προγραμμάτων οδήγησης συσκευών
Σημαντικό Για να καταργήσετε την εγκατάσταση μιας ενημερωμένης έκδοσης προγράμματος οδήγησης, της οποίας κάνατε λήψη από το Windows Update, πρέπει να έχετε τη δυνατότητα να εγκαταστήσετε ξανά το αρχικό πρόγραμμα οδήγησης που συνόδευε τη συσκευή σας. Η κατάργηση της εγκατάστασης ενός προγράμματος οδήγησης ή μιας ενημερωμένης έκδοσης του προγράμματος οδήγησης χωρίς την αντικατάστασή του θα έχει ως αποτέλεσμα τη διακοπή της λειτουργίας της συσκευής υλικού.

Χρησιμοποιήστε τη Διαχείριση Συσκευών, για να καταργήσετε ενημερωμένες εκδόσεις προγραμμάτων οδήγησης από τον υπολογιστή σας:

  1. Κάντε κλικ με το δεξιό κουμπί του ποντικιού στο εικονίδιο Ο Υπολογιστής μου στην επιφάνεια εργασίας του υπολογιστή σας.
  2. Κάντε κλικ στην εντολή Ιδιότητες και στη συνέχεια κάντε κλικ στην καρτέλα Διαχείριση Συσκευών.
  3. Κάντε κλικ μία φορά για να επιλέξετε τη συσκευή υλικού και στη συνέχεια κάντε κλικ στο κουμπί Ιδιότητες.
  4. Κάντε κλικ στην καρτέλα Πρόγραμμα οδήγησης και στη συνέχεια κάντε κλικ στο κουμπί Ενημέρωση προγράμματος οδήγησης.
  5. Όταν ξεκινήσει ο "Οδηγός αναβάθμισης προγράμματος οδήγησης", κάντε κλικ στο κουμπί επιλογής Καθορισμός της θέσης του (για προχωρημένους) και στη συνέχεια κάντε κλικ στο κουμπί Επόμενο.
  6. Επιλέξτε το κουμπί επιλογής Εμφάνιση λίστας με τα γνωστά προγράμματα οδήγησης για αυτήν τη συσκευή, ώστε να επιλεγεί ένα συγκεκριμένο πρόγραμμα οδήγησης και στη συνέχεια κάντε κλικ στο κουμπί Επόμενο.
  7. Κάντε κλικ στο κουμπί Από δισκέτα και στη συνέχεια κάντε κλικ στο κουμπί Αναζήτηση, για να εντοπίσετε το αρχείο .inf για το πρόγραμμα οδήγησης που θέλετε να εγκαταστήσετε (με αυτόν τον τρόπο θα καταργηθεί το πρόγραμμα οδήγησης του Windows Update και θα αντικατασταθεί με το αρχικό πρόγραμμα οδήγησης της συσκευής υλικού).
Σημείωση Όταν επιστρέψετε στο Windows Update, είναι πιθανό να σας προσφερθεί ξανά η ίδια ενημερωμένη έκδοση του προγράμματος οδήγησης. Μην την εγκαταστήσετε. Εάν έχετε ερωτήσεις ή προβλήματα κατά την εκτέλεση της διαδικασίας για την κατάργηση της εγκατάστασης μιας ενημερωμένης έκδοσης προγράμματος οδήγησης του Windows Update, χρησιμοποιήστε τις συνδέσεις υποστήριξης που παρέχονται για να επικοινωνήσετε με κάποιον μηχανικό υποστήριξης.

ΣΥΝΗΘΕΙΣ ΕΡΩΤΗΣΕΙΣ
Τι είναι το Windows Update;
Το Windows Update είναι η ηλεκτρονική επέκταση των Windows που σχεδιάστηκε για να σας βοηθήσει να διατηρείτε τον υπολογιστή σας ενημερωμένο. Το Windows Update έχει έναν κεντρικό κατάλογο που ονομάζεται "Ενημερωμένες εκδόσεις προϊόντων" και περιλαμβάνει νέες βελτιώσεις προϊόντων, όπως αρχεία συστήματος, προγράμματα οδήγησης συσκευών, Service Pack, καθώς και νέες δυνατότητες των Windows. Οι βελτιώσεις αυτές σας προτείνονται, ανάλογα με τα στοιχεία που είναι απαραίτητα για τον υπολογιστή σας.

Τι είναι οι ενημερωμένες εκδόσεις προϊόντων;
Η σελίδα "Ενημερωμένες εκδόσεις προϊόντων" είναι ένας κατάλογος με ενημερώσεις κώδικα, ενημερωμένες εκδόσεις και βελτιώσεις για τα Windows και πολλά προγράμματα που λειτουργούν με Windows. Μπορείτε να κάνετε αναζήτηση στον κατάλογο, να επιλέξετε όσα στοιχεία θέλετε και να κάνετε λήψη και εγκατάσταση αυτών των στοιχείων απευθείας από το Web στον υπολογιστή σας.

Γιατί λαμβάνω ένα μήνυμα που αναφέρει ότι ένα στοιχείο ελέγχου ActiveX σε αυτήν τη σελίδα δεν είναι ασφαλές;
Αυτό το μήνυμα εμφανίζεται, όταν στη ρύθμιση ασφαλείας του Internet Explorer έχει οριστεί η επιλογή "Υψηλό". Για να λειτουργεί σωστά η τοποθεσία του Windows Update, πρέπει να αλλάξετε τις ρυθμίσεις ασφαλείας του προγράμματος περιήγησης για τη ζώνη Internet στην επιλογή "Μεσαίο" ή σε χαμηλότερο επίπεδο. Για να γίνει αυτό, κάντε κλικ στο μενού Εργαλεία και στη συνέχεια κάντε κλικ στην εντολή Επιλογές Internet. Κάντε κλικ στην καρτέλα Ασφάλεια και κατόπιν κάντε κλικ στο εικονίδιο Internet.

Τι είναι τα στοιχεία ελέγχου ActiveX;
Τα στοιχεία ελέγχου ActiveX είναι στοιχεία λογισμικού πολλών χρήσεων που ενσωματώνουν τεχνολογία ActiveX. Αυτά τα στοιχεία μπορούν να χρησιμοποιηθούν για την προσθήκη ειδικής λειτουργικότητας, όπως κίνηση ή αναδυόμενα μενού, σε ιστοσελίδες, εφαρμογές επιφάνειας εργασίας και εργαλεία ανάπτυξης λογισμικού. Το Windows Update χρησιμοποιεί στοιχεία ελέγχου ActiveX για να ελέγξει τι είδους λογισμικό έχει εγκατασταθεί στον υπολογιστή σας, ώστε να σας προσφέρει μια σωστή λίστα με ενημερωμένες εκδόσεις και άλλα στοιχεία λογισμικού, των οποίων ίσως να θέλετε να κάνετε λήψη.

Ποια είναι η καλύτερη ανάλυση για την προβολή της τοποθεσίας του Windows Update;
Οι ιστοσελίδες της τοποθεσίας του Windows Update σχεδιάστηκαν για να προβάλλονται σε οθόνη με ανάλυση 800 επί 600 ή υψηλότερη. Για να ορίσετε αυτήν την ανάλυση, κάντε κλικ στο κουμπί Έναρξη, τοποθετήστε το δείκτη του ποντικιού στην επιλογή Ρυθμίσεις και κατόπιν κάντε κλικ στην επιλογή Πίνακας Ελέγχου. Στη συνέχεια, κάντε διπλό κλικ στο εικονίδιο Οθόνη και κάντε κλικ στην καρτέλα Ρυθμίσεις.

Οι πληροφορίες σχετικά με τον υπολογιστή μου μεταβιβάζονται στη Microsoft;
Όχι. Κάθε φορά που μεταβαίνετε στον κατάλογο "Ενημερωμένες εκδόσεις προϊόντων", το Windows Update χρησιμοποιεί ένα στοιχείο ελέγχου, για να κάνει έλεγχο για ενημερωμένες εκδόσεις που αφορούν ειδικά τον υπολογιστή σας. Αυτός ο έλεγχος πραγματοποιείται τοπικά στον υπολογιστή σας, για να εξασφαλιστεί το απόρρητο και η ασφάλεια των δεδομένων σας, και δεν αποστέλλεται καμία πληροφορία στη Microsoft.

Πώς μπορώ να στείλω σχόλια στην ομάδα του Windows Update;
Για να στείλετε σχόλια σχετικά με αυτήν την τοποθεσία στην ομάδα του Windows Update, κάντε κλικ στη σύνδεση Επικοινωνία και στη συνέχεια στην επιλογή Windows Update στη μαύρη γραμμή εργαλείων, η οποία βρίσκεται στο επάνω μέρος κάθε σελίδας.

Όταν ένα νέο ενημερωτικό δελτίο ασφαλείας είναι διαθέσιμο στην τοποθεσία Microsoft TechNet Security στο Web, (τα περιεχόμενα αυτής της τοποθεσίας είναι στην αγγλική γλώσσα) είναι διαθέσιμο και στο Windows Update;
Συχνά υπάρχει καθυστέρηση μεταξύ της δημοσίευσης ενός ενημερωτικού δελτίου ασφαλείας στην τοποθεσία TechNet Security και της διάθεσης αυτής της ενημερωμένης έκδοσης ασφαλείας στο Windows Update. Αυτό οφείλεται στο γεγονός ότι τα νέα περιεχόμενα προστίθενται στο Windows Update με βάση ένα προγραμματισμένο χρονοδιάγραμμα. Ενώ το Windows Update καταβάλλει κάθε προσπάθεια ώστε οι σημαντικές ενημερωμένες εκδόσεις ασφαλείας να κυκλοφορούν το συντομότερο δυνατό, πρέπει να γίνεται έλεγχος και επαλήθευση των νέων περιεχομένων, για να εξασφαλιστεί ότι το Windows Update σας προσφέρει μόνο τις ενημερωμένες εκδόσεις που χρειάζεστε για τον συγκεκριμένο υπολογιστή σας. Συνήθως, οι ενημερωμένες εκδόσεις για τα υποστηριζόμενα προϊόντα εμφανίζονται στο Windows Update σε διάστημα μερικών ημερών ή το αργότερο μερικών εβδομάδων από την ανακοίνωσή τους στην τοποθεσία TechNet. Εάν γνωρίζετε ποια συγκεκριμένη έκδοση μιας ενημερωμένης έκδοσης ισχύει για το δικό σας υπολογιστή και δεν θέλετε να περιμένετε να τη λάβετε από το Windows Update, μπορείτε να κάνετε λήψη αυτής της ενημερωμένης έκδοσης κατευθείαν από την ιστοσελίδα του ενημερωτικού δελτίου ασφαλείας.

Ποια λειτουργικά συστήματα και προγράμματα υποστηρίζονται από το Windows Update;
Το Windows Update προσφέρει ενημερωμένες εκδόσεις ασφαλείας που ισχύουν για τα Windows 98, τα Windows 98 Δεύτερη Έκδοση, τα Windows Millennium Edition (Windows Me), τα Windows NT 4.0, τα Windows 2000, τον Internet Information Server 4.0 (IIS 4.0), τις υπηρεσίες Internet Information Services 5.0 (IIS 5.0) και τον Internet Explorer. Ωστόσο, εμφανίζονται μόνο οι ενημερωμένες εκδόσεις που ισχύουν για τον δικό σας υπολογιστή. Ορισμένα λειτουργικά συστήματα διακομιστών και προγράμματα που βασίζονται σε διακομιστές δεν υποστηρίζονται από το Windows Update.

Αναφέρατε ότι ορισμένα προϊόντα διακομιστών δεν υποστηρίζονται. Ποια προϊόντα διακομιστών δεν υποστηρίζονται;
Το Windows Update ξεκίνησε ως μια τοποθεσία Web που προσέφερε ενημερωμένες εκδόσεις για οικιακούς υπολογιστές πελατών των Windows. Με την πάροδο του χρόνου, το Windows Update επέκτεινε την προσφορά ενημερωμένων εκδόσεων ώστε να καλύπτει ορισμένα προϊόντα διακομιστών (όπως ο IIS) που χρησιμοποιούνται συχνότερα στα σπίτια. Ωστόσο, οι ενημερωμένες εκδόσεις ασφαλείας που αφορούν τέτοια προϊόντα διακομιστών, όπως οι ελεγκτές τομέων των Windows, ο Microsoft Exchange Server, ο Microsoft SQL Server, ο Microsoft ISA Server, κ.λπ. δεν υποστηρίζονται στο Windows Update. Οι πελάτες που χρησιμοποιούν τα προϊόντα αυτά πρέπει να ανατρέχουν τα ενημερωτικά δελτία ασφαλείας στην τοποθεσία Microsoft TechNet Security στο Web για πληροφορίες σχετικά με τις τοποθεσίες από τις οποίες μπορούν να κάνουν λήψη ενημερωμένων εκδόσεων για αυτά τα προϊόντα. (Τα περιεχόμενα αυτής της τοποθεσίας είναι στην αγγλική γλώσσα.)

Το Windows Update παρείχε πάντοτε ενημερώσεις κώδικα ασφαλείας και άλλες ενημερωμένες εκδόσεις για τον Internet Information Server 4.0 (IIS 4.0) και τις υπηρεσίες Internet Information Services 5.0 (IIS 5.0);
Όχι. Στις 23 Μαΐου 2001, το Windows Update προσέφερε ένα συγκεντρωτικό πακέτο με κρίσιμες ενημερώσεις για τις υπηρεσίες IIS 5.0 για πρώτη φορά. Παρόλο που αυτό το πακέτο δεν περιέχει όλες τις ενημερωμένες εκδόσεις των προϊόντων IIS που έχουν κυκλοφορήσει μέχρι σήμερα, παρέχει ένα συνδυασμό των πιο πρόσφατων ενημερώσεων κώδικα που κυκλοφόρησαν από το Μάρτιο του 2000 έως σήμερα. Για τις ενημερωμένες εκδόσεις που κυκλοφόρησαν πριν από το Μάρτιο του 2000, μεταβείτε στηΣελίδα του TechNet για τις υπηρεσίες Web των Windows (IIS). (Τα περιεχόμενα αυτής της τοποθεσίας είναι στην αγγλική γλώσσα.) Όταν δημιουργηθούν νέες ενημερώσεις κώδικα ασφαλείας ή άλλες κρίσιμες ενημερώσεις για τα προϊόντα IIS, το πακέτο αυτό θα αντικατασταθεί από ένα νέο πακέτο, το οποίο θα περιλαμβάνει τόσο τις νέες ενημερωμένες εκδόσεις όσο και όλες τις ενημερώσεις κώδικα του προηγούμενου πακέτου. Με τον τρόπο αυτό, δεν έχει σημασία πόσο συχνά επισκέπτεστε το Windows Update. Θα γνωρίζετε ότι πάντα θα λαμβάνετε τις πιο πρόσφατες ενημερωμένες εκδόσεις για τα προϊόντα IIS που διαθέτετε.
