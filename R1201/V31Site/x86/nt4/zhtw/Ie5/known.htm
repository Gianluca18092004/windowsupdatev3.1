Windows Update 已知問題
 以下是有關 Windows Update 網站的已知問題。如果您發現未列於下的問題，請在黑色的工具列上按一下 [意見信箱]，然後選擇 [Windows Update]，讓我們了解您的問題。
  1. 當您每次瀏覽此網站的這個區域時，Windows Update 會重新整理產品更新的目錄。這樣可以看到目前適用於您電腦的最新資訊。例如，假設您下載兩個佈景主題，然後回到 [重大更新] 目錄，我們會重新啟始設定重大更新的目錄，確保您看到的是更新過的訊息，不會看到您所下載過的這兩個佈景主題。它們仍然會顯示在 [顯示已更新組件]，但現在會顯示成已安裝。

  2. 在進行下載時，您將無法使用左邊的瀏覽列。這是為了確保您可以成功地下載檔案。當下載完成時，就可以使用瀏覽列。

  3. 在您選擇 Windows Update 首頁的連結後，該連結並不會改變顏色。

  4. 無法完全使用鍵盤來存取網站上的某些功能。

  5. 在很罕見的狀況下，如果您使用多個瀏覽器視窗瀏覽本網站，並同時執行多個產品更新的控制項來檢查修正檔案，您的瀏覽器會長時間被鎖住。

  6. 從重大更新目錄下載元件之後，若顯示下載確認畫面時，不要開啟新的瀏覽器視窗。否則，就會顯示 script 錯誤訊息。

  7. 在罕見的情況下，當您透過數據機連線到 Internet，可能會看到 script 錯訊。若發生此錯誤，請重新連線。

  8. 如果安全性設定為高，將無法存取產品更新目錄。若要使用此網站的產品更新區域，需將安全性設定為中或低。若要變更安全性設定值：
    1. 在 [工具] 功能表按一下 [Internet 選項]。
    2. 按一下 [安全性] 標籤。
    3. 選擇安全性的設定為 [中安全性]。

  9. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  10. Windows Update 網站可能會顯示與您作業系統或瀏覽器不同的語言。例如，您的作業系統及瀏覽器可能是英文，但卻出現法文版的 Windows Update 網站。Windows Update 網站是依照您瀏覽器上的語言設定顯示。若要變更您瀏覽器的語言設定：
    1. 在 Internet Explorer，按一下 [工具] 功能表。
    2. 按一下 [Internet 選項]。
    3. 在 [一般] 標籤，按一下 [語言] 按鈕。
    4. 選擇您要使用的語言，然後使用 [上移] 按鈕，將您的選擇移動到語言清單的頂端。

  11. 防毒軟體使用者：某些防毒軟體可能會妨礙下載及安裝某些元件。為防止發生此狀況，從重大更新目錄下載項目前，請先取消防毒軟體的功能。

  12. Windows Update 的 Internet Explorer 我的最愛連結可能無法正確運作。若要取得最佳結果，請使用下列位置中的連結存取 Windows Update： [開始] 功能表 (按一下 [開始]，然後按一下 [Windows Update])， 或在 Internet Explorer 位址列中鍵入 windowsupdate.microsoft.com。

  13. 若您最近升級 Internet Explorer 瀏覽器，當您返回 Windows Update 時您可看到在您執行較早版本的 Internet Explorer 時安裝的修正檔。您所提供的修正檔現在針對您升級的瀏覽器而提供。若要確認您新的 Internet Explorer 瀏覽器具有較高的安全等級，Microsoft 建議您為新的瀏覽器安裝這些修正檔。
