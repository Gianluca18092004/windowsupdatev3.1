由於以下原因之一，您會到達此網頁：
  1. 到 Windows Update 網站的控制項及/或結構的更新可能引起這個錯誤訊息。如需詳細資訊，請讀取 Microsoft Knowledge Base (KB) Article Q243787。
  2. 您選擇不要下載軟體控制項，或是在您下載控制項時曾發生問題，在此情形下，您將無法使用 Windows Update 網站的許多資訊。如果您現在要下載控制項，請按上面的 [再次嘗試]。
  3. 您沒有此電腦的管理權限。您若要使用 Windows Update，請洽詢您的系統管理人員，更新您的軟體。
  4. 您可能必須清除瀏覽器的快取。如需詳細資訊，請讀取 Microsoft KB Article Q193385。
  5. 您可能必須移除 Windows Update 目錄。如需詳細資訊，請讀取 Microsoft KB Article Q243787。
  6. Windows Update 網站現在不提供 Windows NT® 4.0 Terminal Server Edition 的修正?。若需更多資訊，請參? Microsoft Knowledge Base (KB) Article Q323564。
