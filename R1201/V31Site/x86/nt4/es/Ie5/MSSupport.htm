SOPORTE TÉCNICO DE ACTUALIZACIONES DE SOFTWARE
Sin costo alguno, el soporte técnico mediante correo electrónico para el sitio Windows Update incluye: ayuda para la exploración del sitio Web, mensajes de error o acerca de la calidad del contenido del sitio y ayuda básica para descarga. Encontrará información acerca del resto de temas de ayuda, incluidos los problemas de instalación y configuración, en nuestro servicio de soporte técnico personal y profesional en http://support.microsoft.com/directory/. Para productos MSN, visite http://help.msn.com/. En algunos casos es posible que se aplique una tarifa de soporte técnico; consulte los sitios para obtener más detalles

Para obtener información de soporte técnico fuera de EE.UU y Canadá, póngase en contacto con su subsidiaria local. La lista más reciente está disponible en el sitio Web de Soporte técnico en línea donde puede seleccionar la opción de soporte en todo el mundo (Worldwide Support). Si no existe ninguna oficina subsidiaria de Microsoft en su país, póngase en contacto con el distribuidor que le suministró el producto Microsoft.

Los servicios de soporte técnico de Microsoft están sujetos a los precios y condiciones vigentes de Microsoft, que pueden cambiar sin previo aviso.

¿Cómo desinstalo las actualizaciones de software?
Algunas actualizaciones pueden desinstalarse mediante la página Actualizaciones de productos del sitio de Windows Update:
  1. Vaya a la página Actualizaciones de productos.
  2. Si aparece una pregunta, asegúrese de hacer clic en el botón Sí.
  3. Cuando aparezca la página Actualizaciones de productos, haga clic en Mostrar actualizaciones instaladas en la parte superior de la página.
  4. Si se puede desinstalar una actualización, el botón Desinstalar aparecerá a la derecha de la descripción de la actualización.
  5. Haga clic en el botón Desinstalar que se muestra junto a la descripción de la actualización que desea desinstalar.
Las actualizaciones que no se pueden desinstalar desde la página Actualizaciones de productos, pueden desinstalarse mediante otros métodos. Consulte la página Lea esto primero para obtener información de desinstalación.

PREGUNTAS MÁS FRECUENTES
¿Qué es Windows Update?
Windows Update es la extensión en línea de Windows, diseñada para ayudarle a mantener actualizado su equipo. Windows Update dispone de un catálogo central denominado Actualizaciones de productos que incluye las nuevas mejoras de productos como archivos de sistema, controladores de dispositivo, Service Pack y características nuevas de Windows. Estas mejoras se recomiendan en función de las necesidades específicas de su equipo.

¿Qué son las Actualizaciones de productos?
La página Actualizaciones de productos es un catálogo de soluciones, actualizaciones y mejoras para Windows y otros programas que funcionan con Windows. Puede examinar el catálogo, seleccionar los componentes que desee y descargarlos e instalarlos en su equipo desde el Web.

¿Por qué obtengo un mensaje comunicando que el control ActiveX de esta página no es seguro?
Este mensaje se muestra cuando la configuración de seguridad de Internet Explorer tiene el valor Alta. Para que el sitio de Windows Update funcione adecuadamente, debe establecer como Media o Baja la configuración de seguridad de su explorador correspondiente a Zona de Internet. Para hacerlo, haga clic en el menú Herramientas, después haga clic en Opciones de Internet. Seleccione la ficha Seguridad y haga clic en Internet.

¿Qué es un control ActiveX?
Los controles ActiveX son componentes de software reutilizables que incorporan la tecnología ActiveX. Estos componentes pueden utilizarse para agregar funciones especiales, como animaciones o menús emergentes, a páginas Web, aplicaciones de escritorio y herramientas de desarrollo de software. Windows Update utiliza controles ActiveX para comprobar el software instalado en su equipo y ofrecerle la lista correcta de actualizaciones y software que le puede interesar descargar.

¿Cuál es la mejor resolución para ver el sitio de Windows Update?
Las páginas Web del sitio de Windows Update se diseñaron para pantallas con una resolución de 800 por 600 o superior. Para establecer esta resolución, haga clic en el botón Inicio, seleccione Configuración y haga clic en Panel de control. Después haga doble clic en el icono Pantalla y haga clic en la ficha Configuración.

¿Se transmite información de mi equipo a Microsoft?
No. Cada vez que visita el catálogo de Actualizaciones de productos, Windows Update utiliza un control para comprobar las actualizaciones específicas de su equipo. Esta comprobación se realiza localmente en su equipo para asegurar la privacidad y seguridad de sus datos y no se envía ninguna información a Microsoft.

¿Cómo enviar mis comentarios al equipo de Windows Update?
Puede enviar sus comentarios acerca de este sitio al equipo de Windows Update haciendo clic en Escríbanos, y después en Windows Update en la barra de herramientas negra de la parte superior de cada página.

Cuando haya un nuevo boletín de seguridad disponible en el sitio Web de seguridad de Microsoft TechNet, (este sitio está en inglés) también está disponible en Windows Update?
A menudo se produce un retraso entre el momento en que se expone un boletín de seguridad en el sitio de seguridad de TechNet, y el momento en que la actualización de seguridad está disponible en Windows Update. Esto se debe a que el contenido nuevo se agrega a Windows Update siguiendo unos tiempos programados. A pesar de que Windows Update se esfuerza al máximo para presentar actualizaciones de seguridad importantes lo antes posible, el contenido nuevo debe probarse y comprobarse para asegurar que Windows Update le ofrece sólo las actualizaciones que usted pueda necesitar para su equipo concreto. Normalmente, las actualizaciones para productos admitidos aparecen en Windows Update en un plazo de unos días a unas semanas a partir de la fecha en que se anuncian en el sitio de TechNet. Si sabe qué versión concreta de una actualización es aplicable a su equipo y no desea esperar a obtenerla desde Windows Update, puede descargar la actualización directamente desde la página Web del boletín de seguridad.

¿Qué sistemas operativos y programas admite Windows Update?
Windows Update ofrece actualizaciones de seguridad que se aplican a Windows 98, Windows 98 Segunda edición, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Servicios de Internet Information Server 5.0 (IIS 5.0) e Internet Explorer. Sin embargo, usted únicamente verá las actualizaciones que sean aplicables a su equipo en concreto. No se admiten en Windows Update algunos sistemas operativos de servidor y programas basados en servidor.

Se ha mencionado que no se admiten algunos productos de servidor. ¿Cuáles son éstos?
Windows Update comenzó su andadura como un sitio Web que ofrecía actualizaciones para los equipos domésticos de los clientes de Windows. Con el tiempo, Windows Update ha ampliado su oferta de actualizaciones para dar soporte a algunos de los productos de servidor (como IIS) que más a menudo se utilizan en el hogar. Sin embargo, las actualizaciones de seguridad específicas para estos productos de servidor como por ejemplo los controladores de dominio de Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server y otros no se admiten en Windows Update. Los clientes que utilicen estos productos deben revisar los boletines de seguridad presentes en el sitio Web de seguridad de Microsoft TechNet para obtener información de los lugares desde los que se pueden descargar actualizaciones para dichos productos. (Este sitio está en inglés.)

¿Windows Update ha ofrecido siempre correcciones de seguridad y otras actualizaciones para Internet Information Server 4.0 (IIS 4.0) y los Servicios de Internet Information Server 5.0 (IIS 5.0)?
No. El 23 de mayo de 2001, Windows Update ofreció un paquete acumulativo de actualizaciones cruciales para IIS 5.0 por primera vez. Aunque este paquete no incluye todas las actualizaciones para IIS aparecidas hasta la fecha, sí ofrece una combinación de las correcciones más recientes que aparecieron entre marzo de 2000 y la fecha actual. En lo que respecta a actualizaciones que aparecieron con anterioridad a marzo de 2000, visite la página de TechNet relacionada con los servicios Web de Windows (IIS). (Este sitio está en inglés.) Cuando aparezcan nuevas correcciones de seguridad u otras actualizaciones cruciales para IIS, este paquete será sustituido por uno nuevo, que incluirá tanto las actualizaciones nuevas como todas las correcciones del paquete anterior. De esta forma, independientemente de la frecuencia con que visite Windows Update, sabrá en todo momento que obtiene las actualizaciones más recientes para sus productos de IIS.

¿Dónde puedo acudir para obtener información relacionada con todas las actualizaciones de seguridad?
Puede consultar todos los boletines de seguridad en el sitio Web de seguridad de Microsoft TechNet. Asimismo, puede suscribirse a Microsoft Security Notification Service (Servicio de notificaciones de seguridad de Microsoft) para recibir avisos por correo electrónico con relación a boletines de seguridad que hayan aparecido recientemente. (Estos sitios están en inglés.)
