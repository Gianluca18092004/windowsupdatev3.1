Impossibile visualizzare la pagina
Riprova

Questa pagina viene visualizzata per uno dei seguenti motivi:
  1. È probabile che questo messaggio di errore sia stato causato da aggiornamenti al controllo e/o all'architettura del sito Windows Update. Per ulteriori informazioni, vedere l'articolo della Microsoft Knowledge Base (KB) Q243787 (informazioni in lingua inglese).
  2. Si è scelto di non eseguire il download dei controlli del software o si è verificato un problema durante il download dei controlli; in questo caso, gran parte del sito Windows Update non sarà disponibile. Se si desidera eseguire ora il download dei controlli, fare clic su Riprova.
  3. Non si dispone dei privilegi amministrativi per il computer in uso. Per utilizzare Windows Update è necessario chiedere all'amministratore del sistema di aggiornare il software.
  4. È possibile che la cache del browser sia stata svuotata. Per ulteriori informazioni, vedere l'articolo della Microsoft Knowledge Base Q193385 (informazioni in lingua inglese).
  5. Potrebbe essere necessario rimuovere la directory di Windows Update. Per ulteriori informazioni, vedere l'articolo della Microsoft Knowledge Base Q243787 (informazioni in lingua inglese).
  6. Gli aggiornamenti per Windows NT® 4.0 Terminal Server Edition non sono disponibili attraverso il sito Web di Windows Update Web. Per ulteriori informazioni, vedere l'articolo della Microsoft Knowledge Base (KB) numero Q323564.
