Du har kommit till den här sidan av något av följande skäl:
  1. Uppdateringar för kontroll och/eller arkitektur på Windows Update kan vara orsaken till att det här felmeddelandet visas. Om du vill ha mer information läser du Microsoft KB-artikeln Q243787.
  2. Du har valt att inte hämta programkontrollerna, eller så uppstod det ett problem vid hämtningen av dem, vilket betyder att du inte får tillgång till större delen av webbplatsen Windows Update. Om du vill hämta programkontrollerna klickar du på Försök igen ovan.
  3. Du har inte administrativa rättigheter på den här datorn. Om du vill använda Windows Update kontaktar du din systemadministratör för att uppdatera programvaran.
  4. Du kanske behöver rensa webbläsarens cacheminne. Om du vill ha mer information läser du Microsoft KB-artikeln Q193385.
  5. Du kanske behöver ta bort Windows Update-katalogen. Om du vill ha mer information läser du Microsoft KB-artikeln Q243787.
  6. Det finns inga uppdateringar för Windows NT® 4.0 Terminal Server Edition tillgängliga på Windows Update Web. Mer information finns i artikeln Q323564 i Microsoft Knowledge Base (KB).
