Actualizações do controlador de dispositivo
 Como transferir e instalar actualizações do controlador de dispositivo

Importante Para desinstalar uma actualização de um controlador que tenha transferido a partir do Windows Update, terá que ser capaz de reinstalar o controlador original fornecido com o dispositivo. Remover um controlador ou uma actualização de um controlador sem o substituir fará com que o dispositivo de hardware deixe de funcionar.

Use Device Manager to remove driver updates from your computer:

  1. Clique com o botão direito do rato no ícone O meu computador no ambiente de trabalho do computador
  2. Clique em Propriedades e, em seguida, clique no separador Gestor de dispositivos.
  3. Clique uma vez para realçar o dispositivo de hardware e, em seguida, clique em Propriedades.
  4. Clique no separador Controlador e, em seguida, clique em Actualizar controlador.
  5. Quando o Assistente de actualização de controladores de dispositivos iniciar, seleccione Especificar a localização do controlador (Avançado) e, em seguida, clique em Seguinte.
  6. Seleccione Mostrar uma lista de controladores conhecidos para este dispositivo, para que eu possa seleccionar um controlador específico e, em seguida, clique em Seguinte.
  7. Clique em Disco e, em seguida, clique em Procurar para localizar o ficheiro .inf para o controlador que pretende instalar (isto irá remover o controlador do Windows Update e substituí-lo pelo controlador original do dispositivo de hardware).
Nota Quando regressar ao Windows Update, é possível que lhe seja oferecida novamente a mesma actualização de controlador. Não a instale. Se tiver questões ou problemas a desempenhar o procedimento para desinstalar uma actualização de controlador do Windows Update, utilize as hiperligações de suporte fornecidas para contactar um profissional de suporte.
