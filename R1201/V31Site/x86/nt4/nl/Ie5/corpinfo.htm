Informatie over Windows Update
Alleen voor IT-managers van bedrijven

Zoekt u verspreidbare downloadpakketten en belangrijke updates voor uw bedrijfsnetwerk? Breng vandaag nog een bezoek aan de Windows Update Corporate Site!

U kunt de Windows Update-pictogrammen in het menu Start verwijderen. Als u deze pictogrammen uitschakelt, kunnen gebruikers vanuit Windows 98 niet meer naar de Windows Update-site (http://v31update.vornext.gq) gaan.

De Windows Update-pictogrammen verwijderen uit het menu Start:

  1. Verwijder het pictogram uit de groep Start/Instellingen door de waarde NoWindowsUpdate = 1 op te geven voor de registersleutel HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer. (Dit is een DWORD-waarde. Als u hier de waarde 0 opgeeft, wordt het Windows Update-pictogram weergegeven.)
  2. Start de computer opnieuw.
  3. Om het pictogram te verwijderen uit de hoofdgroep van het menu Start, voert u de volgende opdracht uit vanaf de opdrachtregel: Wupdmgr.exe -shortcut.
Opmerking: wijzig het Register alleen als dat strikt noodzakelijk is. Een fout in het Register kan nadelige gevolgen hebben voor de werking van uw computer.
