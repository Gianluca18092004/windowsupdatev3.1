ONDERSTEUNING VOOR SOFTWARE-UPDATES
De gratis ondersteuning voor de Windows Update-site (per e-mail) bestaat uit: hulp bij het navigeren op de website, informatie over de kwaliteit van de site of foutberichten, en basisinformatie over het downloaden van updates. Alle andere vormen van hulp, zoals ondersteuning bij problemen met installeren en instellen, is beschikbaar via de website http://support.microsoft.com/directory/. Als u op zoek bent naar MSN-producten, brengt u een bezoek aan http://help.msn.com/. Deze ondersteuning is niet altijd gratis. Meer informatie hierover vindt u op de site.

Gebruikers buiten de VS en Canada kunnen de dichtstbijzijnde Microsoft-vestiging informatie vragen over ondersteuning. Ga naar de website voor on line ondersteuning en selecteer de optie onder Worldwide Support voor meer informatie. Als er geen Microsoft-vestiging is in uw land, kunt u contact opnemen met het bedrijf waar u het Microsoft-product hebt aangeschaft.

De prijzen en voorwaarden van de ondersteuningsservices van Microsoft kunnen zonder voorafgaande waarschuwing worden gewijzigd.

De installatie van software-updates ongedaan maken
Sommige updates kunnen worden verwijderd via de pagina Updates op producten van de Windows Update-site:
  1. Ga naar de pagina Updates op producten.
  2. Antwoord met Ja als een berichtvenster wordt weergegeven.
  3. Als de pagina Updates op producten wordt weergegeven, klikt u bovenaan de pagina op Geïnstalleerde updates weergeven
  4. Als een update kan worden verwijderd, verschijnt de knop Verwijderen rechts van de beschrijving van de update.
  5. Klik op de knop Verwijderen naast de beschrijving van de update die u wilt verwijderen.
Updates die niet via Updates op producten kunnen worden verwijderd, kunnen soms wel op een andere manier worden verwijderd. Zie de pagina Lees dit eerst voor de betreffende update voor meer informatie hierover.

VEELGESTELDE VRAGEN
Wat is Windows Update?
Windows Update is een on line uitbreiding van Windows waarmee u uw computer actueel kunt houden. Windows Update heeft een centrale catalogus: Updates op producten. Deze catalogus bevat nieuwe productverbeteringen zoals systeembestanden, stuurprogramma's, Service Packs en nieuwe Windows-functies. De verbeteringen die nodig zijn voor uw computer, worden u automatisch aangeraden.

Wat is Updates op producten?
De pagina Updates op producten is een catalogus met oplossingen, updates en verbeteringen voor Windows en veel programma's die met Windows werken. U kunt door de catalogus bladeren, zoveel componenten selecteren als u wilt en ze vervolgens rechtstreeks van het web op uw computer downloaden en installeren.

Waarom krijg ik het bericht dat een ActiveX-besturingselement op deze pagina niet veilig is?
Dit bericht wordt weergegeven als de beveiligingsinstelling van Internet Explorer is ingesteld op Hoog. Voor de Windows Update-site moet u het beveiligingsniveau voor de Internet-zone instellen op Gemiddeld of lager. U kunt de instellingen wijzigen door Internet-opties te kiezen in het menu Extra. Selecteer vervolgens het tabblad Beveiliging en klik op Internet.

Wat is een ActiveX-besturingselement?
ActiveX-besturingselementen zijn herbruikbare softwareonderdelen waarin ActiveX-technologie wordt gebruikt. Deze onderdelen kunnen worden gebruikt om gespecialiseerde functionaliteit, zoals animaties of pop-up menu's, toe te voegen aan webpagina's, bureaubladtoepassingen en softwareontwikkelprogramma's. Windows Update maakt gebruik van ActiveX-besturingselementen om te controleren welke software op uw computer is geïnstalleerd, zodat u een kloppende lijst krijgt met updates en andere software die u kunt downloaden.

Wat is de beste resolutie voor het bekijken van de Windows Update-site?
De webpagina's op de Windows Update-site zijn ontworpen voor een schermresolutie van 800 x 600 of hoger. U kunt deze resolutie instellen door op de knop Start te klikken, Instellingen aan te wijzen en vervolgens op Configuratiescherm te klikken. Dubbelklik vervolgens op het pictogram Beeldscherm en klik op het tabblad Instellingen.

Wordt informatie over mijn computer naar Microsoft verzonden?
Nee. Iedere keer als u Updates op producten bezoekt, wordt gekeken of er updates beschikbaar zijn voor uw computer. Deze controle wordt lokaal op uw computer uitgevoerd om de privacy en veiligheid van uw gegevens te garanderen en er wordt geen informatie naar Microsoft gestuurd.

Hoe stuur ik mijn feedback naar het Windows Update-team?
U kunt feedback over deze site naar het Windows Update-team sturen door in de zwarte werkbalk bovenaan een pagina op Schrijf ons te klikken en vervolgens op Windows Update.

Als er een nieuw beveiligingsbulletin wordt gepubliceerd op de Security-website van Microsoft TechNet (deze site is in het Engels) is het dan ook beschikbaar via Windows Update?
Er ligt vaak enige tijd tussen het moment waarop een beveiligingsbulletin wordt gepubliceerd op de TechNet Security-site en het moment waarop de beveiligingsupdate beschikbaar is via Windows Update. Dit komt omdat de Windows Update-site volgens een bepaald tijdschema wordt bijgewerkt. Hoewel het streven is om belangrijke beveiligingsupdates zo snel mogelijk beschikbaar te stellen, moeten nieuwe updates worden getest en gecontroleerd om ervoor te zorgen dat u alleen de updates te zien krijgt die voor uw systeem nodig zijn. Updates voor ondersteunde producten zijn over het algemeen een paar dagen tot een paar weken na de aankondiging op de TechNet-site verkrijgbaar via Windows Update. Als u weet welke versie van een update van toepassing is op uw computer en u niet wilt wachten totdat de update op de site van Windows Update staat, kunt u de update rechtstreeks downloaden vanaf de webpagina met het beveiligingsbulletin.

Welke besturingssystemen en programma's worden door Windows Update ondersteund?
Windows Update biedt beveiligingsupdates voor Windows 98, Windows 98 Tweede Editie, Windows Millennium Edition (Windows ME), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) en Internet Explorer. U krijgt echter alleen de updates te zien die van toepassing zijn op uw computer. Bepaalde serverbesturingssystemen en op servers gebaseerde toepassingen worden niet door Windows Update ondersteund.

Ik lees dat bepaalde serverproducten niet worden ondersteund. Welke serverproducten zijn dat precies?
Windows Update was oorspronkelijk een website waarop updates beschikbaar werden gesteld voor thuisgebruikers van een Windows-computer. Na verloop van tijd is het aanbod van Windows Update uitgebreid met updates voor een aantal serverproducten (zoals IIS) die steeds vaker door mensen thuis worden gebruikt. Beveiligingsupdates die specifiek bedoeld zijn voor serverproducten zoals Windows-domeincontrollers, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server, enzovoort worden echter niet door Windows Update ondersteund. Gebruikers van dergelijke producten kunnen de beveiligingsbulletins op de Security-website van Microsoft TechNet raadplegen voor informatie over updates op deze producten. (Deze site is in het Engels.)

Zijn alle beveiligingsupdates en andere updates voor Internet Information Server 4.0 (IIS 4.0) en Internet Information Services 5.0 (IIS 5.0) beschikbaar via Windows Update?
Nee. Op 23 mei 2001 was er voor het eerst een cumulatief pakket belangrijke updates voor IIS 5.0 beschikbaar via Windows Update. Dit pakket bevat niet alle IIS-updates die tot nu toe zijn uitgebracht, maar biedt een combinatie van de updates die tussen maart 2000 en nu zijn verschenen. Updates die voor maart 2000 zijn uitgebracht, kunt u downloaden vanaf de TechNet-pagina voor Windows Web services (IIS). (Deze site is in het Engels.) Als er nieuwe beveiligingsupdates of andere belangrijke updates voor IIS zijn, wordt dit pakket vervangen door een nieuw pakket dat zowel de nieuwe updates bevat als alle updates die in het vorige pakket waren opgenomen. Dus hoe vaak u Windows Update ook bezoekt, u kunt er zeker van zijn dat u altijd de meest recente updates voor uw IIS-producten krijgt.

Waar kan ik informatie vinden over alle beveiligingsupdates?
Alle updates worden beschreven in de beveiligingsbulletins op de Security-website van Microsoft TechNet. U kunt ook een abonnement nemen op de Microsoft Security Notification Service. U wordt dan per e-mail op de hoogte gesteld zodra er een beveiligingsbulletin is gepubliceerd. (Deze sites zijn in het Engels.)
