 Windows Update ist eine Onlineerweiterung von Windows. Gegenwärtig besteht Windows Update aus zwei Bereichen: dem Bereich Produktupdates und dem Bereich Supportinformationen. Mithilfe der Navigationsleiste auf der linken Seite des Bildschirms können Sie diese Bereiche durchsuchen.

Info
Der Bereich Produktupdates ist ein Katalog mit Fehlerkorrekturen, Updates und Verbesserungen für Windows und viele Windows-Programme. Mit Hilfe des Katalogs können Sie Komponenten aussuchen, die die Leistungsfähigkeit Ihres Computers verbessern.

Blättern Sie im Katalog, wählen Sie die gewünschten Komponenten aus, und downloaden und installieren Sie diese direkt vom Web auf Ihren Computer (falls Sie noch nie einen Download vom Web durchgeführt haben, lesen Sie den Abschnitt "Erforderliche Schritte" weiter unten). Dieser Katalog kann auf Wunsch anhand der bereits installierten Software ermitteln, welche Updates und Verbesserungen speziell für Ihren Computer zur Verfügung stehen.

Funktionsweise
Beim Aufruf der Webseite Produktupdates werden Sie in einem Dialogfeld gefragt, ob Produktupdates ermitteln soll, welche Komponenten bereits auf Ihrem Computer installiert sind. Durch diesen Vorgang soll die Sicherheit und Integrität der Systeminformationen sichergestellt werden. Die Informationen werden nicht an Microsoft übermittelt oder über das Internet gesendet. Die Systemüberprüfung gewährleistet, dass Sie stets die aktuellsten und am besten geeigneten Versionen der ausgewählten Komponenten von der Website downloaden. Darüber hinaus verschwenden Sie keine Zeit mit dem Download von Komponenten, die bereits auf Ihrem Computer installiert sind.

Die auf der Webseite angezeigte Komponentenliste können Sie durchsuchen, die gewünschten Komponenten auswählen und direkt auf Ihren Computer downloaden.
Stellen Sie sicher, dass Sie Wichtige Updates, die für Ihr System empfohlen werden, downloaden. Wichtige Updates beheben bekannte Probleme auf Ihrem Computer, z.B. in der Sicherheit.

Erforderliche Schritte
Produktupdates erleichtert den Download von Updates und Verbesserungen zu Software auf Ihrem Computer.
  1. Suchen Sie zunächst im Katalog.
  2. Wenn Sie sich eine bestimmte Komponente ausgesucht haben, klicken Sie auf das Kontrollkästchen daneben. Im Kontrollkästchen wird dann ein Häkchen angezeigt. Klicken Sie vor dem Download einer Komponente auf den Link "Wichtige Hinweise", um wichtige Informationen zu folgenden Punkten anzuzeigen:
    • Eine ausführliche Beschreibung der Komponente
    • Informationen zum Einsatz der Komponente
    • Anweisungen zur Deinstallation
    • Richtlinien zum Support
  3. Wenn Sie ein bereits ausgewähltes Produkt nicht downloaden möchten, klicken Sie erneut auf das Kontrollkästchen, um das Häkchen zu entfernen.
  4. Wenn Sie alle gewünschten Komponenten ausgewählt haben, klicken Sie auf die Schaltfläche Download.
  5. Es wird eine Webseite angezeigt. Vergewissern Sie sich, dass alle gewünschten Komponenten in der Liste angezeigt werden. Klicken Sie auf die Schaltfläche Zurück, um Änderungen vorzunehmen.
  6. Klicken Sie auf die Schaltfläche Download starten. Alle ausgewählten Komponenten werden gedownloadet und dann installiert. Bei der Installation bestimmter Komponenten werden Sie möglicherweise während der Installation zur Eingabe zusätzlicher Informationen aufgefordert.
Wenn die Meldung Installation abgeschlossen angezeigt wird, ist der Vorgang schon beendet! Für einige Komponenten müssen Sie möglicherweise den Computer nach der Installation der Komponenten herunterfahren und neu starten.

Die meisten Komponenten können direkt von der Seite Produktupdates aus deinstalliert werden. Gehen Sie folgendermaßen vor:

  1. Wechseln Sie zur Webseite Produktupdates. Klicken Sie auf Ja, wenn das Dialogfeld angezeigt wird.
  2. Wenn die Seite Produktupdates angezeigt wird, klicken Sie oben auf der Seite auf die Schaltfläche Installierte Updates anzeigen.
  3. Klicken Sie auf die Schaltfläche Deinstallieren rechts neben der Komponente, die Sie deinstallieren möchten.

Info
Der Bereich Supportinformationen ist das Informationszentrum zu Windows Update. Hier erhalten Sie Hilfe zum Umgang mit Windows Update, indem Sie entweder in den Seiten Häufig gestellte Fragen (FAQ) und Bekannte Probleme nachschlagen oder sich an den Microsoft-Support wenden.
