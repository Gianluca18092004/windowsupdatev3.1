Bekannte Probleme von Windows Update
 Die folgenden Probleme im Zusammenhang mit der Windows Update-Website sind bekannt. Sollten Sie ein Problem finden, das nicht aufgeführt ist, teilen Sie uns dies bitte mit. Klicken Sie dazu in der schwarzen Symbolleiste auf Feedback und dann auf Windows Update.
  1. Windows Update führt jedes Mal, wenn Sie diesen Bereich der Website besuchen, eine Neuinitialisierung des Produktupdates-Katalogs durch. Dadurch wird gewährleistet, dass immer der neueste Stand Ihres Computers angezeigt wird. Wenn Sie beispielsweise zwei neue Desktopmotive downloaden und dann wieder zum Produktupdates-Katalog wechseln, wird der Katalog neu initialisiert, damit die Ansicht aktualisiert wird und die beiden gedownloadeten Motive nicht mehr angezeigt werden. Diese werden zwar weiterhin in der Ansicht Installierte Updates anzeigen aufgeführt, sind jedoch als bereits installiert markiert.

  2. Während eines Downloads können Sie die linke Navigationsleiste nicht verwenden. Dadurch soll sichergestellt werden, dass der Download erfolgreich durchgeführt wird. Nach Abschluss des Downloads wird die Navigationsleiste wieder aktiviert.

  3. Die Farbe der Hyperlinks auf der Homepage für Windows Update ändert sich nicht, nachdem sie ausgewählt wurde.

  4. Bestimmte Funktionen der Website sind nicht über die Tastatur zugänglich, um die Zugriffsbereitschaft der Website zu gewährleisten.

  5. In sehr seltenen Fällen kann es vorkommen, dass der Browser für einen langen Zeitraum blockiert ist, wenn mehrere Instanzen der Website ausgeführt werden und gleichzeitig der Produktupdates-Katalog auf Updates überprüft wird.

  6. Öffnen Sie nach dem Download einer Komponente aus dem Produktupdates-Katalog kein neues Browserfenster, solange die Seite Download bestätigt angezeigt wird. Otherwise a script error message will appear.

  7. In seltenen Fällen kann bei dem Versuch, eine Internetverbindung über ein Modem herzustellen, ein Fehler bei der Skriptausführung auftreten. Versuchen Sie in diesem Fall, den Vorgang zu wiederholen.

  8. Wenn die Sicherheitseinstellungen auf Hoch eingestellt sind, können Sie nicht auf den Produktupdates-Katalog zugreifen. Für den Zugriff auf diesen Bereich der Website müssen Sie Ihre Sicherheitseinstellungen auf Mittel oder Niedrig einstellen. So ändern Sie die Sicherheitseinstellungen:
    1. Klicken Sie im Menü Extras auf Internetoptionen.
    2. Wählen Sie auf der Registerkarte Sicherheit die Einstellung Mittel.

  9. Software installieren oder aktualisieren dürfen nur Benutzer mit Administrationsrechten auf diesem Windows NT-Rechner. Daher können Benutzer ohne Administrationsrechte die Steuerelemente, die zum Anzeigen von bestimmten Seiten dieser Site erforderlich sind, nicht downloaden.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Unter Umständen wird die Windows Update-Site in einer anderen Sprache als der Sprache Ihres Browsers bzw. Betriebssystems angezeigt. Browser und Betriebssystem sind beispielsweise in Englisch, aber die Windows Update-Site erscheint in Französisch. Die Windows Update-Site wird in der Sprache angezeigt, der in der Spracheinstellung des Browsers die Priorität zugewiesen ist. So ändern Sie die Sprachpriorität des Browsers:
    1. Klicken Sie in Internet Explorer auf das Menü Extras.
    2. Klicken Sie auf Internetoptionen.
    3. Klicken Sie auf der Registerkarte Allgemein auf die Schaltfläche Sprachen.
    4. Wählen Sie die gewünschte Sprache aus, und verschieben Sie sie mit Hilfe der Schaltfläche Nach oben an die Spitze der Sprachenliste.

  12. Benutzer von Antivirensoftware: Manche Antivirenprogramme können den Download und die Installation bestimmter Komponenten stören. Um derartige Konflikte zu verhindern, deaktivieren Sie die Antivirensoftware, bevor Sie Komponenten aus dem Produktupdates-Katalog downloaden.

  13. Möglicherweise werden Links zur Windows Update-Site nicht ordnungsgemäß ausgeführt, wenn sie in Internet Explorer als Favoriten gespeichert sind. Sie können jedoch problemlos zu der Site gelangen, indem Sie den Windows Update-Link im Menü Extras von Internet Explorer (klicken Sie in Internet Explorer auf Extras und dann auf Windows Update), oder in der Internet Explorer-Adressleiste v31update.vornext.gq eingeben.

  14. Wenn Sie kürzlich ein Update auf eine neuere Version von Internet Explorer durchgeführt haben und die Windows Update-Site besuchen, werden möglicherweise Updates angezeigt, die Sie bereits unter der vorher verwendeten Version von Internet Explorer installiert haben. Die in dieser Situation vorgeschlagenen Updates sind spezifisch für Ihren aktualisierten Browser bereitgestellte Updates. Um sicherzustellen, dass Internet Explorer mit der stärksten verfügbaren Verschlüsselung ausgeführt wird, empfiehlt Microsoft, diese Updates für Ihren neuen Browser zu installieren.

  15. Der Link Weitere Informationen steht auf der Windows Update-Website nicht mehr zur Verfügung. Weitere Informationen über die verfügbaren Sicherheitsupdates von Windows Update erhalten Sie, indem Sie die folgenden Schritte durchführen:
    1. Wechseln Sie zur Microsoft Security-Website.
    2. Klicken Sie auf Search for Bulletins and Patches.
    3. Geben Sie in das Textfeld Search die gewünschte Artikelnummer der Microsoft Knowledge Base (KB) ein.
    4. Klicken Sie auf Suchen.

  16. Wenn Sie ein Update entfernen möchten, das Sie von der Windows Update-Website installiert haben, gehen Sie folgendermaßen vor:
    1. Klicken Sie auf Start, zeigen Sie auf Einstellungen, und klicken Sie dann auf Systemsteuerung (unter Windows XP klicken Sie auf Start und dann auf Systemsteuerung).
    2. Klicken Sie auf Software.
    3. Klicken Sie auf die Komponente, die Sie entfernen möchten, und anschließend auf Hinzufügen/Entfernen. (Unter Windows XP und Windows 2000 klicken Sie auf Ändern/Entfernen.)
    Anmerkung Wenn die Komponente nicht unter Software auufgeführt ist, kann sie nicht vom Computer entfernt werden.
