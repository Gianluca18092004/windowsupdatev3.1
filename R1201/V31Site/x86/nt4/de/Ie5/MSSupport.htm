Anweisungen zur Deinstallation von Softwareupdates
Einige Updates können über die Seite Produktupdates auf der Windows Update-Site deinstalliert werden:
  1. Wechseln Sie zur Seite Produktupdates.
  2. Klicken Sie auf Ja, wenn eine Meldung mit einer Aufforderung angezeigt wird.
  3. Wenn die Seite Produktupdates angezeigt wird, klicken Sie oben auf der Seite auf Installierte Updates anzeigen.
  4. Wenn für ein Update die Deinstallation zur Verfügung steht, wird rechts neben der Beschreibung zu dem Update eine Schaltfläche Deinstallieren angezeigt.
  5. Klicken Sie neben der Beschreibung des zu deinstallierenden Updates auf Deinstallieren.
Updates, die nicht über die Seite Produktupdates deinstalliert werden können, können möglicherweise mithilfe anderer Verfahren deinstalliert werden. Weitere Informationen zur Deinstallation erhalten Sie über die Infoseite Bitte zuerst lesen, die für alle Updates bereitgestellt wird.

HÄUFIG GESTELLTE FRAGEN (FAQ)
Was ist Windows Update?
Windows Update ist eine Onlineerweiterung für Windows, die entwickelt wurde, um Ihren Computer auf dem neuesten Stand zu halten. Windows Update enthält den zentralen Katalog Produktupdates mit neuen Produktverbesserungen, wie zum Beispiel Systemdateien, Gerätetreibern, Service Packs und neuen Windows-Features. Empfehlungen zur Installation der Verbesserungen erfolgen auf der Grundlage der für Ihren Computer benötigten Komponenten.

Wozu dient die Seite "Produktupdates"?
Die Seite Produktupdates ist ein Katalog mit Fixes, Updates und Verbesserungen für Windows und viele Windows-Programme. Blättern Sie in dem Katalog, wählen Sie die gewünschten Komponenten aus, und downloaden und installieren Sie diese direkt aus dem Web auf Ihren Computer.

Warum erhalte ich eine Meldung, dass ein ActiveX-Steuerelement auf dieser Seite nicht sicher ist?
Diese Meldung wird angezeigt, wenn die Sicherheitsstufe in Internet Explorer auf Hoch eingestellt ist. Sie müssen die Sicherheitsoptionen in Ihrem Browser auf Mittel oder Niedrig einstellen, damit die Windows Update-Site ordnungsgemäß funktioniert. Klicken Sie dazu im Menü Extras auf Internetoptionen. Wählen Sie die Registerkarte Sicherheit aus, und klicken Sie dann auf Internet.

Was ist ein ActiveX-Steuerelement?
ActiveX-Steuerelemente sind wiederverwendbare Komponenten, die die ActiveX-Technologie nutzen. Mit diesen Komponenten können Webseiten, Desktopanwendungen und Software-Entwicklungstools um spezielle Funktionen erweitert werden, z. B. Animationen oder Popupmenüs. Windows Update überprüft mit ActiveX-Steuerelementen, welche Software auf dem Computer installiert ist, um anhand dieser Informationen eine aktuelle Vorschlagsliste mit Updates und sonstiger Software zu erstellen, die Sie downloaden sollten.

Welches ist die optimale Auflösung für die Anzeige der Windows Update-Site?
Die Webseiten der Windows Update-Site sind für eine Bildschirmauflösung von 800x600 Punkt oder höher konzipiert. Klicken Sie zum Einstellen dieser Auflösung auf die Schaltfläche Start, zeigen Sie auf Einstellungen, und klicken Sie dann auf Systemsteuerung. Doppelklicken Sie anschließend auf das Symbol Anzeige, und klicken Sie auf die Registerkarte Einstellungen.

Werden Informationen über meinen Computer an Microsoft übermittelt?
Nein. Bei jedem Aufruf des Produktupdates-Katalogs überprüft Windows Update Ihren Computer mithilfe eines Steuerelements auf bestimmte Updates. Diese Überprüfung wird lokal auf Ihrem Computer durchgeführt, um die Vertraulichkeit und Sicherheit Ihrer Daten zu wahren; dabei werden keine Informationen an Microsoft gesendet.

Wie sende ich Feedback an das Windows Update-Team?
Wenn Sie Kommentare zu dieser Site an das Windows Update-Team senden möchten, klicken Sie auf Feedback und dann auf der schwarzen Symbolleiste am oberen Seitenrand auf Windows Update.

Wenn ein neues Security Bulletin in der Microsoft TechNet Security-Website (diese Site ist in englischer Sprache) verfügbar ist, ist es dann auch in der Windows Update-Site verfügbar?
Häufig kommt es zu Verzögerungen zwischen der Veröffentlichung eines Security Bulletins in der TechNet Security-Site und der Verfügbarkeit des Sicherheitsupdates bei Windows Update. Der Grund dafür ist der Windows Update zugrunde liegende Terminplan. Bei Windows Update wird versucht, wichtige Sicherheitsupdates so schnell wie möglich zu veröffentlichen, aber neue Inhalte müssen getestet und überprüft werden, um sicherzustellen, dass Windows Update Ihnen nur die für Ihren Computer benötigten Updates zur Verfügung stellt. Normalerweise sind Updates für unterstützte Produkte bei Windows Update innerhalb weniger Tage bis Wochen nach Bekanntgabe in der TechNet-Site verfügbar. Wenn Sie wissen, welches spezielle Update für Ihren Computer bestimmt ist und nicht darauf warten möchten, dass es Ihnen von Windows Update zur Verfügung gestellt wird, können Sie das Update direkt von der Security Bulletin-Webseite downloaden.

Welche Betriebssysteme und Programme werden von Windows Update unterstützt?
Windows Update bietet Sicherheitsupdates für Windows 98, Windows 98 Second Edition, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, IIS 4.0 (Internet Information Server), IIS 5.0 (Internet Information Services) und Internet Explorer an. Ihnen werden jedoch nur die für Ihren Computer bestimmten Updates angezeigt. Einige Serverbetriebssysteme und serverbasierte Programme werden von Windows Update nicht unterstützt.

Es wird gesagt, dass einige Serverprodukte nicht unterstützt werden. Um welche Serverprodukte handelt es sich dabei?
Windows Update wurde als Website gestartet, die Updates für die Heimcomputer von Windows-Anwendern angeboten hat. Nach und nach wurde das Windows Update-Angebot mit Updates für einige Serverprodukte (wie IIS), die häufig auf Heimcomputern verwendet werden, erweitert. Sicherheitsupdate für Serverprodukte wie Windows-Domänencontroller, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server usw. werden jedoch von Windows Update nicht unterstützt. Kunden, die diese Produkte verwenden, sollten sich in den Security Bulletins in der Microsoft TechNet Security-Website informieren, wo Updates für diese Produkte gedownloadet werden können. (Diese Site ist in englischer Sprache.)

Hat Windows Update schon immer Sicherheitsfixes und weitere Updates für IIS 4.0 (Internet Information Server 4.0) und IIS 5.0 (Internet Information Services 5.0) zur Verfügung gestellt?
Nein. Am 23. Mai 2001 hat Windows Update erstmals ein kumulatives Paket wichtiger Updates für IIS 5.0 zur Verfügung gestellt. Das Paket enthält zwar noch nicht alle Updates für IIS, stellt aber eine Kombination der aktuellsten Fixes zur Verfügung, die zwischen März 2000 und heute veröffentlicht wurden. Updates, die vor März 2000 veröffentlicht wurden, finden Sie in der TechNet-Seite zu Windows-Webdiensten (IIS). (Diese Site ist in englischer Sprache.) Wenn neue Sicherheitsfixes oder sonstige wichtige Updates für IIS erstellt werden, wird das Paket durch ein neues Paket ersetzt, das sowohl die neuen Updates als auch sämtliche Fixes des vorherigen Pakets enthält. Auf diese Weise erhalten Sie immer die aktuellsten Updates für Ihr IIS-Produkt, unabhängig davon, wie häufig Sie Windows Update besuchen.

Wo kann ich Informationen über alle Sicherheitsupdates erhalten?
Sie können alle Security Bulletins in der Microsoft TechNet Security-Website anzeigen. Sie können auch den Benachrichtigungsdienst Microsoft Security Notification Service abonnieren und E-Mail-Warnungen zu kürzlich veröffentlichten Security Bulletins erhalten. (Diese Sites sind in englischer Sprache.)
