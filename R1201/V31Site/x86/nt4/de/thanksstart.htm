Vielen Dank für Ihr Interesse an
Windows Update

Windows Update ist die Onlineerweiterung von Windows, die Ihnen dabei hilft, die Möglichkeiten Ihres Computers in vollem Umfang zu nutzen.

Hinweis Sie müssen ein gültiges lizenziertes Windows-Betriebssystem ausführen, um auf die Windows Update-Website zugreifen zu können.

Windows® 98 und Windows® 2000-Benutzer:
Rufen Sie den Windows Update-Service über den Link im Menü Start auf.

Windows® 95 und Windows NT® 4.0-Benutzer:
Sie müssen Internet Explorer 4 oder höher verwenden, um Windows Update nutzen zu können.

Alle Benutzer:
Nur Browser, die ActiveX-Steuerelemente unterstützen haben Zugang zu Windows Update. Weitere Informationen über ActiveX-Steuerelemente finden Sie unter Windows Update-Häufig gestellte Fragen (FAQ).

Besuchen Sie die Website Internet Explorer-Downloads, um die neueste Internet Explorer-Version zu downloaden.

Wenn Internet Explorer 5 installiert ist, können Sie die Windows Update-Site besuchen, indem Sie die Adresse http://v31update.vornext.gq in die Adressleiste eingeben.

Um verfügbare Updates zu erhalten können währenddessen:

  • Windows 2000-Benutzer die Site Windows 2000-Downloads besuchen.
  • Windows NT 4.0-Benutzer die Site Microsoft Download Center besuchen.
  • Windows 98-Benutzer die Site Windows 98-Downloads besuchen.
  • Windows 95-Benutzer die Site Windows 95-Downloads besuchen.
(Diese Sites sind auf Englisch.)
