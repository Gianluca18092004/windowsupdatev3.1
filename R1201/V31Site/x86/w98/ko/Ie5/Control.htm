다시 시도

다음 중 하나의 이유로 이 페이지로 이동되었습니다.
  1. Windows Update 사이트 컨트롤 및/또는 아키텍처의 업데이트가 이 오류 메시지의 원인일 수 있습니다. 자세한 내용은 Microsoft Knowledge Base (KB) Q243787 문서를 참조해 주십시오.
  2. 소프트웨어 컨트롤을 다운로드하지 않기로 선택했거나 컨트롤을 다운로드하는 데 문제가 있어 Windows Update 사이트를 사용할 수 없습니다. 지금 컨트롤을 다운로드하려면 위에 있는 다시 시도를 클릭하십시오.
  3. 이 컴퓨터에 대한 관리 권한이 없습니다. Windows Update를 사용하려면 시스템 관리자에게 문의하여 소프트웨어를 업데이트하십시오.
  4. 브라우저의 캐시를 삭제해야 할 수도 있습니다. 자세한 내용은 Microsoft KB Q193385 문서를 참조해 주십시오.
  5. Windows Update 디렉터리를 삭제해야 할 수도 있습니다. 자세한 내용은 Microsoft KB Q243787 문서를 참조해 주십시오.
  6. Windows NT(R) 4.0 Terminal Server Edition용 업데이트는 Windows Update 사이트에서 제공되지 않습니다. 자세한 내용은 Microsoft 기술 자료(KB) 문서 Q323564를 참조하십시오. 이 사이트는 영문 사이트입니다.
