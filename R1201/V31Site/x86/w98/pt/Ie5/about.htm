Acerca do Windows Update
 O Windows Update corresponde a uma extensão online do Windows. De momento, o Windows Update é constituído por duas áreas: o catálogo Actualizações de produtos e a área Informações de suporte. Utilize a barra de navegação que se encontra no lado esquerdo do ecrã para pesquisar estas áreas.

Acerca das Actualizações de produtos
A área Actualizações de produtos corresponde a um catálogo de correcções, actualizações e melhorias para o Windows, assim como para muitos programas que funcionam com o Windows. O catálogo permite-lhe "comprar" componentes que farão que o seu computador trabalhe melhor.

Percorra o catálogo, seleccione o número de componentes pretendidos, transfira-os e instale-os directamente a partir da Web para o seu computador. (Se nunca efectuou transferências a partir da Web antes, recomenda-se que leia a secção "O que é necessário fazer?" abaixo indicada.) Se quiser, o catálogo pode determinar as actualizações e melhorias mais indicadas para o seu computador, com base no software que já tem instalado.

Como funciona?
Quando acede pela primeira vez à página Actualizações de produtos, o Windows Update verifica o sistema para ver o que já está instalado e fornece-lhe uma lista de sugestões de componentes. Esta verificação é efectuada no seu computador para assegurar a segurança e privacidade das informações do seu sistema. Parte alguma desta informação é enviada para a Microsoft ou pela Internet. Esta verificação do sistema garante a obtenção das versões mais recentes e precisas daquilo que quiser transferir do site. Para além disso, não perde tempo a transferir componentes que já tem instalados.

Quando a lista de componentes for apresentada na página, poderá navegá-la, seleccionar os componentes pretendidos e transferi-los directamente para o seu computador.
Certifique-se de que transfere todas as Actualizações fundamentais recomendadas para o seu sistema. As Actualizações fundamentais irão corrigir problemas conhecidos (como, por exemplo, problemas de segurança) específicos do seu computador.

O que é necessário fazer?
As Actualizações de produtos facilitam a transferência de actualizações e melhorias para o software existente no computador.
  1. Em primeiro lugar, pesquise o catálogo.
  2. Quando decidir que pretende um determinado componente, clique na caixa que se encontra ao lado do mesmo. A caixa fica assinalada com uma marca de verificação. Antes de indicar que pretende transferir o componente, clique na hiperligação "Ler isto primeiro" para obter informações importantes, incluindo:
    • Uma descrição mais pormenorizada do componente
    • Como começar a utilizar o componente
    • Instruções para desinstalar
    • Política de suporte
  3. Se chegar à conclusão de que não pretende um dos produtos que já seleccionou, basta voltar a clicar na caixa para remover a marca de verificação.
  4. Quando tiver seleccionado todos os componentes pretendidos, clique no botão Transferir.
  5. Uma página aparece. Certifique-se de que todos os componentes que pretende se encontram na lista. Para efectuar uma mudança, clique no botão Retroceder.
  6. Clique no botão Iniciar transferência. Todos os componentes seleccionados serão transferidos e, em seguida, instalados. Alguns dos componentes poderão solicitar informações adicionais durante a instalação.
Quando aparecer a mensagem Instalação concluída, tal significa que já terminou! Para alguns componentes poderá ser necessário encerrar e reiniciar o computador uma vez concluída a instalação de todos os componentes.

A maior parte dos componentes pode ser desinstalada a partir da página Actualizações de produtos. Para desinstalar:

  1. Vá para a página Actualizações de produtos. Quando for apresentada uma caixa de diálogo, clique em Sim.
  2. Quando for apresentada a página Actualizações de produtos, clique no botão Mostrar actualizações instaladas no início da página.
  3. Clique no botão Desinstalar que se encontra à direita do componente que pretende desinstalar.
