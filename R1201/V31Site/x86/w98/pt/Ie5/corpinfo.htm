É possível remover os ícones do Windows Update que aparecem no menu 'Iniciar'. A desactivação dos ícones do Windows Update impede que os utilizadores acedam ao site do Windows Update (http://windowsupdate.microsoft.com) a partir de qualquer local do produto Windows 98.

Para remover os ícones do Windows Update do menu 'Iniciar':

  1. Remover o ícone do grupo Iniciar/Definições criando a chave de registo HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer para ler: NoWindowsUpdate = 1 (Trata-se de um valor DWORD. Uma definição com o valor 0 permite ícones do Windows Update.)
  2. Reiniciar o computador.
  3. Para remover o ícone do grupo superior da barra 'Iniciar', execute o seguinte a partir da linha de comandos: Wupdmgr.exe -shortcut.
Nota:Só deve editar o registo caso seja absolutamente necessário. Existe um erro no registo; o computador poderá não funcionar correctamente.
