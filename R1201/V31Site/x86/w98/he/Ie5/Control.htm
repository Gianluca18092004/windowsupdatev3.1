הורד את מתקין הרכיבים הקריטיים

הגעת לעמוד זה בשל אחת מהסיבות הבאות:

  1. עידכונים לפקד האתר Windows Update ו/או לארכיטקטורה עלולים לגרום להופעת הודעת שגיאה זו. לקבלת מידע נוסף, קרא את המאמר Microsoft Knowledge Base (KB) Article Q243787.
  2. בחרת לא להוריד את פקדי התוכנה או שאירעה בעיה בהורדת הפקדים, ובמקרה זה חלק גדול מאתר Windows Update לא יהיה זמין לשימושך. אם ברצונך להוריד את הפקדים כעת, נא לחץ על הקישור נסה שוב לעיל.
  3. אין לך הרשאות מנהל עבור מחשב זה. כדי להשתמש ב- Windows Update, עליך לפנות אל מנהל המערכת לשם עדכון התוכנה שלך.
  4. ייתכן שיהיה עליך לנקות את מטמון הדפדפן שלך. לקבלת מידע נוסף, קרא את המאמר Microsoft KB Article Q193385.
  5. ייתכן שיהיה עליך להסיר את ספרית Windows Update. לקבלת מידע נוסף, קרא את המאמר Microsoft KB Article Q243787.
