À propos de Windows Update
 Windows Update est l'extension en ligne de Windows et comprend deux zones : le catalogue de Mise à jour des produits et le Support technique. Pour naviguer dans ces zones, utilisez la barre de navigation située dans la partie gauche de l'écran.

À propos des Mises à jour de produits
La zone Mises à jour de produits constitue un catalogue qui contient des correctifs, des mises à jour et des améliorations concernant Windows et de nombreux autres programmes qui fonctionnent avec Windows. Ce catalogue vous permet de choisir les composants de votre choix pour améliorer le fonctionnement de votre ordinateur.

Parcourez le catalogue, sélectionnez autant de composants de que vous voulez et procédez à leur téléchargement et à l'installation directement depuis le Web sur votre ordinateur. (Si vous n'avez jamais effectué de téléchargement depuis le Web auparavant, lisez le paragraphe « Comment procéder ? » ci-dessous.) Si vous le souhaitez, le catalogue peut déterminer quelles améliorations et mises à jour peuvent être appliquées à votre ordinateur, d'après les logiciels que vous avez déjà installés.

Mode d'emploi
Lorsque vous visitez la page de Mises à jour de produits pour la première fois, Windows Update scanne votre ordinateur pour identifier les logiciels installés et vous propose une liste de composants. Cette opération est effectuée sur votre ordinateur pour garantir la sécurité et la confidentialité des informations relatives à votre ordinateur. Aucune de ces informations n'est divulguée à Microsoft ou sur Internet. Cette vérification de l'ordinateur vous permet de bénéficier des dernières versions mises à jour des composants que vous téléchargez depuis le site. En outre, vous ne perdez pas de temps à télécharger des composants déjà installés.

Lorsque la liste des composants apparaît sur la page, vous pouvez la parcourir , choisir des composants et les télécharger directement sur votre ordinateur.
Veillez à télécharger toutes les Mises à jour critiques recommandées pour votre ordinateur. Ces dernières permettent de corriger les problèmes identifiés qui affectent votre ordinateur, tels que les problèmes relatifs à la sécurité.

Comment procéder
La partie Mises à jour de produits facilite le téléchargement de mises à jour et d'améliorations de logiciels sur votre ordinateur.
  1. Parcourez le catalogue.
  2. Lorsque vous voulez un composant particulier, cochez la case correspondante. Une encoche apparaît dans la case. Avant de télécharger un composant, cliquez sur le lien "Lire en premier" pour consulter des informations importantes, parmi lesquelles :
    • Description détaillée du composant
    • Démarrage du composant
    • Instructions de désinstallation
    • Support technique
  3. Si vous changez d'avis quant à un produit déjà sélectionné, il vous suffit de cliquer de nouveau sur la case à cocher pour supprimer la coche.
  4. Après avoir sélectionné tous les composants souhaités, cliquez sur le bouton Télécharger.
  5. Une page s'affiche. Vérifiez que tous les composants de votre choix apparaissent dans la liste. Pour effectuer une modification, cliquez sur le bouton Retour.
  6. Cliquez sur le bouton Démarrer le téléchargement. Tous les composants sélectionnés se téléchargent et s'installent sur votre ordinateur. Pendant l'installation de certains composants, vous devez éventuellement fournir des informations supplémentaires.
Lorsque le message Installation terminée apparaît, c'est fait ! L'installation de certains composants nécessite le redémarrage de l'ordinateur.

La plupart des composants peuvent être désinstallés directement depuis la page Mises à jour de produits. Procédez comme suit :

  1. Allez à la page Mises à jour de produits. Lorsque la boîte de dialogue s'affiche, cliquez sur Oui.
  2. Dans la page Mises à jour de produits, cliquez sur le bouton Afficher les mises à jour installées en haut de la page.
  3. Cliquez sur le bouton Désinstaller situé à droite du composant à désinstaller.

À propos du support technique
Le Support technique constitue le centre d'information de Windows Update. Ici, vous pouvez obtenir de l'aide en lisant nos pages Forum Aux Questions et Problèmes identifiés ou contacter le support technique de Microsoft.
