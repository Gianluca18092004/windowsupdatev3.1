Windows Update
 
Windows Update est l'extension en ligne de Windows qui vous aide à tirer le meilleur parti de votre ordinateur. Cliquez sur À propos de Windows Update pour en savoir plus.

Mises à jour de produits
 C'est à partir de cette page que vous pouvez télécharger et installer les dernières mises à jour pour votre ordinateur.
