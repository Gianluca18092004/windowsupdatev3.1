KUNDESTØTTE FOR PROGRAMVAREOPPDATERING
Gratis e-poststøtte for Windows Update-området inkluderer: støtte til Web-områdenavigasjon, kvalitet på innhold eller feilmeldinger for Web-området, og grunnleggende hjelp til nedlasting. All annen hjelp, inkludert installasjon og installasjonsproblemer, er tilgjengelig via våre kundestøttealternativer Personal og Professional på Web-området http://support.microsoft.com/directory/. Når det gjelder MSN-produkter, går du til http://help.msn.com/. I noen tilfeller kan det hende du må betale et støttegebyr. Se Web-områdene for detaljer.

Hvis du vil ha informasjon om kundestøtte utenfor USA og Canada, tar du kontakt med Microsofts representasjonskontor i Norge. Hvis du vil se den seneste listen, går du til Web-området Support Online og velger alternativet for Worldwide Support. Hvis det ikke finnes noe representasjonskontor for Microsoft i landet der du er, kontakter du stedet der du kjøpte Microsoft-produktet.

Microsofts støttetjenester varierer med Microsofts gjeldende priser, vilkår og betingelser, og disse kan endres uten varsel.

Hvordan avinstallerer jeg programvareoppdateringer?
Noen oppdateringer kan avinstalleres ved å bruke siden for produktoppdateringer på Web-området Windows Update:
  1. Gå til siden Produktoppdateringer.
  2. Klikk Ja hvis en spørremelding vises.
  3. Når siden Produktoppdateringer vises, klikker du knappen Vis installerte oppdateringer øverst på siden.
  4. Hvis en oppdatering kan avinstalleres, vises en Avinstaller-knapp til høyre for beskrivelsen av oppdateringen.
  5. Klikk Avinstaller ved siden av beskrivelsen du vil avinstallere.
Oppdateringer som ikke kan avinstalleres fra produktoppdateringer, kan kanskje avinstalleres på andre måter. Hvis du vil ha mer informasjon om avinstallering, går du til siden Les dette først for hver oppdatering.

VANLIGE SPØRSMÅL
Hva er Windows Update?
Windows Update er en elektronisk utvidelse til Windows og er laget for at datamaskinen din skal virke og kjøre programmer bedre. Windows Update har en sentral produktoppdateringskatalog som inneholder nye produktforbedringer som systemfiler, enhetsdrivere, oppgraderingspakker og nye Windows-funksjoner. Disse forbedringene blir anbefalt ut fra hva som er nødvendig for akkurat din datamaskin.

Hva er produktoppdateringer?
Produktoppdateringssiden er en katalog over problemløsninger, oppdateringer og forbedringer for Windows og mange Windows-kompatible programmer. Du kan bla gjennom katalogen, velge så mange komponenter du vil, og installere dem på datamaskinen direkte fra World Wide Web.

Hvorfor får jeg melding om at en ActiveX-kontroll på denne siden ikke er sikker?
Denne meldingen vises når sikkerhetsinnstillingen i Internet Explorer er satt til Høyt. For at Web-området Windows Update skal fungere ordentlig, må du endre Web-leserens sikkerhetsinnstillinger for Internett-sone til Middels eller lavere. Dette gjør du ved å klikke Verktøy-menyen og velge Alternativer for Internett. Velg kategorien Sikkerhet og klikk deretter Internett.

Hva er en ActiveX-kontroll?
ActiveX-kontroller er programvarekomponenter som bygger inn ActiveX-teknologi, og som kan brukes om igjen. Disse komponentene kan brukes for å legge til spesielle funksjoner, for eksempel animasjoner eller hurtigmenyer, på Web-sider, i dataprogrammer og i utviklingsverktøy for programvare. Windows Update bruker ActiveX-kontroller til å kontrollere hvilken programvare som er installert på datamaskinen for å kunne gi deg en korrekt liste over oppdateringer og annen programvare du kan ha lyst til å laste ned.

Hvilken oppløsning er best for visning av Windows Update-området?
Web-sidene på Windows Update-området ble laget for visning med en skjermoppløsning på 800 X 600 eller høyere. Du angir denne oppløsningen ved å klikke startknappen, velge Innstillinger og klikke Kontrollpanel. Deretter dobbeltklikker du Vis-ikonet og klikker kategorien Innstillinger.

Overføres informasjon om min datamaskin til Microsoft?
Nei. Hver gang du åpner produktoppdateringskatalogen, laster vi ned en kontroll som ser etter oppdateringer for datamaskinen din. Denne prosedyren utføres lokalt på datamaskinen din for å sikre private data og sikkerhet, og ingen informasjon blir sendt til Microsoft.

Hvordan sender jeg tilbakemelding til Windows Update-avdelingen?
Du kan gi tilbakemelding på dette Web-området til Windows Update-avdelingen ved å klikke Gi tilbakemelding og Windows Update på den svarte verktøylinjen på toppen av alle sidene.

Når en nytt sikkerhetsbulleteng er tilgjengelig på Web-området Microsoft TechNet Security (dette Web-området er på engelsk), er den da også tilgjengelig på Windows Update?
Det er ofte en forsinkelse mellom tidspunktet for når en sikkerhetsbulleteng legges ut på TechNet Security-området, og når sikkerhetsoppdateringen blir tilgjengelig på Windows Update. Dette er fordi nytt innhold legges til Windows Update på et fastlagt tidspunkt. Mens Windows Update gjør alt for gi ut viktige sikkerhetsoppdateringer så raskt som mulig, må nytt innhold testes og bekreftes for å sikre at Windows Update tilbyr deg bare de oppdateringene du trenger for din datamaskin. Vanligvis blir oppdateringer for støttede produkter lagt opp på Windows Update alt fra noen få dager til noen få uker etter at de kunngjøres på TechNet-området. Hvis du vet hvilken versjon av en oppdatering som passer for din datamaskin, og du ikke vil vente med å få den fra Windows Update, kan du laste ned oppdateringen direkte fra Web-siden for sikkerhetsbulletengen.

Hvilket operativsystem og hvilke programmer støttes av Windows Update?
Windows Update tilbyr sikkerhetsoppdateringer som gjelder for Windows 98, Windows 98 Second Edition, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) og Internet Explorer. Du kan imidlertid bare vise oppdateringene som gjelder for din datamaskin. Enkelte serveroperativsystemer og serverbaserte programmer støttes ikke av Windows Update.

Du nevnte at bestemte serverprodukter ikke støttes. Hvilke serverprodukter støttes ikke?
Windows Update begynte som et Web-område som tilbød oppdateringer for hjemmedatamaskiner for Windows-kunder. Etter hvert har Windows Update utvidet tilbudet til oppdateringer som dekker noen serverprodukter (som IIS) som oftere brukes i hjemmet. Sikkerhetsoppdateringer som imidlertid gjelder spesielt for slike serverprodukter som Windows-domenekontrollere, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server og så videre, støttes ikke av Windows Update. Kunder som bruker disse produktene, bør lese gjennom sikkerhetsbulletengene på Web-området for Microsoft TechNet Security hvis de vil ha mer informasjon om hvor de kan laste ned oppdateringer for disse produktene. (Dette Web-området er på engelsk.)

Har Windows Update alltid levert sikkerhetsopprettinger og andre oppdateringer for Internet Information Server 4.0 (IIS 4.0) og Internet Information Services 5.0 (IIS 5.0)?
Nei. 23. mai 2001 tilbød Windows Update en kumulativ pakke med viktige oppdateringer for IIS 5.0 for første gang. Denne pakken inkluderer ikke alle IIS-oppdateringene som er utgitt til nå, men den gir en kombinasjon av de seneste rettelsene som ble utgitt mellom mars 2000 og nå. Hvis du vil ha oppdateringer som ble gitt ut før mars 2000, går du til TechNet-siden for Windows Web-tjenester (IIS). (Dette Web-området er på engelsk.) Når en ny sikkerhetsoppretting eller andre viktige oppdateringer for IIS opprettes, blir denne pakken erstattet med en ny pakke, som vil inneholde både de nye oppdateringene og alle rettelsene som var i pakken fra før. På denne måte vil du vite at samme hvor ofte du besøker Windows Update, så får du alltid de nyeste oppdateringene for IIS-produktene.

Hvor kan jeg gå for å få informasjon om alle sikkerhetsrelaterte oppdateringer?
Du kan se på alle sikkerhetsbulletengene på Web-området Microsoft TechNet Security. Du kan også abonnere på Microsoft Security Notification Service og motta e-postvarslinger om nylig utgitte sikkerhetsbulletenger. (Disse Web-områdene er på engelsk.)
