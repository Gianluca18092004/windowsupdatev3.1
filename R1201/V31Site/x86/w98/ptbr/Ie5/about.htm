Sobre o Windows Update
 O Windows Update é uma extensão on-line do Windows. No momento, o Windows Update consiste em duas áreas: o catálogo 'Atualizações do produto' e a área 'Informações de suporte'. Use a barra de navegação no lado esquerdo da tela para navegar nessas áreas.

Sobre a área 'Atualizações do produto'
A área 'Atualizações do produto' é um catálogo de correções, atualizações e melhorias feitas ao Windows e a vários programas que trabalham com o Windows. O catálogo permite que você "compre" componentes que farão seu computador funcionar melhor.

Navegue pelo catálogo, selecione quantos componentes desejar, faça o download e instale-os diretamente da Web em seu computador. (Se você nunca tiver feito o download de nada na Web, leia antes a seção "Que preciso fazer?" abaixo.) Se você preferir, o catálogo pode determinar as atualizações e melhorias que funcionarão especificamente com o seu computador, com base no software que você já tem instalado.

Como funciona?
Quando você visita a página 'Atualizações do produto' pela primeira vez, o Windows Update verifica o seu sistema para saber o que você tem instalado e fornece uma lista de componentes sugeridos. Essa verificação é feita em seu computador de forma a garantir a segurança e a privacidade de suas informações de sistema. Nenhuma dessas informações são enviadas para a Microsoft ou pela Internet. Essa verificação do sistema garante que você obterá as versões mais atualizadas e precisas de qualquer item deste site que você escolher para download a partir do site. Além disso, você não perderá tempo fazendo o download de componentes que já instalados.

Quando a lista de componentes é exibida na tela, você pode navegar por ela, decidir os componentes que deseja e fazer o download deles diretamente para o seu computador.
Certifique-se de fazer o download de todos as Atualizações críticas recomendadas para o seu sistema. As Atualizações críticas corrigirão problemas conhecidos (como questões de segurança) específicos em seu computador.

Que preciso fazer?
A área 'Atualizações do produto' facilita o download de atualizações e melhorias feitas ao software de seu computador.
  1. Primeiro, navegue pelo catálogo.
  2. Quando decidir que deseja um certo componente, clique na caixa ao lado dele. Aparecerá uma marca de seleção na caixa. Antes de decidir-se pelo download de um componente, clique no link "Ler isto antes" para obter informações importantes, que incluem:
    • Uma descrição mais detalhada do componente
    • Como começar a usar o componente
    • Instruções para a desinstalação
    • Políticas de suporte
  3. Se você decidir que não deseja um produto já selecionado, basta clicar na caixa novamente para remover a marca de seleção.
  4. Quando tiver selecionado todos os componentes desejados, clique no botão Download.
  5. Uma página é exibida. Certifique-se de que todos os componentes desejados estejam na lista. Para fazer alterações, clique no botão Voltar.
  6. Clique no botão Iniciar download. Todos os componentes selecionados serão transferidos e, em seguida, instalados. Alguns componentes podem solicitar informações adicionais durante a instalação.
Quando receber a mensagem Instalação concluída, acabou! Alguns componentes podem pedir para que você encerre e reinicie o computador após a instalação de todos os componentes ser concluída.

A maioria dos componentes pode ser desinstalada diretamente da página 'Atualizações do produto'. Para desinstalar:

  1. Vá para a página 'Atualizações do produto'. Quando a caixa de diálogo for exibida, clique em Sim.
  2. Quando a página 'Atualizações do produto' for exibida, clique no botão Mostrar atualizações instaladas na parte superior da página.
  3. Clique no botão Desinstalar à direita do componente que você deseja desinstalar.

Sobre a área 'Informações de suporte'
A área 'Informações de suporte' é o centro de informações do Windows Update. Aqui, você pode obter ajuda sobre como usar o Windows Update lendo as páginas 'Perguntas freqüentes' e 'Problemas conhecidos' ou contatando o suporte da Microsoft.
