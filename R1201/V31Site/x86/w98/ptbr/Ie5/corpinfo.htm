É possível remover os ícones do Windows Update que aparecem no menu 'Iniciar'. Desativar os ícones do Windows Update evita que os usuários acessem o site Windows Update (http://v31update.vornext.gq) de qualquer lugar dentro do produto Windows 98.

Para remover os ícones do Windows Update do menu 'Iniciar':

  1. Remova o ícone do grupo 'Iniciar/Configurações' criando a chave de registro HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer como: NoWindowsUpdate = 1 (Esse é um valor DWORD. Uma configuração 0 ativa os ícones do Windows Update.)
  2. Reinicie o computador.
  3. Para remover o ícone do grupo mais acima da barra 'Iniciar', execute o seguinte comando a partir da linha de comandos: Wupdmgr.exe -shortcut.
Observação Você não deve editar seu registro, a não ser que seja absolutamente necessário. Se houver um erro em seu registro, seu computador poderá não funcionar corretamente.
