Supportoplysninger
 Hvis du har problemer med webstedet Windows Update, er der hjælp at hente via onlinesupport. Du skal blot vælge en af mulighederne nedenfor.

Ved enkelte af mulighederne kan du blive bedt om at oprette en Microsoft-onlineprofil. Onlineprofilen fungerer som billet, når du modtager Internetbaseret support fra Microsoft.


Support til softwareopdateringer: Få vist supportoplysninger om de softwareopdateringer, der tilbydes på Windows Update.

Support til enhedsdrivere: Få vist supportoplysninger om de enhedsdrivere, der tilbydes på Windows Update.

Ofte stillede spørgsmål: Hvis du har spørgsmål om Windows Update, kan du starte med at se på vores liste over svar.

Kendte problemer: Du kan også se en liste over kendte problemer, der i øjeblikket findes på webstedet Windows Update.

Skriv til os: Du kan skrive kommentarer og forslag i undersøgelsesformularen.

Hvordan afinstalleres softwareopdateringer?
Nogle opdateringer kan afinstalleres ved hjælp af siden Opdateringer på webstedet Windows Update:
  1. Skift til siden Opdateringer.
  2. Klik på Ja, hvis der vises en meddelelsesprompt.
  3. Klik på knappen Vis installerede opdateringer øverst på siden, når siden Opdateringer vises.
  4. Hvis en opdatering kan afinstalleres, vises knappen Afinstaller til højre for beskrivelsen af opdateringen.
  5. Klik på Afinstaller ved siden af beskrivelsen af den opdatering, du vil afinstallere.
Opdateringer, der ikke kan afinstalleres fra Opdateringer, kan afinstalleres ved hjælp af andre metoder. Se siden Bør læses først for hver opdatering for at få yderligere oplysninger om afinstallation.

OFTE STILLEDE SPØRGSMÅL
Hvad er Windows Update?
Windows Update er onlineudvidelsen til Windows 98, som er udviklet for at holde computeren opdateret. Windows Update har et centralt katalog kaldet Opdateringer, der indeholder nye produktforbedringer, f.eks. systemfiler, enhedsdrivere, Service Packs og nye Windows-funktioner. Disse forbedringer anbefales ud fra det specifikke behov, du har på computeren.

Hvad er Opdateringer?
Siden Opdateringer er et katalog med programrettelser, opdateringer og forbedringer til Windows og mange programmer, der kører under Windows. Du kan gennemse kataloget, vælge så mange komponenter, du ønsker, og hente og installere dem direkte fra Internettet.

Hvorfor får jeg en meddelelse om, at et ActiveX-objekt på denne side ikke er sikkert?
Denne meddelelse vises, når sikkerhedsindstillingen i Internet Explorer er angivet til Høj. Webstedet for Windows Update fungerer først korrekt, når du har ændret browserens sikkerhedsindstillinger for Internetzonen til Mellem eller lavere. Dette gør du ved at klikke på menuen Funktioner og derefter klikke på Internetindstillinger. Vælg fanen Sikkerhed, og klik derefter på Internet.

Hvad er et ActiveX-objekt?
ActiveX-objekter er genanvendelige softwarekomponenter, der indeholder ActiveX-teknologien. Disse komponenter kan bruges til at føje særlige funktioner, f.eks. animation eller genvejsmenuer, til websider, desktopprogrammer og værktøjer til programudvikling. Windows Update bruger ActiveX-objekter til at undersøge, hvilken type programmer der er installeret på computeren, så du kan få en komplet liste over opdateringer og andre programmer, du kan være interesseret i at hente.

Hvilken opløsning er bedst til visning af webstedet Windows Update?
Websiderne til webstedet Windows Update er udviklet til at blive vist med en skærmopløsning på 800 x 600 eller højere. Hvis du vil angive denne opløsning, skal du klikke på knappen Start, pege på Indstillinger og derefter klikke på Kontrolpanel. Dobbeltklik derefter på ikonet Skærm, og klik på fanen Indstillinger.

Overføres oplysninger om min computer til Microsoft?
Nej. Hver gang du besøger kataloget Opdateringer, udføres en kontrol, der undersøger, om der er opdateringer til computeren. Denne kontrol udføres lokalt på computeren for at bevare hemmeligholdelsen og sikkerheden af dine data, og der sendes ingen oplysninger til Microsoft.

Hvordan sender jeg feedback til Windows Update-teamet?
Du kan sende feedback om dette websted til Windows Update-teamet ved at klikke på Skriv til os og derefter klikke på Windows Update på den sorte værktøjslinje øverst på hver side.

Når en ny sikkerhedsbulletin er tilgængelig på webstedet Microsoft TechNet Security. (dette websted er på engelsk). er den så også tilgængelig i Windows Update?
Der er ofte en forsinkelse, fra sikkerhedsbulletinen udsendes på webstedet TechNet Security og til sikkerhedsopdateringen bliver tilgængelig i Windows Update. Det er, fordi indholdet føjes til Windows Update efter et skema. Da Windows Update gør alt for at udsende vigtige sikkerhedsopdateringer så hurtigt som muligt, skal nyt indhold afprøves og bekræftes for at sikre, at Windows Update kun tilbyder de opdateringer, du har brug for til computeren. Opdateringer til understøttede produkter vises typisk i Windows Update fra få dage til få uger efter, de annonceres på webstedet for TechNet. Hvis du ved, at en bestemt version af en opdatering gælder for din computer, og du ikke vil vente, indtil du kan hente den fra Windows Update, kan du hente opdateringen direkte på sikkerhedsbulletinens webside.

Hvilke operativsystemer og programmer understøttes af Windows Update?
Windows Update tilbyder sikkerhedsopdateringer, der gælder for Windows 98, Windows 98 Second Edition, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) og Internet Explorer. Du får dog kun vist de opdateringer, der gælder for din computer. Visse serveroperativsystemer og serverbaserede programmer understøttes ikke af Windows Update.

Det nævnes, at visse serverprodukter ikke understøttes. Hvilke serverprodukter understøttes ikke?
Windows Update startede som et websted med opdateringer til Windows-kunders hjemme-pc'er. Windows Update er med tiden udvidet til at omfatte opdateringer, der dækker visse serverprodukter, f.eks. IIS, som nu bruges mere og mere i private hjem. Sikkerhedsopdateringer, der er beregnet til serverprodukter som Windows-domænecontrollere, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server osv., understøttes ikke i Windows Update. Kunder, som bruger disse produkter, kan se sikkerhedsbulletinen på webstedet Microsoft TechNet Security for at få oplysninger om, hvor opdateringer til disse produkter kan hentes. Dette websted er på engelsk.

Har Windows Update altid leveret sikkerhedsrettelser og andre opdateringer til Internet Information Server 4.0 (IIS 4.0) og Internet Information Services 5.0 (IIS 5.0)?
Nej. Den 23. maj 2001 udgav Windows Update for første gang en samlet pakke med vigtige opdateringer til IIS 5.0. Da denne pakke ikke omfatter alle de IIS-opdateringer, der til dato er udsendt, får du en kombination af de seneste rettelser, der er udsendt mellem marts 2000 og nu. Opdateringer, der er udsendt inden marts 2000, finder du på TechNet page for Windows Web services (IIS). Dette websted er på engelsk. Når der kommer nye sikkerhedsrettelser eller andre vigtige opdateringer til IIS, erstattes denne pakke med en ny pakke, som omfatter både nye opdateringer og alle opdateringerne i den forrige pakke. På denne måde ved du, uanset hvor ofte du besøger Windows Update, at du altid får de seneste opdateringer til IIS-produkterne.

Hvor kan jeg få oplysninger om alle sikkerhedsrelaterede oplysninger?
Du kan få vist alle sikkerhedsbulletiner på webstedet for Microsoft TechNet Security. Du kan også abonnere på Microsoft Security Notification Service og modtage e-mail-meddelelser om de sikkerhedsbulletiner, der er udsendt for nylig. Disse websteder er på engelsk.
