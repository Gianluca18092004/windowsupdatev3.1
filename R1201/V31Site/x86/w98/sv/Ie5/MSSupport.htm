Om Windows Update

Den här ikonen visas längst upp på alla sidor på den här webbplatsen. Klicka på ikonen om du vill ha information om hur du använder Windows Update.

SUPPORT FÖR PROGRAMUPPDATERINGAR
Gratis support via e-post för Windows Update omfattar hjälp med navigering, innehåll och felmeddelanden på webbplatsen samt grundläggande hjälp för hämtning. All annan support, inklusive hjälp om installation och inställningar finns tillgänglig via våra supporterbjudanden på http://support.microsoft.com/directory/. För MSN-produkter går du till http://help.msn.com/. I vissa fall tillkommer en supportavgift. Mer information hittar du på webbplatserna.

Om du vill ha supportinformation utanför USA och Kanada kontaktar du ditt lokala Microsoft-kontor. Om du vill ha en aktuell översikt går du till webbplatsen för onlinesupport och väljer alternativet Worldwide Support. Om det inte finns något Microsoft-kontor i ditt land kontaktar du stället där du köpte Microsoft-produkten.

Priser och villkor för Microsofts supporttjänster kan ändras utan föregående varning.

Hur avinstallerar jag programuppdateringar?
Vissa uppdateringar kan avinstalleras direkt från sidan Produktuppdateringar på webbplatsen Windows Update:
  1. Gå till sidan Produktuppdateringar.
  2. Klicka på Ja om ett frågemeddelande visas.
  3. När sidan Produktuppdateringar visas klickar du på Visa installerade uppdateringar högst upp på sidan.
  4. Om uppdateringen kan avinstalleras visas knappen Avinstallera till höger om beskrivningen för uppdateringen.
  5. Klicka på Avinstallera vid beskrivningen för uppdateringen som du vill avinstallera.
Uppdateringar som inte kan avinstalleras från sidan Produktuppdateringar kanske kan avinstalleras på annat sätt. Läs sidan Läs detta först för respektive uppdatering om du vill ha mer information om hur du avinstallerar uppdateringen.

VANLIGA FRÅGOR OCH SVAR
Vad är Windows Update?
Windows Update är ett onlinetillägg för Windows som gör det enkelt för dig att uppdatera ditt system. Windows Update innehåller katalogen Produktuppdateringar med produktförbättringar, t ex systemfiler, drivrutiner, uppdateringar och korrigeringsfiler samt nya Windows-funktioner. Du rekommenderas att installera olika programförbättringar beroende på vad som behövs för just din dator.

Vad är Produktuppdateringar?
Sidan Produktuppdateringar är en katalog med korrigeringsfiler, uppdateringar och programförbättringar för Windows, och många program som körs under Windows. Välj de komponenter du vill ha och hämta och installera dem direkt från webben.

Varför visas ett meddelande om att en ActiveX-kontroll på den här sidan inte är säker?
Meddelandet visas när säkerhetsinställningen i Internet Explorer är satt till Hög. Webbplatsen Windows Update fungerar bäst om säkerhetsinställningen är satt till Normal eller Låg. Om du vill ändra inställningen klickar du först på Verktyg-menyn och sedan klickar du på Internet-alternativ. Välj fliken Säkerhet och klicka sedan på ikonen Internet.

Vad är ActiveX-kontroller?
ActiveX-kontroller är återanvändningsbara programkomponenter med ActiveX-teknik. Komponenterna kan användas för att lägga till särskilda funktioner, t ex animeringar eller pop up-menyer, till webbsidor, datorprogram och programutvecklingsverktyg. Windows Update använder ActiveX-kontroller för att kontrollera vilka program som är installerade på din dator så att du kan få en felfri lista över uppdateringar och annan programvara som du kan hämta.

Vilken skärmupplösning fungerar bäst med Windows Update?
Webbsidorna på Windows Update har utformats för att fungera bäst i upplösningar på 800x600 och högre. Du kan ställa in upplösningen genom att klicka på Start och välja Inställningar. Klicka sedan på Kontrollpanelen. Dubbelklicka på ikonen Bildskärm och välj fliken Inställningar.

Skickas information om min dator vidare till Microsoft?
Nej. Varje gång du öppnar Produktkatalogen hämtas en ActiveX-kontroll som kontrollerar vilka uppdateringar som behövs till din dator. Kontrollen görs lokalt på din dator så att ingen information kan läcka ut. Informationen skickas alltså inte vidare till Microsoft.

Hur lämnar jag mina synpunkter till teamet bakom Windows Update?
Du kan lämna synpunkter om webbplatsen till teamet bakom Windows Update genom att först klicka på Tyck till, och sedan på Windows Update i det svarta verktygsfältet högst upp på varje sida.

När en ny Security Bulletin-artikel finns tillgänglig på webbplatsen Microsoft TechNet Security (webbplatsen är på engelska), finns den då också tillgänglig på Windows Update?
Det dröjer ofta ett tag från det att artikeln publiceras på webbplatsen TechNet tills säkerhetsuppdateringen finns tillgänglig på Windows Update. Detta beror på att nytt innehåll läggs till på Windows Update vid schemalagda tider. Windows Update-teamet anstränger sig för att publicera viktiga säkerhetsuppdateringar så fort som möjligt, men nytt innehåll måste testas och verifieras för att säkerställa att Windows Update endast innehåller de uppdateringar som du behöver till datorn. Vanligtvis publiceras produktuppdateringar på Windows Update inom några dagar eller veckor från det att de presenteras på webbplatsen TechNet. Om du vet vilken version av en uppdatering som gäller för din dator, och om du inte vill vänta tills den finns tillgänglig på Windows Update, kan du hämta uppdateringen direkt från webbsidan med Security Bulletin-artikeln.

Vilka operativsystem och program stöds av Windows Update?
Windows Update tillhandahåller säkerhetsuppdateringar som gäller för Windows 98, Windows 98, andra utgåvan, Windows Millennium (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) och Internet Explorer. Endast de uppdateringar som gäller för din dator visas dock. Vissa serveroperativsystem och serverbaserade program stöds inte av Windows Update.

Ni nämner att vissa serverprodukter inte stöds. Vilka serverprodukter gäller detta?
Windows Update var ursprungligen en webbplats med uppdateringar för Windows-användarnas hemdatorer. Med tiden så har uppdateringsutbudet på Windows Update utökats till att omfatta vissa serverprodukter (t ex IIS) som blir allt vanligare i hemmet. Säkerhetsuppdateringar för serverprodukter, som t ex Windows-domänkontrollanter, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server osv, finns inte på Windows Update. Kunder som använder dessa produkter bör läsa artiklarna om säkerhet på webbplatsen Microsoft TechNet Security om de vill ha information om var de kan hämta uppdateringar för dessa produkter. (Webbplatsen är på engelska.)

Har säkerhetskorrigeringar och andra uppdateringar för Internet Information Server 4.0 (IIS 4.0) och Internet Information Services 5.0 (IIS 5.0) alltid funnits på Windows Update?
Den 23 maj 2001 publicerades för första gången ett paket med viktiga uppdateringar för IIS 5.0 på Windows Update. Paketet innehåller inte alla IIS-uppdateringar som har publicerats, men det innehåller en kombination av de senaste korrigeringarna som har publicerats från mars 2000 och senare. Om du vill ha uppdateringar som publicerades före mars 2000 går du till TechNet-sidan Windows Web Services (IIS). (Webbplatsen är på engelska.) När nya säkerhetskorrigeringar eller andra viktiga uppdateringar för IIS skapas så ersätts det här paketet med ett nytt paket, som innehåller de nya uppdateringarna samt alla korrigeringar i det tidigare paketet. Detta gör att du alltid kan vara säker på att få de senaste uppdateringarna för dina IIS-produkter oavsett hur ofta du besöker Windows Update.

Var kan jag få information om alla uppdateringar som har med säkerhet att göra?
Du kan visa alla Security Bulletin-artiklar på webbplatsen Microsoft TechNet Security. Du kan också prenumerera på Microsoft Security Notification Service och ta emot e-postmeddelanden när nya Security Bulletin-artiklar publiceras. (Webbplatserna är på engelska.)
