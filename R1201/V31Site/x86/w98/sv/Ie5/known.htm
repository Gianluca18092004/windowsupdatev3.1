Tillbaka till Windows Update

Kända problem med Windows Update
 Nedan följer en förteckning över kända problem med webbplatsen Windows Update. Om du har stött på ett fel som inte finns med i listan ber vi dig kontakta oss genom att klicka på Write Us (kontakta oss) i det svarta verktygsfältet och sedan på Windows Update.
  1. Windows Update uppdaterar förteckningen över produktuppdateringar varje gång du besöker webbsidan. På så sätt kan du alltid se i vilket skick din dator befinner sig. Om du till exempel hämtar två uppdateringar och installerar dem, kommer produktuppdateringsförteckningen att uppdateras och de hämtade uppdateringarna tas bort från webbsidan. De installerade uppdateringarna visas fortfarande under Visa tidigare installationer.

  2. Det går inte att använda det vänstra navigeringsfältet så länge som filhämtning pågår. Detta görs för att inte störa filhämtningen. När du har hämtat filen kan du använda navigeringsfältet igen.

  3. Länkarna på hemsidan för Windows Update kommer inte att byta färg när du har klickat på dem.

  4. Webbplatsen är ännu inte helt handikappanpassad med snabbkommandon och liknande.

  5. I sällsynta fall kan din webbläsare hänga sig om du har flera instanser av webbplatsen öppen och söker efter produktuppdateringar samtidigt.

  6. När du har hämtat en komponent bör du inte öppna något nytt webbläsarfönster förrän du har bekräftat rutan som säger att hämtningen har slutförts. Annars visas ett felmeddelande.

  7. I sällsynta fall kan du få ett skriptfel om du ansluter till Intetrnet via modem. Om detta sker kan du försöka att ansluta igen.

  8. Om webbläsarens säkerhetsinställningar är satta till nivån Hög kan du inte få tillgång till förteckningen över produktuppdateringar. Du måste sätta säkerhetsinställningarna till Normal eller Låg. Så här ändrar du säkerhetsinställningarna:
    1. Klicka på Internet-alternativ i menyn Verktyg.
    2. Klicka på fliken Säkerhet.
    3. Välj inställningen Normal.

  9. Endast användare med administratörsbehörighet får installera eller uppdatera programvaran på den här datorn om du kör Windows NT. Omvänt gäller också att de användare som saknar administratörsbehörighet inte kommer att kunna hämta de kontroller som behövs för att visa vissa av webbplatsens sidor.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Webbplatsen Windows Update kan visas på ett annat språk än det som används av din webbläsare eller ditt operativsystem. Om webbläsaren och operativsystemet är på engelska kan det ändå hända att Windows Update visas på franska. Windows Update visas på det språk som förvalts i webbläsarens språkinställningar. Så här gör du för att ändra språkinställning:
    1. Klicka på menyn Verktyg i Internet Explorer.
    2. Välj Internet-alternativ.
    3. Klicka på knappen Språk på fliken Allmänt.
    4. Välj språk och använd knappen Flytta upp för att flytta det valda språket högst upp i språklistan.

  12. Använder du antivirusprogram? Vissa antivirusprogram kan stoppa hämtning och installation av komponenter. Avaktivera därför eventuella antivirusprogram innan du hämtar uppdateringar.

  13. Om du har lagt in en länk till Windows Update som favorit i Internet Explorer kan det hända att Windows Update inte fungerar korrekt. För bästa resultat bör du gå till Windows Update med hjälp av länken menyn Verktyg i Internet Explorer (klicka på Verktyg och på Windows Update), eller skriv v31update.vornext.gq i adressfältet i Internet Explorer.

  14. Om du nyligen har uppdaterat Internet Explorer kan det hända att Windows Update visar uppdateringar som du redan har installerat i en tidigare version av webbläsaren. De uppdateringar som visas nu är särskilt avsedda för den nya webbläsarversionen. Microsoft rekommenderar att du installerar uppdateringarna för den nya webbläsaren så att du alltid har bästa möjliga skydd.

  15. Länken Read More (mer information) finns inte längre på webbplatsen Windows Update. Vill du har mer information om säkerhetsuppdateringarna gör du så här:
    1. Gå till webbplatsen Microsoft Security.
    2. Klicka på Search for Bulletins and Patches.
    3. Skriv in artikelnumret till Microsoft Knowledge Base i sökrutan.
    4. Klicka på Search.

  16. Så här gör du om du vill you ta bort en uppdatering som du har installerat från Windows Update:
    1. Klicka på Start, peka på Inställningar och klicka sedan på Kontrollpanelen. (I Windows XP klickar du på Start och sedan på Kontrollpanelen.)
    2. Klicka på Lägg till/ta bort program.
    3. Klicka på det program som du vill ta bort och klicka på Lägg till/ta bort. (I Windows XP och Windows 2000 klickar du på Ändra/ta bort.)
    Obs! Om uppdateringen inte visas under Lägg till/ta bort program går den inte att ta bort.
