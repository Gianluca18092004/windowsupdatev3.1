Come si disinstallano gli aggiornamenti software?
Alcuni aggiornamenti possono essere disinstallati mediante la pagina Aggiornamenti prodotti del sito Windows Update:
  1. Visualizzare la pagina Aggiornamenti prodotti.
  2. Fare clic sul pulsante Sì se viene visualizzato un messaggio.
  3. Nella parte superiore della pagina Aggiornamenti prodotti fare clic su Mostra aggiornamenti installati.
  4. Se un aggiornamento può essere disinstallato, a destra della descrizione è disponibile il pulsante Disinstalla.
  5. Fare clic su Disinstalla accanto alla descrizione dell'aggiornamento che si desidera disinstallare.
Per informazioni su come disinstallare un aggiornamento che non può essere disinstallato mediante Aggiornamenti prodotti, vedere la pagina Leggimi relativa all'aggiornamento in questione.

DOMANDE FREQUENTI
Che cos'è Windows Update?
Windows Update è l'estensione in linea di Windows ed è stato realizzato per garantire l'aggiornamento costante dei computer degli utenti. Windows Update include un catalogo centrale denominato Aggiornamenti prodotti che elenca gli aggiornamenti disponibili per file di sistema, driver di periferica, service pack e funzioni di Windows. Gli aggiornamenti proposti vengono selezionati di volta in volta in base alle esigenze specifiche del computer dell'utente che si connette al sito.

Che cos'è Aggiornamenti prodotti?
Nella pagina Aggiornamenti prodotti è riportato un catalogo di patch, aggiornamenti e miglioramenti ai prodotti Windows e molti programmi compatibili con Windows. È possibile sfogliare il catalogo, selezionare i componenti desiderati, scaricarli dal Web e installarli direttamente nel computer.

Perché viene visualizzato un messaggio che indica che un controllo ActiveX in questa pagina non è sicuro?
Questo messaggio viene visualizzato quando il livello di protezione in Internet Explorer è impostato su Alto. Per un migliore utilizzo del sito Windows Update, è necessario impostare le opzioni di protezione del browser su Medio o su un'impostazione inferiore. A tale scopo, scegliere Opzioni Internet dal menu Strumenti, selezionare la scheda Protezione e quindi fare clic su Internet.

Che cos'è un controllo ActiveX?
I controlli ActiveX sono componenti software riutilizzabili che contengono tecnologia ActiveX. Tali componenti possono essere utilizzati per aggiungere speciali funzionalità, quali effetti di animazione o menu di scelta rapida, alle pagine Web, alle applicazioni e agli strumenti di sviluppo del software. Windows Update utilizza i controlli ActiveX per rilevare il software installato nel computer e per proporre un elenco di aggiornamenti e altri prodotti software da installare.

Qual è la migliore risoluzione per la visualizzazione del sito Windows Update?
Per una visione ottimale si consiglia di impostare la risoluzione 800 x 600 o superiore. Dal menu Start scegliere Impostazioni e quindi fare clic su Pannello di controllo. Fare doppio clic sull'icona Schermo e quindi fare clic sulla scheda Impostazioni.

È possibile che le informazioni contenute nel computer vengono inviate a Microsoft?
No. Ogni volta che si visita il catalogo Aggiornamenti prodotti, Windows Update esegue un controllo per stabilire quali aggiornamenti sono effettivamente necessari per il computer dell'utente. Questa operazione viene effettuata localmente nel computer per garantire la riservatezza e la protezione dei dati e nessuna informazione viene inviata a Microsoft.

In che modo è possibile inviare i propri commenti e suggerimenti al team di Windows Update?
Commenti e suggerimenti su questo sito possono essere inviati al team di Windows Update facendo clic su Scrivici, quindi su Windows Update sulla barra degli strumenti nera nella parte superiore di ciascuna pagina.

I nuovi Security Bulletin pubblicati sul sito Web Microsoft TechNet Security (informazioni in lingua inglese) sono disponibili anche in Windows Update?
Spesso trascorre del tempo fra la pubblicazione di un Security Bulletin nel sito TechNet Security e la pubblicazione dell'aggiornamento per la protezione su Windows Update. Ciò è dovuto ai tempi programmati per la pubblicazione dei nuovi contenuti in Windows Update. I responsabili di Windows Update si impegnano a rilasciare gli aggiornamenti importanti per la protezione il prima possibile, ma i nuovi contenuti devono essere sottoposti a test e verifiche rigorosi, in modo da proporre agli utenti solo gli aggiornamenti realmente necessari per il computer in uso. Di norma gli aggiornamenti per i prodotti supportati vengono pubblicati su Windows Update in un arco di tempo compreso fra alcuni giorni e alcune settimane dalla pubblicazione sul sito TechNet. Coloro che sanno quale versione di un aggiornamento è applicabile al computer in uso, possono evitare di attenderne la pubblicazione su Windows Update e scaricare l'aggiornamento direttamente dalla pagina Web del Security Bulletin.

Quali sistemi operativi e programmi sono supportati da Windows Update?
Windows Update offre aggiornamenti della protezione che si applicano a Windows 98, Windows 98 Seconda edizione, Windows Millennium Edition (Windows ME), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) e Internet Explorer. Tuttavia, risultano visibili solo gli aggiornamenti che si applicano al computer in uso. Alcuni sistemi operativi server e programmi basati su server non sono supportati da Windows Update.

Quali prodotti server non sono supportati?
Windows Update è nato come sito Web che offriva aggiornamenti per i computer di casa degli utenti. Nel tempo l'offerta è stata ampliata e ora sono disponibili anche aggiornamenti per alcuni prodotti server (quali IIS), che sempre più spesso trovano impiego anche in ambienti non lavorativi. Non sono supportati gli aggiornamenti della protezione specifici di prodotti server quali controller di dominio Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server e così via. Per informazioni sul download degli aggiornamenti per tali prodotti, vedere i Security Bulletin disponibili nel sito Web Microsoft TechNet Security (informazioni in lingua inglese).

In Windows Update sono stati forniti aggiornamenti e correzioni per la protezione per Internet Information Server 4.0 (IIS 4.0) e Internet Information Services 5.0 (IIS 5.0)?
No. Il 23 maggio 2001 su Windows Update è stato pubblicato, per la prima volta, un pacchetto cumulativo di aggiornamenti importanti per IIS 5.0. Questo pacchetto non contiene tutti gli aggiornamenti rilasciati per IIS, bensì è composto da una combinazione delle correzioni più recenti rilasciate a partire da marzo 2000. Per scaricare gli aggiornamenti rilasciati prima di marzo 2000, visitare la pagina di TechNet per i servizi Web di Windows (IIS) (informazioni in lingua inglese). Quando verranno creati nuovi aggiornamenti importanti o correzioni per la protezione per IIS, il pacchetto verrà sostituito da uno nuovo, che conterrà sia i nuovi aggiornamenti, sia tutte le correzioni presenti nel pacchetto precedente. In questo modo gli utenti avranno sempre la certezza di scaricare gli aggiornamenti più recenti per i prodotti IIS, indipendentemente dalla frequenza con cui visitano il sito Windows Update.

Dove posso trovare informazioni su tutti gli aggiornamenti per la protezione?
È possibile visualizzare tutti i Security Bulletin nel sito Web Microsoft TechNet Security. È inoltre possibile abbonarsi al servizio Microsoft Security Notification Service per ricevere, tramite posta elettronica, informazioni sui nuovi Security Bulletin pubblicati (informazioni in lingua inglese).
