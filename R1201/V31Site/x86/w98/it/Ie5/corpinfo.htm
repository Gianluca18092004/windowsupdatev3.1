Informazioni su Windows Update
Solo per i responsabili IT aziendali

Per trovare download e aggiornamenti importanti da distribuire in una rete aziendale, visitare il sito Windows Update Corporate.

È possibile rimuovere le icone di Windows Update che appaiono nel menu di avvio. Disattivando le icone di Windows Update, gli utenti non potranno più accedere al sito Windows Update (http://v31update.vornext.gq) da Windows 98.

Per rimuovere le icone di Windows Update dal menu di avvio:

  1. Per rimuovere l'icona dal gruppo Impostazioni del menu di avvio creare la chiave del Registro di sistema HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer con: NoWindowsUpdate = 1 (Questo è un valore DWORD. L'impostazione 0 attiva le icone di Windows Update.)
  2. Riavviare il computer.
  3. Per rimuovere l'icona dalla parte superiore del menu di avvio, eseguire la seguente istruzione dal prompt dei comandi: Wupdmgr.exe -shortcut.
Nota Evitare di modificare il Registro di sistema a meno che non sia assolutamente necessario. Se il registro contiene un errore, il computer potrebbe non funzionare correttamente.
