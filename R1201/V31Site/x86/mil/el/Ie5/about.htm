Πληροφορίες σχετικά με το Windows Update
 Το Windows Update είναι μια ηλεκτρονική επέκταση των Windows. Αυτήν τη στιγμή, το Windows Update αποτελείται από δύο περιοχές: τον κατάλογο "Ενημερωμένες εκδόσεις προϊόντων" και την περιοχή "Πληροφορίες υποστήριξης". Χρησιμοποιήστε τη γραμμή πλοήγησης στην αριστερή πλευρά της οθόνης σας για να κάνετε αναζήτηση σε αυτές τις περιοχές.

Πληροφορίες σχετικά με τις ενημερωμένες εκδόσεις προϊόντων
Η περιοχή "Ενημερωμένες εκδόσεις προϊόντων" είναι ένας κατάλογος με ενημερώσεις κώδικα, ενημερωμένες εκδόσεις και βελτιώσεις για τα Windows και πολλά προγράμματα που λειτουργούν σε Windows. Ο κατάλογος σας επιτρέπει να "παραγγείλετε" στοιχεία που θα βελτιώσουν τη λειτουργία του υπολογιστή σας.

Κάντε αναζήτηση στον κατάλογο, επιλέξτε όσα στοιχεία θέλετε και κάντε λήψη και εγκατάσταση αυτών των στοιχείων απευθείας από το Web στον υπολογιστή σας. (Εάν είναι η πρώτη φορά που κάνετε λήψη στοιχείων από το Web, διαβάστε την ενότητα "Απαραίτητες ενέργειες", παρακάτω.) Εάν θέλετε, ο κατάλογος μπορεί να προσδιορίσει ποιες ενημερωμένες εκδόσεις και βελτιώσεις είναι κατάλληλες ειδικά για τον υπολογιστή σας, με βάση το ήδη εγκατεστημένο λογισμικό.

Τρόπος λειτουργίας
Όταν επισκέπτεστε πρώτη φορά τη σελίδα "Ενημερωμένες εκδόσεις προϊόντων", το Windows Update σαρώνει το σύστημά σας για να δει τι έχετε εγκαταστήσει και σας παρέχει μια λίστα με προτεινόμενα στοιχεία. Αυτή η σάρωση πραγματοποιείται στον υπολογιστή σας για να εξασφαλιστεί η ασφάλεια και το απόρρητο των πληροφοριών του συστήματός σας. Καμία από αυτές τις πληροφορίες δεν αποστέλλεται στη Microsoft ή στο Internet. Αυτός ο έλεγχος συστήματος εξασφαλίζει ότι θα λάβετε τις πιο ενημερωμένες και κατάλληλες εκδόσεις για οτιδήποτε επιλέγετε να λάβετε από την τοποθεσία. Επιπλέον, δεν θα χάσετε χρόνο λαμβάνοντας στοιχεία που έχετε ήδη εγκαταστήσει.

Όταν η λίστα με τα στοιχεία εμφανίζεται στη σελίδα, μπορείτε να κάνετε αναζήτηση σε αυτήν, να αποφασίσετε ποια στοιχεία θέλετε και να κάνετε λήψη αυτών των στοιχείων απευθείας στον υπολογιστή σας.
Βεβαιωθείτε ότι κάνατε λήψη όλων των Κρίσιμων ενημερώσεων που προτείνονται για το σύστημά σας. Οι "Κρίσιμες ενημερώσεις" θα διορθώσουν γνωστά θέματα (όπως θέματα ασφαλείας) που αφορούν ειδικά τον υπολογιστή σας.

Απαραίτητες ενέργειες
Με την τοποθεσία "Ενημερωμένες εκδόσεις προϊόντων", η λήψη ενημερώσεων και βελτιώσεων για το λογισμικό του υπολογιστή σας γίνεται ευκολότερη.
  1. Αρχικά, κάντε αναζήτηση στον κατάλογο.
  2. Όταν αποφασίσετε ότι θέλετε ένα συγκεκριμένο στοιχείο, κάντε κλικ στο πλαίσιο δίπλα του. Ένα σημάδι εμφανίζεται μέσα στο πλαίσιο. Πριν να επιλέξετε να κάνετε λήψη ενός στοιχείου, κάντε κλικ στη σύνδεση "Βασικές πληροφορίες" για να λάβετε σημαντικές πληροφορίες, που περιλαμβάνουν:
    • Μια λεπτομερέστερη περιγραφή του στοιχείου
    • Τον τρόπο έναρξης της χρήσης του στοιχείου
    • Οδηγίες για την κατάργηση της εγκατάστασης
    • Πολιτική υποστήριξης
  3. Εάν αποφασίσετε ότι δεν θέλετε ένα προϊόν το οποίο έχετε ήδη επιλέξει, απλώς κάντε πάλι κλικ στο πλαίσιο για να καταργήσετε το σημάδι επιλογής.
  4. Όταν έχετε επιλέξει όλα τα στοιχεία που θέλετε, κάντε κλικ στο κουμπί Λήψη.
  5. Εμφανίζεται μια σελίδα. Βεβαιωθείτε ότι όλα τα στοιχεία που θέλετε αναγράφονται στη λίστα. Για να κάνετε μια αλλαγή, κάντε κλικ στο κουμπί Επιστροφή.
  6. Κάντε κλικ στο κουμπί Έναρξη λήψης. Θα γίνει λήψη και στη συνέχεια εγκατάσταση όλων των στοιχείων που επιλέξατε. Κατά την εγκατάσταση ορισμένων στοιχείων, μπορεί να σας ζητηθούν πρόσθετες πληροφορίες.
Όταν εμφανιστεί το μήνυμα Η εγκατάσταση ολοκληρώθηκε, έχετε τελειώσει. Για ορισμένα στοιχεία, αφού ολοκληρωθεί η εγκατάσταση όλων των στοιχείων, ίσως σας ζητηθεί να τερματίσετε τη λειτουργία του υπολογιστή σας και να κάνετε επανεκκίνηση.

Είναι δυνατή η κατάργηση της εγκατάστασης των περισσότερων στοιχείων από τη σελίδα "Ενημερωμένες εκδόσεις προϊόντων". Για να καταργήσετε την εγκατάσταση:

  1. Μεταβείτε στη σελίδα "Ενημερωμένες εκδόσεις προϊόντων". Όταν εμφανιστεί το παράθυρο διαλόγου, κάντε κλικ στην επιλογή Ναι.
  2. Όταν εμφανιστεί η σελίδα "Ενημερωμένες εκδόσεις προϊόντων", κάντε κλικ στο κουμπί Εγκατεστημένες ενημερώσεις που βρίσκεται στο επάνω μέρος της σελίδας.
  3. Κάντε κλικ στο κουμπί Κατάργηση εγκατάστασης στα δεξιά του στοιχείου που θέλετε να καταργήσετε.

Πληροφορίες σχετικά με την Υποστήριξη
Η σελίδα "Πληροφορίες υποστήριξης" είναι το κέντρο πληροφοριών του Windows Update. Εδώ, μπορείτε να λάβετε βοήθεια σχετικά με τη χρήση του Windows Update, διαβάζοντας τις σελίδες "Συνήθεις ερωτήσεις" και "Γνωστά θέματα" ή επικοινωνώντας με την Υποστήριξη της Microsoft.
