關於 Windows Update
 Windows Update 是 Windows 的線上擴充。目前，Windows Update 包含兩區域：產品更新目錄與支援資訊區。請使用您畫面左方的瀏覽列，瀏覽這些區域。

關於產品更新
產品更新區是 Windows 以及許多與 Windows 搭配程式的修正檔與加強功能目錄。此目錄能讓您選配可以使電腦運作更順暢的元件。

瀏覽此目錄、選擇您想使用的元件，然後直接經由 Web 下載並安裝到電腦上。(如果您以前未曾從網站下載過任何東西，您可以閱讀下面的 "我需要做什麼？"。)如果有需要，此目錄可以針對您已經安裝的軟體，判斷出適用於您電腦的修正檔與增強功能。

如何運作？
當您首次蒞臨產品更新網頁時，Windows Update 會掃瞄您的系統，查看您已經安裝的軟體，並顯示建議安裝的元件清單。為確保您系統資訊的安全性與私密性，會在您的電腦上執行此掃瞄。且不會經由 Internet 傳送此資訊至 Microsoft。這項系統檢查能夠確保您從此網站可以取得最新且最適用於您電腦的程式版本。此外，您也不會浪費時間下載您已經安裝的元件。

當網頁上出現元件清單時，您可以瀏覽清單，決定您要的元件，然後將它們下載到您的電腦。
請記得下載對您系統所建議的重大更新。重大更新會針對您的系統，修正已知的問題 (例如安全性問題等)。

我需要做什麼？
產品更新使您得以輕易地下載修正檔與增強功能到您電腦上的軟體。
  1. 首先，瀏覽目錄。
  2. 當您決定下載某元件時，請核選該元件旁的方塊。在該方塊中會出現核取記號。在您選擇下載元件前，請按一下 [重要說明] 的連結，取得重要的資訊，包括：
    • 對該元件更詳盡的描述
    • 如何開始使用該元件
    • 解除安裝的步驟
    • 支援政策
  3. 如果您決定不要已經選取的產品，只要再按一下該方塊，即可移除核取方塊。
  4. 當您已經選擇所有需要的元件後，請按一下 [下載] 按鈕。
  5. 接著會顯示列出清單的網頁。請確定所有想要的元件已列在清單中。要變更，請按 [上一步] 按鈕。
  6. 按一下 [開始下載] 按鈕。會開始下載所有您所選的元件，然後安裝。某些元件在安裝期間可能會要求您提供額外資訊。
當您看見「安裝完成」訊息時，表示已經成功！某些元件在安裝完所有元件後，可能會要求您關閉電腦，然後再重新啟動。

大多數的元件都可以從產品更新網頁處解除安裝。若要解除安裝：

  1. 移至產品更新網頁。當出現對話方塊時，請按一下 [是]。
  2. 出現產品更新網頁時，請按一下位於網頁上方的 [顯示已更新組件] 按鈕。
  3. 在您想要解除安裝的元件旁，按一下 [解除安裝] 按鈕。
