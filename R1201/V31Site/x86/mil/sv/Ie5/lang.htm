Tillbaka till Windows Update

Varför visas webbplatsen Windows Update på ett annat språk?

View this page in English.

På webbplatsen Windows Update används ett antal inställningar för att avgöra vilket språk som webbplatsen ska visas på och vilka komponenter som ska finnas tillgängliga för hämtning.

Det första som avgörs är språk och versionsnummer för operativsystemet och webbläsaren. De komponenter som finns tillgängliga för hämtning på Windows Update är alltid på rätt språk och har rätt versionsnummer för det operativsystem du använder, oavsett på vilket språk du väljer att visa webbplatsen.

I Windows Update avgörs även operativsystemets nationella inställningar för att webbplatsen ska visas på rätt språk. Om du till exempel har ett engelskt operativsystem men vill se den grekiska webbplatsen, visas webbplatsen på grekiska men de komponenter som finns tillgängliga för hämtning är på engelska.

Internet Explorer har även en språkinställningsfunktion som åsidosätter de nationella inställningarna för ditt operativsystem. Om du använder Windows Update (som finns tillgängligt på flera språk), används listan över språk i språkinställningarna i Internet Explorer för att avgöra vilket språk som ska visas. I listan finns en prioritetsordning, så om du har grekiska som första språk och engelska som andra visas Windows Update på grekiska.

Om du vill se Windows Update på ett annat språk justerar du språkinställningarna i Internet Explorer enligt instruktionerna nedan. Observera att de ändringar som görs påverkar alla flerspråkiga webbplatser du besöker.

  1. Klicka på Verktyg-menyn i Internet Explorer.
  2. Klicka på Internet-alternativ.
  3. Klicka på knappen Språk på fliken Allmänt.
  4. Markera det språk du vill använda och klicka på knappen Flytta upp för att placera ditt val längst upp i listan över språk. Klicka sedan på "OK."
  5. Klicka på "OK" i dialogrutan Internet-alternativ och starta om webbläsaren.
  6. Uppdatera Windows Update-sidan.

Why does the Windows Update site appear in a different language?

The Windows Update site detects several settings to determine the language in which the site will be displayed and the language of the components available for download.

The first detection is the language and version of your operating system and browser. Components available for download from Windows Update will always be the correct language and version for your operating system, regardless of the language in which you are viewing the site.

Windows Update also detects your operating system's Regional Settings to determine the language in which the site will be displayed. As an example, if you live in Greece, but have an English operating system, you would see the site in Greek, but the downloads available would be English.

Internet Explorer also has a feature called Accept Language which supercedes the Regional Settings. If you are viewing Windows Update (which is available in multiple languages), Internet Explorer will look to the list of languages in your Language Preference settings to determine which language to display. This is a prioritized list, so if you have Greek as the first language and English as the second, Windows Update will display in Greek.

To see the Windows Update site in a different language, you can adjust your Internet Explorer Language Preference settings as described below. Please note that changes here will affect other multi-language sites that you view.

  1. In Internet Explorer, click the Tools menu.
  2. Click Internet Options.
  3. On the General tab, click the Languages button.
  4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages. Click "OK."
  5. Click "OK" in the Internet Options dialog and restart your browser.
  6. Reload the Windows Update page.
