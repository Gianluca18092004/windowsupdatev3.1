Como se desinstalam as actualizações de software?
Algumas actualizações podem ser desinstaladas utilizando a página Actualizações de produtos no site do Windows Update:
  1. Vá para a página Actualizações de produtos.
  2. Certifique-se de que clica em Sim caso surja uma mensagem.
  3. Quando a página Actualizações de produtos aparecer, clique em Mostrar actualizações instaladas no início da página.
  4. Se for possível desinstalar uma actualização, aparece um botão Desinstalar à direita da descrição da actualização.
  5. Clique em Desinstalar ao lado da descrição da actualização que pretende desinstalar.
As actualizações que não podem ser desinstaladas a partir da página de actualizações de produtos podem ser desinstaladas através de outros métodos. Consulte a página 'Leia isto primeiro' de cada actualização para obter informações adicionais de desinstalação.

Como se desinstalam as actualizações de controladores de dispositivos?
Importante Para desinstalar uma actualização de controlador que tenha transferido a partir do Windows Update, terá que poder reinstalar o controlador original que veio com o dispositivo. Remover um controlador ou uma actualização de controlador sem o substituir fará com que o dispositivo de hardware deixe de funcionar.

y

  1. Clique com o botão direito do rato sobre o ícone O meu computador no ambiente de trabalho do computador.
  2. Clique em Propriedades e, em seguida, clique no separador Gestor de dispositivos.
  3. Clique uma vez para realçar o dispositivo de hardware e, em seguida, clique em Propriedades.
  4. Clique no separador Controlador e, em seguida, clique em Actualizar controlador.
  5. Quando o assistente de actualização de controladores de dispositivo iniciar, seleccione Especificar a localização do controlador (Avançado) e, em seguida, clique em Seguinte.
  6. Seleccione Mostrar uma lista de controladores conhecidos para este dispositivo, de modo a que eu possa escolher um controlador específico e, em seguida, clique em Seguinte.
  7. Clique em Disco e, em seguida, clique em Procurar para localizar o ficheiro .inf para o controlador que pretende instalar (isto irá remover o controlador do Windows Update e substituí-lo pelo controlador original do dispositivo).
Nota Quando voltar ao Windows Update, é possível que lhe seja oferecida novamente a mesma actualização de controlador. Não a instale. Se tiver questões ou problemas ao seguir o procedimento para desinstalar uma actualização de controlador do Windows Update, utilize as hiperligações de suporte fornecidas para contactar um técnico de suporte.

PERGUNTAS MAIS FREQUENTES
O que é o Windows Update?
O Windows Update é a extensão online do Windows, concebida para ajudar a manter o seu computador actualizado. O Windows Update tem um catálogo central denominado Actualizações de produtos que inclui novos melhoramentos de produtos como, por exemplo, ficheiros de sistema, controladores de dispositivos, pacotes de serviços e novas funções do Windows. Estes melhoramentos são recomendados com base no que é necessário para o seu computador específico.

O que significa Actualizações de produtos?
A página Actualizações de produtos é um catálogo de correcções, actualizações e melhoramentos do Windows e de muitos programas que funcionam com o Windows. Pode navegar através do catálogo, seleccionar os componentes que desejar e transferi-los e instalá-los directamente da Web para o seu computador.

Porque é que recebo uma mensagem indicando que um controlo ActiveX nesta página não é seguro?
Esta mensagem é apresentada quando a definição de segurança no Internet Explorer está definida como Alta. Para que o site do Windows Update funcione correctamente, tem de alterar as definições de segurança do browser relativas à zona da Internet para Média ou Baixa. Para o fazer, clique no menu Ferramentas e, em seguida, clique em Opções da Internet. Seleccione o separador Segurança e, em seguida, clique em Internet.

O que é um controlo ActiveX?
Os controlos ActiveX são componentes de software reutilizáveis que incorporam a tecnologia ActiveX. Estes componentes podem ser utilizados para adicionar funcionalidades especializadas tais como, por exemplo, animação ou menus de contexto, a páginas Web, a aplicações do ambiente de trabalho e a ferramentas de desenvolvimento de software. O Windows Update utiliza controlos ActiveX para verificar qual o software instalado no computador de forma a fornecer-lhe uma lista correcta de actualizações e outro software que possa pretender transferir.

Qual é a melhor resolução para visualizar o site do Windows Update?
As páginas Web no Web site do Windows Update foram concebidas para serem visualizadas com uma resolução de ecrã de 800 por 600 ou superior. Para definir esta resolução, clique no botão Iniciar, aponte para Definições e, em seguida, clique em Painel de controlo. Em seguida, clique duas vezes no ícone Monitor (Windows 98) ou Visualização (Windows 2000) e clique no separador Definições.

São transmitidas informações sobre o meu computador à Microsoft?
Não. Sempre que acede ao catálogo Actualizações de produtos, o Windows Update utiliza um controlo para verificar a existência de actualizações específicas para o seu computador. Esta verificação é feita localmente no seu computador de modo a garantir a privacidade e segurança dos seus dados e nenhuma informação é enviada para a Microsoft.

Como posso enviar os meus comentários à equipa do Windows Update?
Pode enviar os seus comentários sobre este site para a equipa do Windows Update clicando em Escreva-nos e, em seguida, em Windows Update na barra de ferramentas preta na parte superior de cada página.

Quando um novo boletim de segurança está disponível no Web site Microsoft TechNet Security, (este site está em inglês). também está disponível no Windows Update?
Ocorre com frequência um desfasamento entre o momento em que um boletim de segurança é colocado no Web site TechNet Security site, e o momento em que a actualização de segurança é disponibilizada no Windows Update. Isto deve-se ao facto de o novo conteúdo do Windows Update ser adicionado em períodos previamente agendados. Apesar de o Windows Update efectuar todos os esforços para editar actualizações de segurança importantes o mais rápido possível, o novo conteúdo tem que ser testado e verificado para garantir que o Windows Update apenas lhe oferece as actualizações que necessita para o seu computador em particular. Normalmente, as actualizações para produtos suportados aparecem no Windows Update alguns dias ou algumas semanas após terem sido anunciadas no site da Microsoft TechNet. Caso saiba qual a versão em particular que se aplica ao seu computador e não quiser esperar até está estar disponível no Windows Update, pode transferir a actualização directamente a partir da página da Web dos boletins de segurança.

Que sistemas operativos e programas são suportados pelo Windows Update?
O Windows Update fornece actualizações de segurança que se aplicam ao Windows 98, Windows 98 Segunda Edição, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Serviços de informação Internet 5.0 (IIS 5.0) e Internet Explorer. No entanto, apenas lhe serão mostradas as actualizações que se aplicam especificamente ao seu computador. Alguns sistemas operativos de servidor e programas de servidor não são suportados pelo Windows Update.

É mencionado que alguns produtos de servidor não são suportados. Que produtos de servidor não são suportados?
O Windows Update começou como um Web site que oferecia actualizações para os computadores domésticos dos utilizadores do Windows. Com o tempo, o Windows Update expandiu a sua oferta de actualizações para cobrir alguns produtos de servidor (como o IIS) que entretanto começaram a ser utilizados mais frequentemente a nível doméstico. No entanto, as actualizações de segurança que são específicas de produtos de servidor tais como controladores de domínio do Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server, etc. não são suportadas no Windows Update. Os clientes que estejam a utilizar estes produtos deverão recorrer aos boletins de segurança no Web site Microsoft TechNet Security para obter informações sobre onde transferir as actualizações para esses produtos. (Este site encontra-se em inglês.)

O Windows Update sempre forneceu actualizações de segurança e outras actualizações para o Internet Information Server 4.0 (IIS 4.0) e os Serviços de informação Internet 5.0 (IIS 5.0)?
Não. Em 23 de Maio de 2001, o Windows Update oferecia um pacote cumulativo de actualizações fundamentais para o IIS 5.0 pela primeira vez. Embora este pacote não inclua todas as actualizações do IIS editadas até esta data, fornece uma combinação das correcções mais recentes que foram editadas desde Março de 2000 até esta data. Para obter actualizações que foram editadas no antes de Março de 2000, vá a TechNet page for Windows Web services (IIS). (Este site encontra-se em inglês.) Quando novas actualizações de segurança ou outras actualizações fundamentais para o IIS forem criadas, este pacote será substituído por um novo pacote, que incluirá não só as as novas actualizações como também todas as correcções presentes no pacote anterior. Deste modo, independentemente da frequência com que visita o Windows Update, saberá que está sempre a receber as actualizações mais recentes para os seus produtos IIS.

Onde posso obter informações acerca de todas as actualizações relacionadas com segurança?
Pode visualizar todos os boletins de segurança no Web site Microsoft TechNet Security. Também pode subscrever o serviço Microsoft Security Notification Service e receber alertas de correio electrónico acerca dos boletins de segurança recentemente editados. (Estes sites encontram-se em inglês.)
