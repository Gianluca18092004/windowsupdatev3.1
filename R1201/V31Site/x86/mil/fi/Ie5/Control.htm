Sivua ei voi näyttää
Yritä uudelleen

Tämä sivu voi tulla näyttöön useasta syystä:
  1. Windows Update -sivuston hallintaan ja arkkitehtuuriin tehdyt päivitykset voivat olla syynä siihen, että olet saanut tämän virhesanoman. Lisätietoja asiasta on Microsoft Knowledge Base (KB) -artikkelissa Q243787.
  2. Valitsit, että ActiveX-komponentteja ei ladata, tai niiden lataamisessa oli ongelmia. Suuri osa Windows Update -sivuston toiminnoista vaatii ActiveX-komponentteja. Jos haluat ladata komponentin nyt, napsauta yllä olevaa Yritä uudelleen -linkkiä.
  3. Sinulla ei ole järjestelmänvalvojan oikeuksia tähän tietokoneeseen. Pyydä järjestelmänvalvojaa päivittämään ohjelmistosi Windows Updaten avulla.
  4. Sinun täytyy ehkä tyhjentää selaimen välimuisti. Lisätietoja asiasta on Microsoft KB -artikkelissa Q193385.
  5. Sinun täytyy ehkä poistaa Windows Update -kansio. Lisätietoja asiasta on Microsoft KB -artikkelissa Q243787.
  6. Windows Update -sivustossa ei ole Windows NT® 4.0 Terminal Server Edition -käyttöjärjestelmän päivityksiä. Lisätietoja on Microsoft Knowledge Base (KB) -artikkelissa Q323564.
