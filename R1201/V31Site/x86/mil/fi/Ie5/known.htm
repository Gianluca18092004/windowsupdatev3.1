Palaa Windows Update -sivulle

Windows Update -sivuston tunnetut ongelmat
 Seuraavassa on lueteltu Windows Update -Web-sivuston tunnettuja ongelmia. Jos esiintyvää ongelmaa ei ole lueteltu seuraavassa, ilmoita siitä meille napsauttamalla mustan työkalurivin Palaute-painiketta ja valitsemalla sitten Windows Update.
  1. Windows Update päivittää Tuotepäivitykset-luettelon aina, kun käyt sivuston tässä osassa. Näin näet tietokoneesi tilan. Jos esimerkiksi lataat kaksi uutta teemaa ja palaat Tuotepäivitykset-luetteloon, näkymä päivittyy etkä näe lataamiasi kahta teemaa saatavilla olevien päivitysten joukossa. Ne näkyvät yhä Näytä asennetut päivitykset -näkymässä, mutta ne näkyvät nyt asennettuina.

  2. Et voi käyttää vasenta selauspalkkia lataamisen aikana. Tämä varmistaa lataamisen onnistumisen. Kun päivitys on ladattu, selauspalkkia voi taas käyttää.

  3. Windows Update -kotisivun linkkien väri ei muutu valinnan jälkeen.

  4. Sivuston tiettyjen osien toiminnot eivät ole täysin käytettävissä näppäimistöllä.

  5. Jos avaat tämän sivuston useaan kertaan yhtä aikaa ja Tuotepäivitykset-ohjain etsii päivityksiä, selain lukkiutuu joksikin aikaa.

  6. Kun lataat päivityksen Tuotepäivitykset-luettelosta, älä avaa uutta ikkunaa, kun Lataaminen vahvistettu -sivu on näytössä, sillä tällöin saat virhesanoman.

  7. Jos Internet-yhteys on muodostettu modeemin kautta, näyttöön saattaa tulla komentosarjavirhe. Jos näin tapahtuu, muodosta yhteys uudelleen.

  8. Jos suojausasetuksen arvo on suuri, et voi käyttää Tuotepäivitykset-luetteloa. Sivuston Tuotepäivitykset-osan käyttöä varten suojausasetus saa olla korkeintaan Normaali. Voit muuttaa suojausasetuksia seuraavasti:
    1. Valitse Työkalut-valikosta Internet-asetukset.
    2. Valitse Suojaus-välilehti.
    3. Valitse Normaali.

  9. Vain käyttäjät, joilla on järjestelmänvalvojan oikeudet, voivat asentaa ohjelmiston tai päivittää sitä tässä NT-tietokoneessa. Vastaavasti käyttäjät, joilla ei ole järjestelmänvalvojan oikeuksia, eivät voi ladata joidenkin sivuston sivujen tarkastelemiseen tarvittavia ohjainkomponentteja.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Windows Update -sivuston kieli saattaa poiketa selaimen tai käyttöjärjestelmän kielestä. Selaimen ja käyttöjärjestelmän kieli voi olla englanti, mutta Windows Update -sivusto näkyy ranskaksi. Windows Update -sivusto näkyy sillä kielellä, joka on määritetty selaimen kieliasetukseksi. Voit muuttaa kieliasetusta seuraavasti:
    1. Avaa Internet Explorerin Työkalut-valikko.
    2. Valitse Internet-asetukset.
    3. Valitse Yleiset-välilehden Kielet-painike.
    4. Valitse haluamasi kieli ja aseta haluamasi kieli kieliluettelon ylimmäksi Siirrä ylös -painikkeen avulla.

  12. Virustentorjuntaohjelmien käyttäjät: Jotkin virustentorjuntaohjelmat saattavat häiritä joidenkin komponenttien lataamista. Poista virustentorjuntaohjelmat käytöstä ennen tuotteiden lataamista Tuotepäivitykset-sivulta.

  13. Internet Explorerin Suosikit-valikossa oleva linkki Windows Update -sivustoon ei ehkä toimi oikein. Pääset parhaiten sivustoon käyttämällä Internet Explorerin Työkalut-valikossa olevaa linkkiä (valitse Internet Explorerin Työkalut-valikosta Windows Update) tai kirjoittamalla v31update.vornext.gq Internet Explorerin osoiteriville.

  14. Jos olet äskettäin päivittänyt Internet Explorer -selaimen uudemmaksi versioksi, saatat nähdä uudelleen edellisen selaimen aikana asennettuja päivityksiä, kun palaat Windows Update -sivustoon. Nämä päivitykset on tarkoitettu selaimen uutta versiota varten. Microsoft suosittelee, että asennat nämä päivitykset myös uuteen selaimeen. Näin varmistat sen, että selaimessa on paras mahdollinen suojaus.

  15. Lisätietoja-linkki ei ole enää käytettävissä Windows Update -Web-sivustossa. Saat lisätietoja Windows Update -Web-sivustossa saatavilla olevista suojauspäivityksistä toimimalla seuraavasti:
    1. Siirry Microsoft Security -Web-sivustoon.
    2. Napsauta Search for Bulletins and Patches -linkkiä.
    3. Kirjoita Search-tekstiruutuun Microsoft Knowledge Base (KB) -artikkelin numero.
    4. Napsauta Search-painiketta.

  16. Jos haluat poistaa Windows Update -Web-sivustosta asentamasi päivityksen, toimi seuraavasti:
    1. Napsauta Käynnistä-painiketta, valitse Asetukset ja valitse sitten Ohjauspaneeli. (Jos käytössä on Windows XP -käyttöjärjestelmä, valitse Käynnistä ja sitten Ohjauspaneeli.)
    2. Kaksoisnapsauta Lisää tai poista sovellus -kuvaketta.
    3. Napsauta kohdetta, jonka haluat poistaa, ja valitse sitten Lisää tai poista. (Jos käytössä on Windows XP- tai Windows 2000 -käyttöjärjestelmä, valitse Muuta tai poista.)
    Huomautus Jos kohde ei näy Lisää tai poista sovellus -ikkunassa, sitä ei voi poistaa tietokoneesta.
