Ohjelmapäivitysten asennuksen poistaminen
Jotkin päivitykset voidaan poistaa Windows Update -sivuston Tuotepäivitykset-sivua käyttämällä seuraavasti:
  1. Siirry Tuotepäivitykset-sivulle.
  2. Vastaa Kyllä mahdollisen sanomaikkunan esittämään kysymykseen.
  3. Kun Tuotepäivitykset-sivu tulee näkyviin, valitse sivun yläosassa oleva Näytä asennetut osat -painike.
  4. Jos päivityksen asennus on mahdollista poistaa, päivityksen kuvauksen oikealla puolella on Poista-painike.
  5. Napsauta sen päivityksen vieressä olevaa Poista-painiketta, jonka asennuksen haluat poistaa.
Jos päivitystä ei voi poistaa Tuotepäivitykset-sivua käyttämällä, sen voi ehkä poistaa jollain muulla tavalla. Saat lisätietoja osan poistamisesta kyseisen osan Lue tämä ensin -sivulta.

USEIN KYSYTYT KYSYMYKSET
Mikä on Windows Update?
Windows Update on Windowsin online-lisäosa, joka auttaa sinua pitämään tietokoneesi ajan tasalla. Windows Updatessa on keskeinen luettelo, jonka nimi on Tuotepäivitykset ja joka sisältää uusia tuotteen parannuksia, kuten järjestelmätiedostoja, laiteohjaimia, Service Packeja ja uusia Windows-toimintoja. Näitä parannuksia suositellaan sen mukaan, mitä kyseisessä tietokoneessa tarvitaan.

Mikä on Tuotepäivitykset?
Tuotepäivitykset-sivu on luettelo, jossa on korjauksia, päivityksiä ja parannuksia Windowsiin ja useisiin Windows-ohjelmiin. Voit selata luetteloa, valita niin monta osaa kuin haluat sekä ladata ja asentaa ne suoraan Webistä tietokoneeseen.

Miksi näyttöön tulee ilmoitus siitä, että tämän sivun ActiveX-komponentti ei ole turvallinen?
Tämä sanoma tulee näyttöön, kun Internet Explorerin suojausasetus on suuri. Jotta Windows Update -sivusto toimisi oikein, selaimen Internet-vyöhykkeen suojausasetuksen on oltava normaali tai pienempi. Voit tehdä tämän valitsemalla Työkalut ja valitsemalla Internet-asetukset. Valitse Suojaus-välilehti ja valitse Internet.

Mikä on ActiveX-komponentti?
ActiveX-komponentit ovat uudelleenkäytettäviä ohjelmistokomponentteja, joissa käytetään ActiveX-tekniikkaa. Näitä komponentteja voidaan käyttää lisäämään eritystoimintoja, kuten animaatiota tai ponnahdusvalikoita, Web-sivuihin, työpöytäsovelluksiin ja ohjelmiston kehitystyökaluihin. Windows Update käyttää ActiveX-komponentteja tarkistamaan, mitä ohjelmistoa tietokoneeseen on asennettu, jotta sivusto voi antaa oikean päivitysluettelon ja luettelon muusta ladattavissa olevasta ohjelmistosta.

Mikä tarkkuus soveltuu parhaiten Windows Update -sivuston tarkastelemiseen?
Windows Update -sivuston Web-sivut on suunniteltu tarkasteltaviksi vähintään tarkkuudella 800 x 600. Voit määrittää tämän tarkkuuden napsauttamalla Käynnistä-painiketta, valitsemalla Asetukset ja valitsemalla sitten Ohjauspaneeli. Kaksoisnapsauta Näyttö-kuvaketta ja valitse Asetukset-välilehti.

Lähetetäänkö tietokoneeni tiedot Microsoftille?
Ei. Aina kun siirryt Tuotepäivitykset-luetteloon, Windows Update tarkistaa tietokoneeseesi liittyvät päivitykset komponentin avulla. Tämä tarkistus tapahtuu tietokoneessasi, mikä varmistaa yksityisyyden ja tietosuojan, eikä tietoja lähetetä Microsoftille.

Kuinka lähetän palautetta Windows Update -ryhmälle?
Voit lähettää palautetta tästä sivustosta Windows Update -ryhmälle napsauttamalla sivun yläosan mustalla työkalurivillä olevaa Palaute-painiketta ja valitsemalla Windows Update.

Kun uusi suojaustiedote on saatavilla Microsoft TechNet Security -sivustossa, (sivusto on englanninkielinen) onko se saatavilla myös Windows Updatessa?
Kestää usein hetken, ennen kuin TechNet Security -sivustoon lisätty suojaustiedote on saatavilla Windows Updatessa. Tämä johtuu siitä, että Windows Updateen lisätään sisältöä tietyin väliajoin. Windows Update pyrkii julkistamaan tärkeät suojauspäivitykset mahdollisimman nopeasti, mutta uusi sisältö pitää testata, jotta Windows Update tarjoaisi sinulle vain sellaisia päivityksiä, joita tietokoneesi tarvitsee. Tavallisesti tuettujen tuotteiden päivitykset tulevat Windows Updateen viimeistään muutaman viikon kuluttua siitä, kun niistä on kerrottu TechNet-sivustossa. Jos tiedät, mikä päivityksen versio sopii tietokoneeseesi etkä halua odottaa, kunnes päivitys tulee Windows Updateen, voit ladata päivityksen suoraan suojaustiedotteen Web-sivulta.

Mitä käyttöjärjestelmiä ja ohjelmia Windows Update tukee?
Windows Updatesta löytyy suojauspäivityksiä Windows 98:aan, Windows 98 Second Editioniin, Windows Millennium Editioniin (Windows Me), Windows NT 4.0:aan, Windows 2000:een, Internet Information Server 4.0:aan (IIS 4.0), Internet Information Services 5.0:aan (IIS 5.0) ja Internet Exploreriin. Sinulle kuitenkin näytetään vain ne päivitykset, jotka soveltuvat tietokoneeseesi. Windows Update ei tue tiettyjä palvelinkäyttöjärjestelmiä ja palvelinpohjaisia sovelluksia.

Joitakin palvelintuotteita ei kuulemma tueta. Mitkä tuotteet ovat tällaisia?
Windows Update oli alkujaan Web-sivusto, josta löytyi päivityksiä Windows-käyttäjien kotikoneisiin. Ajan myötä Windows Update on laajentanut päivitysvalikoimaa sisältämään joitakin usein kotona käytettyjä palvelintuotteita (esimerkiksi IIS). Suojauspäivityksiä, joita käytetään vain palvelintuotteiden (kuten Windows-toimialueiden ohjauskoneiden, Microsoft Exchange Serverin, Microsoft SQL Serverin, Microsoft ISA Serverin) kanssa, Windows Update ei kuitenkaan tue. Asiakkaat saavat tietoja näiden tuotteiden päivitysten lataamisesta Microsoft TechNet -Web-sivustosta. (Sivusto on englanninkielinen.)

Onko Windows Updatessa aina ollut suojauspäivityksiä ja muita päivityksiä Internet Information Server 4.0:aa (IIS 4.0) ja Internet Information Services 5.0:aa (IIS 5.0) varten?
Ei. Windows Updateen tuli ensimmäinen IIS 5.0:n tärkeiden päivitysten paketti 23.5.2001. Tämä paketti ei sisällä kaikkia IIS:ää varten julkistettuja päivityksiä vaan kokoelman uusimpia päivityksiä, jotka on julkistettu maaliskuun 2000 jälkeen. Ennen maaliskuuta 2000 julkistetut päivitykset löytyvät sivustosta TechNet page for Windows Web services (IIS). (Sivusto on englanninkielinen.) Kun IIS:ää varten julkistetaan uusia suojauspäivityksiä tai muita tärkeitä päivityksiä, tämä paketti korvataan uudella paketilla, johon kuuluvat sekä uudet päivitykset että kaikki edellisessä paketissa olleet päivitykset. Näin tiedät aina saavasi uusimmat IIS-tuotteiden päivitykset, kun käyt Windows Updatessa.

Mistä saan tietoja suojauspäivityksistä?
Voit tarkastella kaikkia suojaustiedotteita Microsoft TechNet Security -Web-sivustossa. Voit myös tilata Microsoft Security Notification Service -palvelun, jolloin saat sähköpostitse ilmoituksen julkaistuista suojaustiedotteista. (Sivustot ovat englanninkielisiä.)
