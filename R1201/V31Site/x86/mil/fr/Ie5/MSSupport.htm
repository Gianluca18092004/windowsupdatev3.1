En ce qui concerne les produits MSN, visitez le site http://help.msn.com/. Certains services de support sont payants ; veuillez consulter les sites pour plus de détails.

Comment désinstaller les mises à jour de logiciels ?
Certaines mises à jour peuvent être désinstallées via la page Mises à jour de produits du site Web Windows Update :
  1. Allez à la page Mises à jour de produits.
  2. Cliquez sur Oui si un message s'affiche.
  3. En haut de la page Mises à jour de produits, cliquez sur le bouton Afficher les mises à jour installées.
  4. Si une mise à jour peut être désinstallée, un bouton Désinstaller s'affiche à droite de sa description.
  5. Cliquez sur Désinstaller en regard de la description de la mise à jour à désinstaller.
Certaines des mises à jour ne pouvant être désinstallées via la page Mises à jour de produits peuvent l'être par d'autres méthodes. Veuillez consulter la page Lire en premier de chaque mise à jour pour toute information sur la désinstallation.

Forum Aux Questions
Qu'est-ce que Windows Update ?
Extension en ligne de Windows, Windows Update est conçu pour vous aider à maintenir votre ordinateur à jour. Windows Update possède un catalogue, appelé « Mises à jour de produits », où figurent toutes les améliorations apportées telles que fichiers système, pilotes de périphérique, service packs et nouvelles fonctionnalités Windows. Ces améliorations vous sont recommandées en fonction de la configuration de votre ordinateur.

Qu'appelle-t-on Mises à jour de produits ?
La page Mises à jour de produits est un catalogue de correctifs, de mises à jour et d'améliorations pour Windows et de nombreux programmes fonctionnant sous Windows. Vous pouvez parcourir le catalogue, sélectionner autant de composants que vous le voulez, les télécharger et les installer directement du Web sur votre ordinateur.

Que faire si je reçois un message m'indiquant qu'un contrôle ActiveX de la page visitée n'est pas sûr ?
Ce message s'affiche lorsque votre paramètre de sécurité dans Internet Explorer est défini sur Élevé. Pour que le site de Windows Update fonctionne correctement, vous devez faire passer les paramètres de sécurité du navigateur de la zone Internet à Moyen ou Bas. Pour cela, cliquez sur le menu Outils, cliquez ensuite sur Options Internet. Sélectionnez l'onglet Sécurité et cliquez ensuite sur Internet.

Qu'est-ce qu'un contrôle ActiveX ?
Les contrôles ActiveX sont des composants logiciels réutilisables qui intègrent la technologie ActiveX. Ils peuvent être utilisés pour ajouter des fonctionnalités spéciales, telles que des animations ou des menus contextuels, aux pages Web, aux applications de bureautique et aux outils de développement de logiciels. Windows Update utilise des contrôles ActiveX pour détecter les logiciels installés sur votre ordinateur et vous fournir une liste appropriée de mises à jour et autres logiciels téléchargeables.

Quelle est la meilleure résolution à adopter pour consulter le site Windows Update ?
Les pages Web du site Windows Update ont été conçues pour être visualisées avec une résolution d'écran de 800 x 600 ou supérieure. Pour définir cette résolution, cliquez sur le bouton Démarrer, pointez sur Paramètres, puis cliquez sur Panneau de configuration. Double-cliquez ensuite sur l'icône Affichage, puis cliquez sur l'onglet Paramètres.

Est-ce que les informations concernant mon ordinateur sont transmises à Microsoft ?
Non. Chaque fois que vous consultez le catalogue Mises à jour de produits, Windows Update utilise un contrôle pour déterminer les mises à jour adaptées à votre ordinateur. Cette opération s'effectue au niveau local sur votre ordinateur pour assurer la confidentialité et la sécurité de vos données. Aucune information n'est envoyée à Microsoft.

Comment envoyer mes commentaires à l'équipe de Windows Update ?
Vous pouvez transmettre des commentaires sur ce site en cliquant sur Écrivez-nous, puis sur Windows Update dans la barre d'outils visible dans le haut de la page.

Dès qu'un nouveau bulletin de sécurité sort sur le site Web Microsoft TechNet (ce site est en anglais), est-il également disponible sur Windows Update ?
Il s'écoule toujours un laps de temps entre la publication d'un Bulletin de sécurité sur le site TechNet et sa parution sur le site Windows Update. Ce délai est dû au fait que l'ajout de nouveaux contenus sur Windows Update se fait à intervalles réguliers. Bien que Windows Update mette tout en oeuvre pour sortir le plus rapidement possible les mises à jour de sécurité importantes, il est indispensable de tester et de vérifier les nouveaux contenus afin que Windows Update ne vous propose que des mises à jour adaptées à votre ordinateur. Il faut généralement compter entre quelques jours et quelques semaines après leur publication sur le site TechNet pour que les mises à jour de produits supportés apparaissent sur Windows Update. Si vous connaissez la version de la mise à jour qu'il vous faut et que vous n'avez pas la patience d'attendre sa parution sur Windows Update, rien ne vous empêche de la télécharger directement à partir de la page Web du Bulletin de sécurité.

Quels sont les systèmes d'exploitation et programmes supportés par Windows Update ?
Windows Update propose des mises à jour de sécurité qui portent sur Windows 98, Windows 98 Deuxième Édition, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) et Internet Explorer. Ne vous sont présentées toutefois que les mises à jour qui s'appliquent à votre propre ordinateur. Certains systèmes d'exploitation et programmes serveurs ne sont pas supportés par Windows Update.

Vous venez de dire que certains produits serveurs n'étaient pas supportés. Quels sont-ils ?
À l'origine, Windows Update était un site Web qui proposait des mises à jour aux ordinateurs familiaux de clients Windows. Avec le temps, Windows Update a élargi son offre et propose aujourd'hui des mises à jour concernant certains produits serveur (IIS, par exemple) dont l'utilisation se répand au domicile. En revanche, les mises à jour de sécurité propres à certains produits serveurs tels que les contrôleurs de domaine Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server, etc. ne se trouvent pas sur Windows Update. Les utilisateurs de ces produits doivent consulter les Bulletins de sécurité publiés sur le site Web Microsoft TechNet Security pour savoir où télécharger les mises à jour dont ils ont besoin pour ces produits. (ce site est en anglais).

Le site Windows Update a-t-il toujours proposé des correctifs de sécurité et d'autres mises à jour pour Internet Information Server 4.0 (IIS 4.0) et Internet Information Services 5.0 (IIS 5.0) ?
Non. Ce n'est que le 23 mai 2001 que Windows Update a proposé, pour la première fois, un package cumulatif des mises à jour critiques pour IIS 5.0. Si ce package ne contient pas toutes les mises à jour IIS publiées jusqu'alors, il réunit néanmoins les correctifs les plus récents sortis entre Mars 2000 et maintenant. Pour les mises à jour antérieures à mars 2000, consultez la page TechNet consacrée aux services Web Windows (IIS). (ce site est en anglais). Dès la sortie de nouveaux correctifs de sécurité ou de nouvelles mises à jour critiques relatifs à IIS, ce package sera remplacé par un nouveau qui regroupera les nouvelles mises à jour ainsi que tous les correctifs précédents. De cette façon, quelle que soit la fréquence de vos visites, vous serez sûr d'obtenir les mises à jour les plus récentes pour vos produits IIS.

Où puis-je trouver des informations sur toutes les mises à jour de sécurité existantes ?
Vous pouvez consulter tous les Bulletins de sécurité sur le site Web Microsoft TechNet Security. Vous pouvez également vous abonner à Microsoft Security Notification Service afin d'être averti par courrier électronique de la publication des nouveaux bulletins. (ces sites sont en anglais).
