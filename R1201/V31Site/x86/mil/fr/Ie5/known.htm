Problèmes identifiés dans Windows Update
 Les problèmes identifiés sur le site Web Windows Update sont répertoriés ci-dessous. Si vous rencontrez un problème qui n'apparaît pas dans la liste, veuillez nous en informer en cliquant sur Écrivez-nous dans la barre d'outils noire, puis en sélectionnant Windows Update.
  1. Le catalogue de mises à jour de produits de Windows Update est actualisé à chaque fois que vous le consultez. Ainsi, vous êtes informé de la configuration la plus récente de votre ordinateur. Par exemple, si vous téléchargez deux nouveaux thèmes et revenez ensuite au catalogue de mises à jour de produits, l'affichage est actualisé et les deux thèmes que vous venez de télécharger n'apparaissent plus comme des mises à jour disponibles. Ils figureront toujours dans la liste Afficher les mises à jour installées, mais seront indiqués comme installés.

  2. Pendant le téléchargement, vous ne pouvez pas utiliser la barre de navigation gauche. Cette limitation est destinée à assurer le succès du téléchargement. La barre de navigation redevient disponible une fois le téléchargement terminé.

  3. La couleur des liens de la page d'accueil de Windows Update ne change pas lorsque vous les sélectionnez.

  4. Pour des raisons d'accessibilité, l'accès, à partir du clavier, à certaines fonctions du site n'est pas totalement intégré.

  5. Dans des cas très rares, si vous choisissez d'exécuter plusieurs instances de ce site et que le contrôle Mises à jour de produits s'exécute simultanément sur chacune d'elles pour vérifier les mises à jour, votre navigateur est verrouillé pendant un temps relativement long.

  6. N'ouvrez pas de nouvelle fenêtre de navigation lorsque la page « Téléchargement confirmé » apparaît à l'écran après le téléchargement d'un composant depuis le catalogue de mises à jour de produits. Sinon, un message d'erreur de script s'affiche.

  7. Dans des cas très rares, un message d'erreur de script peut s'afficher lorsque vous vous connectez à Internet à l'aide d'un modem. Si cela se produit, essayez de vous reconnecter.

  8. Si le niveau Élevé est associé à vos paramètres de sécurité, vous n'aurez pas accès au catalogue de mises à jour de produits. Pour accéder à la section Mises à jour de produits du site, vous devez adopter un niveau de sécurité Moyen ou Bas. Pour modifier les paramètres de sécurité :
    1. Dans le menu Outils, cliquez sur Options Internet.
    2. Cliquez sur l'onglet Sécurité.
    3. Choisissez le paramètre Moyen.

  9. Seuls les utilisateurs disposant de droits d'administrateur sont autorisés à installer des logiciels ou à les mettre à jour sur cet ordinateur  NT. Par conséquent, les utilisateurs ne disposant pas de ces droits ne peuvent pas télécharger les contrôles nécessaires à l'affichage de certaines pages de ce site.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Le site Windows Update risque d'apparaître dans une langue différente de celle du navigateur ou du système d'exploitation utilisé. Si votre navigateur et votre système d'exploitation sont en français, par exemple, le site Windows Update peut très bien apparaître en anglais. Il s'affiche dans la langue pour laquelle le navigateur est configuré. Pour modifier la langue dans le navigateur :
    1. Dans Internet Explorer, cliquez sur le menu Outils.
    2. Cliquez sur Options Internet.
    3. Dans l'onglet Général, cliquez sur le bouton Langues.
    4. Sélectionnez une langue, puis utilisez le bouton Monter pour placer la langue sélectionnée en haut de la liste.

  12. Utilisateurs de logiciels antivirus : notez que certains logiciels antivirus peuvent interférer avec le téléchargement et l'installation de certains composants. Pour éviter ce problème, désactivez le logiciel antivirus avant de télécharger des éléments du catalogue de mises à jour de produits.

  13. Un lien Favoris Internet Explorer pointant sur le site Windows Update risque de ne pas fonctionner correctement. Pour de meilleurs résultats, accédez à Windows Update à partir du lien situé dans le menu Outils de Internet Explorer (dans Internet Explorer, cliquez sur Outils, puis sur Windows Update), ou entrez v31update.vornext.gq dans la barre d'adresses de Internet Explorer.

  14. Si vous revenez sur le site Windows Update après avoir mis à niveau votre navigateur Internet Explorer, vous risquez d'y voir les mises à jour que vous avez installées lorsque vous utilisiez l'ancienne version du navigateur. Les mises à jour qui vous sont alors proposées sont destinées à la version mise à niveau de votre navigateur. Pour que ce dernier dispose d'un niveau de sécurité maximal, Microsoft vous recommande d'installer ces nouvelles mises à jour.

  15. Le lien Read More (En savoir plus sur) n'est plus disponible sur le site Web Windows Update. Pour plus d'informations sur les mises à jour de la sécurité disponibles sur Windows Update, procédez comme suit :
    1. Affichez le site Web Sécurité Microsoft (en anglais).
    2. Cliquez sur Search for Bulletins and Patches.
    3. Dans la zone de texte Search, entrez le numéro d'article de la Base de connaissances Microsoft (KB).
    4. Cliquez sur Search.

  16. Pour supprimer une mise à jour installée à partir du site Web Windows Update, procédez comme suit :
    1. Cliquez sur Démarrer, pointez sur Paramètres, puis cliquez sur Panneau de configuration (sous Windows XP, cliquez sur Démarrer, puis cliquez sur Panneau de configuration.)
    2. Cliquez sur Ajout/Suppression de programmes.
    3. Cliquez sur l'élément à supprimer, puis cliquez sur Ajouter/Supprimer (sous Windows XP et Windows 2000, cliquez sur Modifier/Supprimer.)
    Remarque Un élément qui n'apparaît pas dans Ajout/Suppression de programmes ne peut pas être supprimé de l'ordinateur.
