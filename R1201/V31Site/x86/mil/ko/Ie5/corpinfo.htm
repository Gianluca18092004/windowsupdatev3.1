회사 IT 관리자를 위한 정보

배포 가능한 다운로드 또는 회사 네트워크를 위한 중요 업데이트가 필요하십니까? Windows Update Corporate 사이트를 방문하십시오.

시작 메뉴에 나타나는 Windows Update 아이콘을 제거할 수 있습니다. Windows Update 아이콘을 제거하면 Windows 98 제품 내 어느 곳에서도 Windows Update 사이트(http://v31update.vornext.gq)를 액세스할 수 없습니다.

시작 메뉴에서 Windows Update 아이콘을 제거하려면

  1. 레지스트리 키를 작성하여 시작/설정 그룹에서 아이콘을 제거합니다. HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer에서 NoWindowsUpdate = 1로 설정합니다. 이것은 DWORD 값입니다. 0으로 설정하면 Windows Update 아이콘이 나타납니다.
  2. 시스템을 다시 시작합니다.
  3. 시작 표시줄의 상위 그룹에서 아이콘을 제거하려면 명령줄에서 Wupdmgr.exe-바로 가기 키를 실행합니다.
참고:꼭 필요한 경우에만 레지스트리를 편집해야 합니다. 레지스트리에 오류가 있으면 컴퓨터가 제대로 작동하지 않습니다.
