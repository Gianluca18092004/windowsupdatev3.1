Удаление обновлений программ
Некоторые обновления можно удалить с помощью страницы "Обновление приложений" узла Windows Update
  1. Перейдите на страницу "Обновление приложений".
  2. В случае появления диалогового окна с вопросом нажмите кнопку Да.
  3. Нажмите кнопку Показать установленные обновления в верхней части появившейся страницы "Обновление приложений".
  4. Если обновление можно удалить, справа от его описания отобразится кнопка Удалить.
  5. Нажмите кнопку Удалить рядом с описанием обновления, которое требуется удалить.
Возможно, обновления, которые нельзя удалить со страницы "Обновление приложений", невозможно удалить другими способами. Дополнительные сведения об удалении каждого обновления можно найти на соответствующей ему странице Сначала прочтите это.

Удаление обновленных драйверов устройств
Внимание! Чтобы удалить драйвер, установленный с веб-узла Windows Update, необходимо иметь возможность переустановить исходный драйвер, поставляемый с устройством. Удаление драйвера без его замены приведет к отказу устройства.

Использование диспетчера устройств для удаления обновленных драйверов устройств.

  1. Щелкните правой кнопкой мыши значок Мой компьютер на рабочем столе компьютера.
  2. Выберите в меню Свойства, затем щелкните вкладку Диспетчер устройств.
  3. Щелкните название устройства, чтобы его выделить, затем нажмите кнопку Свойства.
  4. Щелкните вкладку Драйвер, затем нажмите кнопку Обновить драйвер.
  5. После запуска мастера обновления драйверов устройств выберите в диалоговом окне Указать расположение драйвера, и нажмите кнопку Далее.
  6. Выберите Вывести список всех известных драйверов для этого устройства, из которого я самостоятельно выберу подходящий, затем нажмите кнопку Далее.
  7. Щелкните Установить с диска, затем нажмите кнопку Обзор, чтобы найти файл .inf для устанавливаемого драйвера (при этом драйвер, установленный с узла Windows Update, будет удален и заменен исходным драйвером устройства).
Примечание. При возвращении к узлу Windows Update, вам, возможно, будет вновь предложен для установки тот же самый драйвер. Не устанавливайте его. При возникновении вопросов или неполадок при удалении обновлений драйверов, установленных с узла Windows Update, используйте ссылки для обращения к специалистам службы технической поддержки.

ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
Что такое Windows Update?
Windows Update — это сетевое расширение Windows, помогающее пользователям быть в курсе последних обновлений установленных на компьютере программ. Основной каталог Windows Update называется "Обновление приложений". В нем содержатся обновления и дополнения продуктов, например, системные файлы, драйверы, сервисные пакеты и новые средства Windows. Вы также получите рекомендации, какие из этих дополнений могут пригодиться для вашего компьютера.

Что такое "Обновление приложений"?
Страница "Обновление приложений" - это каталог исправлений, обновлений и расширений для операционной системы Windows и многочисленных приложений для Windows. Можно просмотреть каталог, выбрать нужные компоненты, затем загрузить и установить их прямо с веб-узла.

Почему появляется сообщение о том, что элемент управления ActiveX на этой странице не является безопасным?
Это сообщение появляется, когда установлен "высокий" уровень безопасности обозревателя Internet Explorer. Для корректной работы Windows Update необходимо установить "средний" или более низкий уровень безопасности в группе "Зона Интернета". Для этого щелкните меню Сервис, затем Свойства обозревателя. Выберите вкладку Безопасность и щелкните значок Интернет.

Что такое элемент управления ActiveX?
Элементы управления ActiveX - это многократно используемые компоненты программного обеспечения, основанные на технологии ActiveX. Эти компоненты позволяют добавлять специальные функции, например, анимацию или всплывающие меню на веб-страницы, в приложения и средства разработки программ. Windows Update использует элементы управления ActiveX для проверки, какое программное обеспечение установлено на данном компьютере, чтобы предоставить полный список предлагаемых для загрузки обновлений и другого программного обеспечения.

Каково наилучшее разрешение экрана для просмотра узла Windows Update?
Веб-страницы узла Windows Update разработаны для просмотра с разрешением экрана 800 на 600 или более высоким. Чтобы установить такое разрешение, нажмите кнопку Пуск, выберите Настройка, затем щелкните Панель управления. Дважды щелкните значок Экран и щелкните вкладку Настройка.

Передается ли в корпорацию Майкрософт информация о моем компьютере?
Нет. Каждый раз, когда вы попадаете в каталог обновления приложений, Windows Update выполняет контрольную проверку обновлений, необходимых для вашего конкретного компьютера. Эта проверка выполняется локально на вашем компьютере, чтобы обеспечить конфиденциальность и безопасность ваших данных. При этом никакая информация в корпорацию Майкрософт не передается.

Как передать отзывы и предложения разработчикам Windows Update?
Отзывы и предложения разработчикам Windows Update можно отправить, щелкнув Пишите нам, затем Windows Update на черной панели инструментов в верхней части каждой страницы.

Когда новый бюллетень безопасности появляется на веб-узле Microsoft TechNet Security Web site (узел на английском языке), доступен ли этот бюллетень на веб-узле Windows Update?
С момента публикации нового бюллетеня на веб-узле TechNet Security site до момента его появления на веб-узле Windows Update обычно проходит некоторое время. Это связано с тем, что новые элементы добавляются на узел Windows Update по утвержденному плану. Создатели узла Windows Update делают все возможное, чтобы важные обновления системы безопасности появлялись здесь как можно раньше, но новые элементы необходимо протестировать и убедиться, что пользователь получит на узле Windows Update обновление, предназначенное именно для его компьютера. Обычно обновления поддерживаемых продуктов появляются на веб-узле Windows Update с интервалом от нескольких дней до нескольких недель с момента их публикации на веб-узле TechNet. Пользователи, знающие, какая именно версия обновления подходит для их компьютеров и не желающие ждать, когда обновление появится на веб-узле Windows Update, могут загрузить обновление непосредственно с веб-страницы бюллетеня безопасности.

Какие операционные системы и программы поддерживаются на веб-узле Windows Update?
На веб- узле Windows Update имеются обновления системы безопасности для операционных систем Windows 98, Windows 98 Second Edition, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, сервера Internet Information Server 4.0 (IIS 4.0), служб Internet Information Services 5.0 (IIS 5.0) и веб-обозревателя Internet Explorer. При посещении веб-узла пользователь видит только те обновления, которые подходят для его компьютера. Веб-узел Windows Update не поддерживает некоторые серверные операционные системы и серверные программы.

Упоминалось, что подерживаются не все серверные продукты. Какие именно серверные продукты не поддерживаются?
Веб-узел Windows Update первоначально предназначался для пользователей домашних компьютеров с операционной системой Windows. Позднее на веб-узле Windows Update появились обновления для некоторых серверных продуктов (например, IIS), которые наиболее часто используются на домашних компьютерах. Однако на узле Windows Update не поддерживаются обновления системы безопасности для таких серверных продуктов как контроллеры доменов Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server, и т.п. Пользователям этих продуктов рекомендуется ознакомиться с бюллетенями безопасности на веб-узле Microsoft TechNet Security Web site, чтобы получить сведения об узлах загрузки обновлений для этих серверных продуктов. (Узел на английском языке.)

Всегда ли на узле Windows Update предлагались обновления системы безопасности и другие исправления для сервера Internet Information Server 4.0 (IIS 4.0) и служб Internet Information Services 5.0 (IIS 5.0)?
Нет. 23 мая 2001 г. на узле Windows Update появился первый накопительный пакет обновлений для служб IIS 5.0. Хотя этот пакет не включал все обновления, выпущенные к этому времени для IIS, он содержал набор новейших исправлений, выпущенных с марта 2000 г. Обновления, выпущенные до марта 2000 г. можно найти на веб- странице TechNet page for Windows Web services (IIS). (Узел на английском языке.) Когда будут выпущены новые исправления системы безопасности и другие ключевые обновления для IIS, этот пакет исправлений будет заменен на новый. Новый пакет будет содержать как вновь выпущенные обновления, так и все исправления и обновления из предыдущего пакета. Таким образом, независимо от того, как часто пользователь посещает веб-узел Windows Update, он всегда может получить здесь новейшие обновления для продуктов IIS.

Где можно получить информацию обо всех обновлениях системы безопасности?
Все бюллетени безопасности можно найти на веб-узле Microsoft TechNet Security Web site. Можно также подписаться на службу Microsoft Security Notification Service (Служба оповещения о вопросах безопасности) и получать по электронной почте уведомления о выпущенных бюллетенях безопасности. (Веб-узлы на английском языке.)
