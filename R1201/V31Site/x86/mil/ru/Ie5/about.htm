О программе Windows Update
 Программа Windows Update — это сетевое расширение Windows. В настоящее время Windows Update состоит из двух разделов: каталога обновления приложений и раздела технической поддержки. Для просмотра этих разделов используйте панель ссылок в левой части экрана.

О программе "Обновление приложений"
Раздел ''Обновление приложений'' — это каталог исправлений, обновлений и расширений для операционной системы Windows и многочисленных приложений для Windows. Каталог позволяет выбрать компоненты, повышающие эффективность работы компьютера.

Просмотрите каталог, выберите желаемые компоненты, затем загрузите и установите их непосредственно с веб-узла на свой компьютер. (Если вы никогда раньше не загружали файлы с веб-узлов, прочитайте приведенный ниже раздел "Использование программы"). Каталог поможет определить, какие обновления и расширения будут работать с вашим компьютером с учетом уже установленного на нем программного обеспечения.

Описание программы
Когда вы впервые попадаете на страницу "Обновление приложений", Windows Update просматривает вашу систему для поиска установленных приложений и создает список предлагаемых компонентов. Просмотр выполняется на вашем компьютере, что обеспечивает безопасность и секретность сведений о вашей системе. Эти сведения не будут отправлены в Майкрософт или в Интернет. Проверка системы гарантирует получение вами самых новых версий выбранных компонентов. Кроме того, вам не нужно будет тратить время на загрузку ранее установленных компонентов.

Когда список компонентов появится на этой странице, его можно будет просмотреть, выбрать нужные компоненты и загрузить их в свой компьютер.
Обязательно загрузите все Ключевые обновления, рекомендуемые для данной системы. С помощью "Ключевых обновлений" ликвидируются обнаруженные ранее неполадки (например, неполадки безопасности), применительно к вашему компьютеру.

Использование программы
С помощью программы "Обновление приложений" можно легко загружать обновления и дополнения программного обеспечения.
  1. Сначала просмотрите каталог.
  2. Выбрав компонент, установите флажок рядом с ним. Перед тем, как выбрать компонент для загрузки, щелкните ссылку "Сначала прочтите это" для получения важных сведений.
    • Подробное описание компонента
    • Рекомендации по использованию компонента
    • Инструкция по удалению компонента
    • Сведения о технической поддержке
  3. Если вы решили не загружать ранее выбранный компонент, снимите расположенный рядом с ним флажок.
  4. Выбрав все нужные компоненты, нажмите кнопку Загрузить.
  5. Убедитесь, что все выбранные компоненты перечислены в списке на появившейся странице. Чтобы изменить выбор, нажмите кнопку Назад.
  6. Нажмите кнопку Начать загрузку. Все выбранные компоненты будут загружены, затем установлены. Для установки некоторых компонентов могут потребоваться дополнительные сведения.
После завершения установки появится сообщение Установка завершена. Возможно, для некоторых компонентов после завершения установки потребуется перезапустить компьютер.

Большинство компонентов можно удалить непосредственно со страницы "Обновление приложений". Для удаления выполните следующие действия.

  1. Перейдите на страницу "Обновление приложений". В появившимся диалоговом окне нажмите кнопку Да.
  2. Нажмите кнопку Показать установленные обновления в верхней части появившейся страницы "Обновление приложений".
  3. Нажмите кнопку Удалить справа от описания компонента, который требуется удалить.

Сведения о технической поддержке
Сведения о технической поддержке содержатся на страницах информационного центра узла Windows Update. Здесь можно получить помощь, прочитав разделы "Часто задаваемые вопросы" и "Замеченные неполадки", или связаться со службой поддержки корпорации Майкрософт.
