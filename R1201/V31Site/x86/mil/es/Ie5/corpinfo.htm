About Windows Update
For corporate IT managers only

Looking for distributable downloads and critical updates for your corporate network? Check out the Windows Update Corporate Site today!

It is possible to remove the Windows Update icons that appear in the Start menu. Disabling the Windows Update icons prevents users from accessing the Windows Update site (http://v31update.vornext.gq) from anywhere within the Windows product.

To remove the Windows Update icons from the Start menu:

Windows 2000 Microsoft Management Consoles' snap-in: "Local Computer Policy" makes it possible to disable Windows Update on the Start menu. Although there are many ways to modify and configure MMC to view policies, the most generic way to configure a new console root is as follows:

  1. Click Start, click Run, and type MMC
  2. From Console, select Add/Remove Snap-in
  3. Click Add... choose Group Policy, click Add, click Close, and then click OK
  4. Navigate down Local Computer Policy to User Configuration, to Administrative Templates, to Start Menu and Taskbar
  5. Double click "Disable and remove links to Windows Update"
