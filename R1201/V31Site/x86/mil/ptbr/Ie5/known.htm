Problemas Conhecidos do Windows Update
 Estes são os problemas conhecidos do site Windows Update na Web. Se você encontrar um problema que não está relacionado abaixo, entre em contato conosco clicando em Comentários na barra de ferramentas preta e clicando em Windows Update.
  1. O Windows Update atualiza o catálogo Atualizações do Produto toda vez que você visita esta área do site. Isso permite que você veja a condição mais atual do seu computador. Por exemplo, se você fizer download de dois novos temas e retornar ao catálogo Atualizações do Produto, a visualização será atualizada e você não verá mais esses temas como atualizações disponíveis. Eles serão exibidos, mas dessa vez como instalados, em Mostrar Atualizações Instaladas.

  2. Não será possível usar a barra de navegação esquerda enquanto um download estiver em andamento. Isso visa garantir que o download seja feito com êxito. Quando o download for concluído, a barra de navegação ficará disponível novamente.

  3. Os links da home page do Windows Update não mudarão de cor depois que você clicar neles.

  4. O acesso pelo teclado a algumas funções do site não está totalmente implementado por motivos de acessibilidade.

  5. Raramente, se você optar por executar várias instâncias deste site e executar simultaneamente o controle Atualizações do Produto para verificar a existência de atualizações, o seu navegador ficará travado por um longo período de tempo.

  6. Depois de fazer download de um componente do catálogo Atualizações do Produto, não abra uma nova janela do navegador enquanto a página de confirmação de download estiver na tela. Caso contrário, uma mensagem de erro de script será exibida.

  7. Raramente, quando a conexão à Internet é feita via modem, você poderá ver um erro de script. Se isso acontecer, tente conectar-se novamente.

  8. Se as configurações de segurança estiverem definidas como Alto, você não conseguirá acessar o catálogo Atualizações do Produto. Para usar a área de Atualizações do Produto do site, é preciso definir as configurações de segurança para Médio ou Baixo. Para alterar as configurações de segurança:
    1. No menu Ferramentas, clique em Opções da Internet.
    2. Clique na guia Segurança.
    3. Escolha a configuração Médio.

  9. Somente usuários com privilégios administrativos podem instalar ou fazer atualização de software neste computador NT. Conseqüentemente, os usuários que não tiverem privilégios administrativos talvez não consigam fazer download dos controles necessários para visualizar algumas páginas deste site.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. O site Windows Update pode aparecer em um idioma diferente do seu navegador ou sistema operacional. Por exemplo, seu navegador e seu sistema operacional podem estar em inglês, mas o site Windows Update pode estar em francês. O site Windows Update é exibido no idioma para o qual a preferência de idioma de seu navegador está configurada. Para alterar essa preferência:
    1. No Internet Explorer, clique no menu Ferramentas.
    2. Clique em Opções da Internet.
    3. Na guia Geral, clique no botão Idiomas.
    4. Selecione o idioma desejado e use o botão Mover para cima para situar sua seleção no alto da lista de idiomas.

  12. Usuários de Software Antivírus: alguns softwares antivírus podem interferir no download e na instalação de alguns componentes. Para evitar essa interferência, desative o software antivírus antes de fazer o download dos itens do catálogo Atualizações do Produto.

  13. Um link no menu Favoritos do Internet Explorer para o Windows Update pode não funcionar corretamente. Para obter melhores resultados, acesse o Windows Update usando o link no menu Ferramentas do Internet Explorer (no Internet Explorer, clique em Ferramentas e em Windows Update), ou digite v31update.vornext.gq na barra Endereço do Internet Explorer.

  14. Se você atualizou o seu navegador Internet Explorer recentemente, quando retornar ao Windows Update poderá ver atualizações que já havia instalado quando estava executando uma versão anterior do navegador. As atualizações oferecidas agora destinam-se especificamente ao navegador atualizado. Para garantir que o seu novo Internet Explorer tenha o mais alto nível de segurança, a Microsoft recomenda que você instale essas atualizações em seu novo navegador.

  15. O link Leia Mais não está mais disponível no site Windows Update. Para obter mais informações sobre as atualizações de segurança disponíveis no Windows Update, siga estas etapas:
    1. Vá até o site Microsoft Security na Web.
    2. Clique em Search for Bulletins and Patches.
    3. Na caixa de texto de pesquisa, digite o número do artigo da Microsoft Knowledge Base (KB).
    4. Clique em Search.

  16. Se você quiser remover uma atualização que instalou a partir do site Windows Update na Web, sigas essas etapas:
    1. Clique em Iniciar, aponte para Configurações e clique em Painel de Controle. (No Windows XP, clique em Iniciar e, em seguida, clique em Painel de Controle.)
    2. Clique em Adicionar ou Remover Programas.
    3. Clique no item que deseja remover e clique em Adicionar/Remover. (No Windows XP e no Windows 2000, clique em Alterar/Remover.)
    Observação: se o item não aparecer em Adicionar ou Remover Programas, ele não poderá ser removido do seu computador.
