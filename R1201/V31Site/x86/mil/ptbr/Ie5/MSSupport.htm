Como desinstalar atualizações de software?
É possível desinstalar algumas atualizações através da página Atualizações do produto no site Windows Update:
  1. Vá para a página Atualizações do produto.
  2. Clique em Sim se uma mensagem for exibida.
  3. Quando a página for exibida, clique em Mostrar atualizações instaladas na parte superior da página.
  4. Se for possível desinstalar uma atualização, o botão Desinstalar será exibido à direita da descrição da atualização.
  5. Clique em Desinstalar ao lado da descrição da atualização que você deseja desinstalar.
Para as atualizações que não puderem ser desinstaladas a partir das Atualizações do produto, há outros métodos que podem ser usados. Para obter mais informações sobre desinstalação, consulte a página Leia primeiro referente a cada atualização.

Como desinstalar as atualizações de driver de dispositivo?
Importante Após desinstalar uma atualização de driver cujo download foi feito a partir do Windows Update, você deve reinstalar o driver original que acompanha o dispositivo. A remoção de um driver ou de uma atualização de driver sem a correspondente substituição interrompe o funcionamento do dispositivo de hardware.

Use o Gerenciador de dispositivos para remover as atualizações de driver de seu computador:

  1. Clique com o botão direito no ícone Meu computador na área de trabalho de seu computador.
  2. Clique em Propriedades e na guia Gerenciador de dispositivos.
  3. Clique uma vez para selecionar o dispositivo de hardware e clique em Propriedades.
  4. Clique na guia Driver e em Atualizar driver.
  5. Quando o Assistente de atualização de driver de dispositivo for iniciado, selecione Especificar o local do driver (avançado) e clique em Avançar.
  6. Selecione Exibir uma lista de drivers conhecidos para este dispositivo, de modo que eu possa escolher um driver específico e clique em Avançar.
  7. Clique em Com disco e em Procurar para localizar o arquivo .inf do driver que você deseja instalar (isso removerá o driver do Windows Update e o substituirá pelo driver original do dispositivo de hardware).
Observação Quando você retornar ao Windows Update, é possível que lhe seja oferecida a mesma atualização de driver novamente. Não a instale. Se você tiver dúvidas ou problemas ao desinstalar a atualização de driver do Windows Update, use os links de suporte para obter ajuda.

PERGUNTAS FREQÜENTES
O que é Windows Update?
O Windows Update é a extensão on-line do Windows, desenvolvida para manter seu computador atualizado. O Windows Update possui um catálogo central chamado Atualizações do produto que inclui novos aprimoramentos do produto, como arquivos de sistema, drivers de dispositivo, service packs e novos recursos do Windows. Recomenda-se usar esses aprimoramentos com base nas necessidades específicas de seu computador.

O que são as Atualizações do produto?
As Atualizações do produto compõem um catálogo de correções, atualizações e aprimoramentos do Windows e de muitos programas que funcionam com o Windows. Você pode navegar pelo catálogo, selecionar quantos componentes desejar, fazer o download e instalá-los diretamente da Web em seu computador.

Por que é exibida uma mensagem dizendo que um controle ActiveX nesta página não é seguro?
Essa mensagem é exibida quando suas configurações de segurança do Internet Explorer estão definidas como Alto. Para que o site Windows Update funcione adequadamente, você deve definir as opções de segurança de seu navegador como Médio ou Baixo. Para isso, clique no menu Ferramentas e, em seguida, clique em Opções da Internet. Selecione a guia Segurança e clique em Internet.

O que é um controle ActiveX?
Os controles ActiveX são componentes de software reutilizáveis que incorporam a tecnologia ActiveX. Esses componentes podem ser usados para adicionar funcionalidade especializada, como animações ou menus pop-up, em páginas da Web, aplicativos e ferramentas de desenvolvimento de software. O Windows Update usa controles ActiveX para verificar quais softwares estão instalados em seu computador e fornecer uma lista correta de atualizações e outros softwares que você pode instalar.

Qual é a melhor resolução para exibir o site Windows Update?
As páginas da Web no site Windows Update foram projetadas para serem exibidas em uma resolução de tela de 800 x 600 ou superior. Para definir essa resolução, clique no botão Iniciar aponte para Configurações e clique em Painel de controle. Clique duas vezes no ícone Vídeo e, em seguida, clique na guia Configurações.

As informações sobre o meu computador são transmitidas para a Microsoft?
Não. Cada vez que você vai para o catálogo Atualizações do produto, o Windows Update usa um controle para verificar a existência de atualizações específicas para o seu computador. Essa verificação é feita localmente para garantir a privacidade e a segurança de seus dados e não envia nenhuma informação à Microsoft.

Como envio meus comentários para a equipe do Windows Update?
Você pode enviar comentário e sugestões sobre este site para a equipe do Windows Update, clicando em Comentários e em Windows Update na barra de ferramentas preta no alto de cada página.

Quando um novo boletim de segurança está disponível no site Microsoft TechNet Security, (site em inglês) também está disponível no Windows Update?
Em geral, há um atraso entre o dia em que um boletim de segurança é colocado no site TechNet Security e quando a atualização de segurança está disponível no Windows Update. Isso ocorre porque novos conteúdos são adicionados ao Windows Update conforme uma agenda. Embora seja feito o máximo de esforço para lançar importantes atualizações de segurança o mais rápido possível no Windows Update, o novo conteúdo deve ser testado e verificado para assegurar que o Windows Update ofereça a você somente atualizações necessárias ao seu computador. Geralmente, as atualizações para produtos com suporte aparecem no Windows Update em alguns dias ou semanas do dia em que foram anunciados no site TechNet. Se você sabe qual versão em particular de uma atualização se aplica ao seu computador e não quer esperar para obtê-la por meio do Windows Update, poderá fazer o download da atualização diretamente na página da Web do boletim de segurança.

Que sistemas operacionais e programas têm suporte no Windows Update?
O Windows Update oferece atualizações de segurança que se aplicam ao Windows 98, Windows 98 Segunda Edição, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000, Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) e Internet Explorer. Entretanto, são exibidas apenas as atualizações que se aplicam ao seu computador. Alguns sistemas operacionais de servidor e programas baseados em servidores não possuem suporte no Windows Update.

Você menciona que alguns produtos de servidor não têm suporte. Que produtos de servidor são esses?
O Windows Update começou como um site que oferecia atualizações para clientes domésticos do Windows. Mais tarde, o Windows Update expandiu sua oferta de atualizações para cobrir alguns produtos de servidor (como IIS) que estavam sendo usados com freqüência em residências. Entretanto, as atualizações de segurança que são específicas para esses produtos de servidor, como controladores de domínio do Windows, Microsoft Exchange Server, Microsoft SQL Server, Microsoft ISA Server e outros, não têm suporte no Windows Update. Os clientes que usam esses produtos devem consultar os boletins de segurança no site Microsoft TechNet Security para obter informações sobre onde fazer o download das atualizações desses produtos. (Site em inglês.)

O Windows Update sempre forneceu correções de segurança e outras atualizações para o Internet Information Server 4.0 (IIS 4.0) e Internet Information Services 5.0 (IIS 5.0)?
Não. Em 23 de maio de 2001, o Windows Update ofereceu o primeiro pacote acumulativo de atualizações críticas para o IIS 5.0. Embora esse pacote não incluísse todas as atualizações do IIS lançadas até então, ele fornecia um conjunto das correções mais recentes lançadas desde março de 2000. Para obter atualizações anteriores ao mês de março de 2000, visite a página da TechNet para os serviços Web do Windows (IIS). (Site em inglês.) Quando novas correções de segurança e outras atualizações críticas para o IIS forem criadas, esse pacote será substituído por um novo pacote, que incluirá tanto as novas atualizações como todas as correções do pacote anterior. Desse modo, independentemente da freqüência com que visita o Windows Update, você sabe que sempre terá as atualizações mais recentes para os produtos IIS.

Onde posso obter informações sobre todas as atualizações relacionadas a segurança?
Você pode ler todos os boletins de segurança no site Microsoft TechNet Security. Você também pode assinar o Microsoft Security Notification Service e receber avisos por email sobre os boletins de segurança recém-lançados. (Sites em inglês.)
