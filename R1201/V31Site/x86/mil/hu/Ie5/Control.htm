Ismét

Ez a lap a következõ okok valamelyike miatt jelent meg:
  1. A hibaüzenet megjelenését valószínûleg a Windows Update webhely vezérlõinek és/vagy szerkezetének változásai okozták. További információkért olvassa el a Microsoft Tudásbázis Q243787-es számú cikkét.
  2. Nem engedélyezte a szoftveres vezérlõk letöltését, vagy hiba történt azok letöltése közben, ezért a Windows Update webhely jelentõs része nem érhetõ el. Ha le kívánja tölteni a vezérlõket, kattintson a fenti Ismét elemre.
  3. Nincsenek felügyeleti jogai ehhez a számítógéphez. A Windows Update használatához kérje a rendszergazdától a szoftver frissítését.
  4. Lehet, hogy ürítenie kell a böngészõ gyorsítótárát. További információkért olvassa el a Microsoft Tudásbázis Q193385-ös számú cikkét.
  5. Lehet, hogy el kell távolítania a Windows Update könyvtárat. További információt a Microsoft Tudásbázis Q243787-es számú cikke tartalmaz.
  6. A Windows NT® 4.0 Terminal Server Edition frissítései a Windows Update webhelyen nem érhetõk el. További információ a Microsoft Tudásbázis Q323564-es számú cikkében található.
