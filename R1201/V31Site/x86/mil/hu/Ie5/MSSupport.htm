GYAKRAN ISMÉTELT KÉRDÉSEK
Mi a Windows Update?
A Windows Update a Windows internetes kiterjesztése, mellyel számítógépét naprakészen tarthatja. A Windows Update Termékfrissítések nevû központi katalógusa tartalmazza az új termékfejlesztéseket, például rendszerfájlokat, eszközillesztõket, szervizcsomagokat és a Windows új szolgáltatásait. Ezeknek a fejlesztéseknek a használatát az Ön által használt számítógép jellemzõi alapján ajánljuk.

Mi a Termékfrissítések lap?
A Termékfrissítések lap a Windows javításait, frissítéseit és fejlesztéseit, valamint számos egyéb, a Windows rendszerrel használható programot tartalmazó katalógus. A katalógusban tallózhat, kijelölhet tetszés szerinti számú összetevõt, majd közvetlenül a webrõl a számítógépre töltheti le és telepítheti azokat.

Miért jelenik meg olyan üzenet, amely szerint az ActiveX vezérlõ ezen a lapon nem biztonságos?
Ez az üzenet akkor jelenik meg, ha az Internet Explorer biztonsági szintje a Magas értékre van beállítva. A Windows Update webhely csak akkor mûködik megfelelõen, ha Közepes vagy alacsonyabb szintûre módosítja a böngészõben az internetzóna biztonsági beállításait. Ehhez kattintson az Eszközök menü Internet-beállítások parancsára, kattintson az Adatbiztonság fülre, majd az Internet ikonra.

Mi az ActiveX vezérlõ?
Az ActiveX vezérlõk ActiveX technológiával készült újrafelhasználható szoftverösszetevõk. Ezek az összetevõk teszik lehetõvé különleges funkciók, például animáció vagy felbukkanó menük hozzáadását a weblapokhoz, asztali alkalmazásokhoz és szoftverfejlesztõ eszközökhöz. A Windows Update a számítógépen telepített szoftverek ellenõrzésére ActiveX vezérlõket használ, így pontos listát tud összeállítani a szükség szerint letölthetõ frissítésekrõl és egyéb szoftverekrõl.

Melyik a legmegfelelõbb képernyõfelbontás a Windows Update webhely megjelenítéséhez?
A Windows Update webhely lapjait 800 x 600-as vagy nagyobb felbontásban történõ megjelenítésre tervezték. A megfelelõ felbontás beállításához kattintson a Start gombra, mutasson a Beállítások pontra, majd kattintson a Vezérlõpult parancsra. Ezután kattintson duplán a Képernyõ ikonra, végül kattintson a Beállítások fülre.

Kap a Microsoft adatokat a számítógéprõl?
Nem. A Termékfrissítések webhely felkeresésekor a Windows Update egy vezérlõvel ellenõrzi az Ön számítógépéhez szükséges frissítéseket. Ez az ellenõrzés lokálisan, az Ön számítógépén történik az adatok bizalmasságának és védelmének biztosítása érdekében. A Microsoft ezen a módon nem kap semmilyen információt a számítógéprõl.

Hogyan lehet visszajelzést küldeni a Windows Update készítõinek?
Ha visszajelzést szeretne küldeni a Windows Update készítõinek, a lapok tetején található fekete eszköztáron kattintson az Írjon nekünk, majd a Windows Update pontra.

A Microsoft TechNet Security webhelyen (a webhely angol nyelvû) megjelenõ közlemények a Windows Update webhelyen is megjelennek?
Gyakori a késés a TechNet Security webhelyen megjelenõ közlemények és a hozzájuk kapcsolódó adatvédelmi frissítések Windows Update webhelyen való megjelenése között. Ennek oka az, hogy a Windows Update webhely tartalma egy elõre meghatározott rend szerint frissül. A Windows Update készítõi mindent megtesznek azért, hogy a fontos adatvédelmi javítások minél korábban elkészüljenek, az új tartalmakat azonban tesztelni és ellenõrizni kell annak biztosítása érdekében, hogy a Windows Update csak az adott számítógépnek megfelelõ frissítéseket ajánlja fel. A támogatott termékek frissítései általában néhány naptól néhány hétig terjedõ idõszakon belül jelennek meg a Windows Update webhelyen attól az idõponttól számítva, amikor azok be lettek jelentve a TechNet webhelyen. Ha tudja, hogy egy frissítés mely verziójára van szüksége, és nem akarja megvárni, amíg az elérhetõvé válik a Windows Update webhelyen, közvetlenül is letöltheti a frissítést a Security bulletin webhelyrõl.

Milyen operációs rendszereket és termékeket támogat a Windows Update webhely?
A Windows Update webhely a Windows 98, Windows 98 második kiadás, Windows Millennium Edition (Windows Me), Windows NT 4.0, Windows 2000 rendszerekhez, valamint az Internet Information Server 4.0 (IIS 4.0), Internet Information Services 5.0 (IIS 5.0) és az Internet Explorer programokhoz tartalmaz adatvédelmi frissítéseket. A webhely csak azokat a frissítéseket jeleníti meg, amelyek az adott számítógéphez készültek. Egyes kiszolgálói operációs rendszereket és kiszolgáló alapú programokat a Windows Update nem támogat.

Említik, hogy egyes kiszolgáló termékeket a webhely nem támogat. Melyek ezek?
A Windows Update webhely kezdetben az otthoni Windows-felhasználók számítógépeinek támogatására jött létre. Idõvel egyes kiszolgálói termékek is bekerültek a támogatott termékek körébe, amelyek otthoni használata egyre gyakoribb (például az IIS), a Windows Update azonban nem tartalmaz frissítéseket az olyan kiszolgálói termékekhez, mint például a Windows tartományvezérlõk, a Microsoft Exchange Server, a Microsoft SQL Server, a Microsoft ISA Server stb. Az e termékeket alkalmazó felhasználók a Microsoft TechNet Security webhelyen található adatvédelmi közleményekbõl szerezhetnek információkat a termékekhez készült frissítések letöltésére vonatkozóan. (A webhely angol nyelvû.)

A Windows Update webhelyen mindig hozzáférhetõek voltak az Internet Information Server 4.0 (IIS 4.0) és Internet Information Services 5.0 (IIS 5.0) programokhoz készült adatvédelmi javítások és más frissítések?
Nem. A Windows Update elsõként 2001. május 23-án tette közzé az IIS 5.0-hoz készült fontos frissítéseket tartalmazó csomagot. A csomag nem tartalmazza az IIS-hez addig kiadott összes frissítést, azonban megtalálhatóak benne az újabb, 2000 márciusa óta kiadott frissítések. A 2000 márciusa elõtt kiadott frissítések letöltéséhez látogassa meg a Windows Web Services (IIS) lapot a TechNet webhelyen. (A webhely angol nyelvû.) Ha az IIS-hez új adatvédelmi javítások vagy más fontos frissítések jelennek meg, a csomagot egy új váltja fel, amely az addig meglévõkön túl az új frissítéseket is tartalmazza. Így mindig biztos lehet abban, hogy a legújabb frissítéseket tölti le az IIS termékekhez attól függetlenül, hogy milyen gyakran látogatja meg a Windows Update webhelyet.

Hol juthatok információkhoz az adatvédelemmel kapcsolatos frissítésekrõl?
A Microsoft TechNet Security webhelyen az összes adatvédelemmel kapcsolatos közleményt elolvashatja, ha pedig elõfizet a Microsoft Security Notification Service (Microsoft adatvédelmi értesítés) szolgáltatásra, e-mailben kap értesítéseket a megjelenõ adatvédelmi közleményekrõl. (A webhelyek angol nyelvûek.)
