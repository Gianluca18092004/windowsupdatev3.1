מידע תמיכה
 אם נתקלת בבעיות באתר האינטרנט של Windows Update, תמיכה מקוונת מספקת לך סיוע. עליך רק לבחור באחת האפשרויות הבאות.

בעת בחירת חלק מאפשרויות תמיכה אלה, ייתכן כי תתבקש ליצור "פרופיל מקוון" של Microsoft. הפרופיל המקוון ישמש ככרטיס שלך לקבלת תמיכה מ- Microsoft באינטרנט.


תמיכה עבור עדכוני תוכנה: עיין במידע אודות תמיכה עבור עדכוני התוכנה המוצעים באתר Windows Update.

תמיכה עבור מנהלי התקנים: עיין במידע אודות תמיכה עבור מנהלי ההתקנים המוצעים באתר Windows Update.

תמיכה עבור Windows 98: גלה את אפשרויות התמיכה עבור מערכת ההפעלה Windows 98.

שאלות נפוצות: אם יש לך שאלה אודות Windows Update, כדאי לך לעיין תחילה ברשימת התשובות שלנו.

נושאים מוכרים: כדאי לך גם לעיין ברשימת הנושאים המוכרים הקיימים כעת באתר Windows Update.

כתוב לנו: טופס השאלון המהיר שלנו מספק מקום להערות ולהצעות שלך.

תמיכה עבור עדכוני תוכנה
תמיכה בדואר אלקטרוני עבור אתר Windows Update ללא תשלום כוללת: עזרה בניווט באתר, איכות תוכן האתר או הודעות שגיאה וכן סיוע בסיסי בהורדת רכיבים. כל תחומי הסיוע האחרים, כולל נושאי התקנה והגדרה, זמינים באמצעות תוכניות התמיכה מסוג Personal ו- Professional שאנו מציעים ב- http://support.microsoft.com/directory/‎. עבור מוצרי MSN, בקר בכתובת http://help.msn.com/‎. במקרים מסוימים ייתכן שתחויב בתשלום עבור התמיכה, נא עיין באתרים לקבלת פרטים.

לקבלת מידע אודות תמיכה מחוץ לתחומי ארצות הברית וקנדה, פנה אל הסניף המקומי. כדי לצפות ברשימת הסניפים המעודכנת ביותר, נא עיין באתר התמיכה המקוונת ובחר באפשרות של תמיכה ברחבי העולם. אם לא קיים משרד סניף של Microsoft במדינתך, נא פנה אל הארגון שממנו רכשת את מוצר Microsoft שברשותך.

שירותי התמיכה של Microsoft כפופים למחירים, לתנאים ולתקנות של Microsoft הנמצאים בתוקף בעת השימוש בשירות, והם עשויים להשתנות ללא הודעה מוקדמת.

כיצד ניתן להסיר עדכוני תוכנה?
ניתן להסיר עדכונים מסוימים באמצעות דף עדכוני המוצרים באתר Windows Update:
  1. עבור אל דף עדכוני המוצרים.
  2. הקפד ללחוץ על כן אם מופיעה הודעה.
  3. כאשר דף עדכוני המוצרים מופיע, לחץ על הצג עדכונים מותקנים בראש הדף.
  4. אם ניתן להסיר את העדכון, לחצן הסר מופיע לצד תיאור העדכון.
  5. לחץ על הסר לצד תיאור העדכון שברצונך להסיר.
לגבי עדכונים שאין אפשרות להסירם מתוך דף עדכוני המוצרים, ייתכן שניתן יהיה להסיר אותם בדרכים אחרות. עיין בדף קרא זאת תחילה עבור כל עדכון לקבלת מידע נוסף אודות הסרת ההתקנה.

שאלות נפוצות
מהו Windows Update?
Windows Update הוא השלוחה המקוונת של Windows, המיועדת לסייע לך לעדכן את המחשב שלך באופן שוטף. Windows Update כולל קטלוג מרכזי בשם 'עדכוני מוצרים', המכיל שיפורי מוצרים חדשים כגון קבצי מערכת, מנהלי התקנים, ערכות שירות ואפשרויות חדשות עבור Windows. המלצות על שיפורים אלה ניתנות לך בהתאם לצרכים של המחשב הספציפי שברשותך.

מהו דף עדכוני המוצרים?
דף עדכוני המוצרים הוא קטלוג של תיקונים, עדכונים ושיפורים ל- Windows ולתוכניות רבות הפועלות עם Windows. באפשרותך לעיין בקטלוג, לבחור את כל הרכיבים הרצויים לך, ולהוריד ולהתקין אותם ישירות מהאינטרנט למחשב שלך.

מדוע מופיעה הודעה האומרת כי פקד ActiveX בדף זה אינו בטוח?
הודעה זו מוצגת כאשר הגדרות האבטחה ב- Internet Explorer מוגדרות כרמת אבטחה גבוהה. כדי שאתר Windows Update יפעל כראוי, עליך לשנות את אפשרויות האבטחה בדפדפן עבור אזור האינטרנט לרמת אבטחה בינונית או נמוכה. כדי לעשות זאת, לחץ על תפריט כלים ולאחר מכן לחץ על אפשרויות אינטרנט. בחר בכרטיסיה אבטחה ולאחר מכן לחץ על אינטרנט.

מהו פקד ActiveX?
פקדי ActiveX הם רכיבי תוכנה הניתנים לשימוש חוזר אשר עושים שימוש בטכנולוגיית ActiveX. ניתן להשתמש ברכיבים אלה להוספת פונקציונליות מיוחדת, כגון הנפשה או תפריטים מוקפצים, לדפי אינטרנט, ליישומי שולחן עבודה ולכלי פיתוח תוכנה. Windows Update עושה שימוש בפקדי ActiveX כדי לבדוק אילו תוכנות מותקנות במחשב שלך, לצורך הצגת רשימה מדויקת של עדכונים ופריטי תוכנה אחרים העשויים לעניין אותך.

מהי הרזולוציה הטובה ביותר להצגת אתר Windows Update?
דפי האינטרנט באתר Windows Update מיועדים לצפייה ברזולוציית מסך של 800 על 600 או יותר. כדי לקבוע רזולוציה זו, לחץ על לחצן התחל, הצבע על הגדרות ולאחר מכן לחץ על לוח הבקרה. לחץ פעמיים על סמל תצוגה ולאחר מכן לחץ על הכרטיסיה הגדרות.

האם מידע אודות המחשב שלי מועבר אל Microsoft?
לא. בכל פעם שאתה עובר אל קטלוג עדכוני המוצרים, אתר Windows Update עושה שימוש בפקד כדי לאתר עדכונים ספציפיים עבור המחשב שלך. בדיקה זו מתבצעת באופן מקומי במחשב שלך כדי להבטיח את הפרטיות והאבטחה של הנתונים שלך, והמידע אינו נשלח אל Microsoft.
