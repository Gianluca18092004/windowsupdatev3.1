Tack för ditt intresse för
Windows Update

Windows Update är ett onlinetillägg för Windows som hjälper dig att få ut det mesta möjliga av din dator.

Obs! Du måste köra en Windows-version med giltig licens för att få tillgång till webbplatsen Windows Update.

Användare av Windows® 98 och Windows® 2000:
Starta Windows Update via länken på Start-menyn.

Användare av Windows® 95 och Windows NT® 4.0:
Du behöver en version av Internet Explorer 4 eller senare för att kunna använda Windows Update.

Alla användare:
Windows Update kan endast användas med Internet-webbläsare som stöder ActiveX-kontroller. Om du vill ha mer information om ActiveX-kontroller går du till Windows Update FAQ.

Om du vill hämta den senaste versionen av Internet Explorer går du till webbplatsen för Internet Explorer.

När du har installerat Internet Explorer går du till webbplatsen Windows Update genom att i adressfältet skriva http://windowsupdate.microsoft.com.

Information om var du hittar tillgängliga uppdateringar under tiden:

  • Windows 2000-användare går till webbplatsen Windows 2000 Downloads.
  • Windows NT 4.0-användare går till webbplatsen Microsoft Download Center.
  • Windows 98-användare går till webbplatsen Windows 98 Downloads.
  • Windows 95-användare går till webbplatsen Windows 95 Downloads.
(Dessa webbplatser är på engelska.)
