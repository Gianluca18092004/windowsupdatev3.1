Por que o site do Windows Update aparece em um idioma diferente?

View this page in English.

O site do Windows Update detecta diversas configurações para determinar o idioma no qual o site será exibido e o idioma dos componentes disponíveis para download.

A primeira detecção é o idioma e a versão de seu sistema operacional e navegador. Os componentes disponíveis para download no Windows Update serão sempre do idioma e da versão correta para o seu sistema operacional, independentemente do idioma no qual você está vendo o site.

O Windows Update também detecta as Configurações regionais de seu sistema operacional para determinar o idioma no qual o site será exibido. Por exemplo, se você mora na Grécia, mas tem um sistema operacional em inglês, você veria o site em grego, mas os downloads disponíveis seriam em inglês.

O Internet Explorer também tem um recurso chamado Aceitar idioma que substitui as Configurações regionais. Se você estiver exibindo o Windows Update (disponível em vários idiomas), o Internet Explorer procurará a lista de idiomas nas suas configurações de preferência de idioma para determinar qual idioma exibir. Essa lista é priorizada, de forma que se você tiver definido o grego como o primeiro idioma e o inglês como o segundo, o Windows Update será exibido em grego.

Para ver o site do Windows Update em outro idioma, você poderá ajustar as configurações de preferência de idioma de seu Internet Explorer como descrito abaixo. Observe que as alterações feitas aqui afetarão outros sites disponíveis em vários idiomas que você exibir.

  1. No Internet Explorer, clique no menu Ferramentas.
  2. Clique em Opções da Internet.
  3. Na guia Geral, clique no botão Idiomas.
  4. Selecione o idioma desejado e use o botão Mover para cima para colocar sua seleção no início da lista de idiomas. Clique em "OK."
  5. Clique em "OK" na caixa de diálogo 'Opções da Internet' e reinicie o navegador.
  6. Recarregue a página do Windows Update.

Why does the Windows Update site appear in a different language?

The Windows Update site detects several settings to determine the language in which the site will be displayed and the language of the components available for download.

The first detection is the language and version of your operating system and browser. Components available for download from Windows Update will always be the correct language and version for your operating system, regardless of the language in which you are viewing the site.

Windows Update also detects your operating system's Regional Settings to determine the language in which the site will be displayed. As an example, if you live in Greece, but have an English operating system, you would see the site in Greek, but the downloads available would be English.

Internet Explorer also has a feature called Accept Language which supercedes the Regional Settings. If you are viewing Windows Update (which is available in multiple languages), Internet Explorer will look to the list of languages in your Language Preference settings to determine which language to display. This is a prioritized list, so if you have Greek as the first language and English as the second, Windows Update will display in Greek.

To see the Windows Update site in a different language, you can adjust your Internet Explorer Language Preference settings as described below. Please note that changes here will affect other multi-language sites that you view.

  1. In Internet Explorer, click the Tools menu.
  2. Click Internet Options.
  3. On the General tab, click the Languages button.
  4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages. Click "OK."
  5. Click "OK" in the Internet Options dialog and restart your browser.
  6. Reload the Windows Update page.
