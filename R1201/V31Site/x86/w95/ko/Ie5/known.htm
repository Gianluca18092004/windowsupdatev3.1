Windows Update로 돌아가기

Windows Update의 알려진 문제
 다음은 Windows Update 웹 사이트의 알려진 문제들입니다. 아래 목록에 없는 문제점이 발견되면 검정색 도구 모음의 고객 의견을 클릭한 다음 Windows Update를 클릭하여 문제점을 알려주시기 바랍니다.
  1. Windows Update는 사용자가 사이트를 방문할 때마다 제품 업데이트 카탈로그를 새로 고쳐서 사용자 컴퓨터의 최신 상태를 보여줍니다. 예를 들어, 새로운 두 개의 테마를 다운로드한 다음 제품 업데이트 카탈로그로 되돌아가면 목록이 업데이트되어 다운로드한 두 개의 테마는 업데이트 대상으로 표시되지 않습니다. 이 두 테마는 설치 내역 보기에 나타나지만 이 경우 설치된 항목으로 표시됩니다.

  2. 다운로드가 진행되는 중에는 왼쪽 탐색 모음을 사용할 수 없습니다. 다운로드가 완료되면 탐색 모음을 다시 사용할 수 있습니다.

  3. Windows Update 홈 페이지의 링크는 선택한 후에도 색이 변경되지 않습니다.

  4. 사이트의 일부 기능에 대해 사용할 수 있는 내게 필요한 옵션의 바로 가기 키는 아직 완전히 제공되지 않습니다.

  5. 드문 경우이긴 하지만 이 사이트를 여러 브라우저에서 연 다음 제품 업데이트를 동시에 실행하여 업데이트를 확인하면 브라우저가 오랜 시간 동안 작동하지 않을 수 있습니다.

  6. 제품 업데이트 카탈로그에서 구성 요소를 다운로드한 다음 화면에 "다운로드 완료" 페이지가 표시되는 동안 새 브라우저 창을 열지 마십시오. 그렇지 않으면 스크립트 오류 메시지가 나타납니다.

  7. 간혹 모뎀을 통하여 인터넷에 연결하는 경우 스크립트 오류가 발생할 수 있습니다. 이 경우 연결을 다시 시도하십시오.

  8. 보안 설정이 높음으로 지정되면 제품 업데이트 카탈로그에 액세스할 수 없습니다. 사이트의 제품 업데이트 영역에 액세스하려면 보안 설정을 보통 또는 낮음으로 지정해야 합니다. 보안 설정을 변경하려면 다음을 수행하십시오.
    1. 도구 메뉴에서 인터넷 옵션을 클릭합니다.
    2. 보안 탭을 클릭합니다.
    3. 보통 설정을 선택합니다.

  9. NT 컴퓨터 또는 Windows 2000 컴퓨터에 소프트웨어를 설치하려면 관리자 권한이 없으면 이 사이트의 특정 페이지를 보는 데 필요한 컨트롤을 다운로드할 수 없습니다.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Windows Update 사이트가 브라우저 또는 운영 체제에서 사용하는 언어가 아닌 다른 언어로 표시될 수 있습니다. 예를 들어, 사용자의 브라우저 및 운영 체제는 영어로 표시되는데 Windows Update 사이트는 프랑스어로 표시될 수 있습니다. Windows Update 사이트는 브라우저의 언어 기본 설정에서 지정된 언어로 표시됩니다. 브라우저의 언어 기본 설정을 변경하려면 다음을 수행하십시오.
    1. Internet Explorer에서 도구 메뉴를 클릭합니다.
    2. 인터넷 옵션을 클릭합니다.
    3. 일반 탭에서 언어 단추를 클릭합니다.
    4. 원하는 언어를 선택한 다음 위로 이동 단추를 사용하여 언어 목록의 맨 위로 선택한 언어를 이동시킵니다.

  12. 바이러스 예방 소프트웨어 사용자: 일부 바이러스 예방 소프트웨어 프로그램은 일부 구성 요소의 다운로드 및 설치를 방해할 수 있습니다. 설치를 방해하지 않도록 하려면 제품 업데이트 카탈로그에서 항목을 다운로드하기 전에 바이러스 예방 프로그램을 끝내십시오.

  13. Internet Explorer의 즐겨찾기를 사용하여 Windows Update에 연결하지 못할 수 있습니다. Internet Explorer 도구 메뉴(Internet Explorer에서 도구를 클릭한 다음 Windows Update를 클릭)에 있는 링크를 클릭하거나 Internet Explorer 주소 표시줄에 v31update.vornext.gq을 입력하여 Windows Update에 연결할 수 있습니다.

  14. 최근에 Internet Explorer 브라우저를 업그레이드한 경우 Windows Update로 되돌아가면 Internet Explorer의 이전 버전에서 설치한 업데이트가 나타날 수 있지만, Windows Update에서는 이제 업그레이드한 브라우저용 업데이트가 제공됩니다. 새 Internet Explorer 브라우저에서 가장 높은 보안 수준을 유지하려면 브라우저에 이 업데이트를 설치할 것을 권장합니다.

  15. 자세한 정보 링크는 Windows Update 웹 사이트에서 더 이상 사용할 수 없습니다. Windows Update에서 사용할 수 있는 보안 업데이트에 대한 자세한 정보를 얻으려면 다음 단계를 수행하십시오.
    1. Microsoft Security 웹 사이트로 이동합니다. 이 사이트는 영문 사이트입니다.
    2. Search for Bulletins and Patches를 클릭합니다.
    3. 검색 텍스트 상자에 Microsoft 기술 자료(KB) 문서 번호를 입력합니다.
    4. Search를 클릭합니다.

  16. Windows Update 웹 사이트에서 설치한 업데이트를 제거하려면 다음 단계를 수행하십시오.
    1. 시작을 클릭하고 설정을 가리킨 다음 제어판을 클릭합니다. (Windows XP의 경우 시작을 클릭한 다음 제어판을 클릭합니다.)
    2. 프로그램 추가/제거를 두 번 클릭합니다.
    3. 제거할 항목을 클릭하고 추가/제거를 클릭합니다. (Windows XP 및 Windows 2000의 경우 변경/제거를 클릭합니다.)
    참고: 프로그램 추가/제거에 나타나지 않는 항목은 제거할 수 없습니다.
