WINDOWS UPDATE 정보
 Windows Update는 Windows의 온라인 확장 기능으로 제품 업데이트 카탈로그와 지원 정보의 두 가지 영역으로 구성되어 있습니다. 화면 왼쪽의 탐색 모음을 사용하여 두 영역을 탐색하십시오.

지원 정보
제품 업데이트 정보
제품 업데이트 영역은 Windows와 Windows에서 실행되는 프로그램들의 수정, 업데이트, 향상된 기능 등을 모아 놓은 카탈로그입니다. 이 카탈로그를 통해 사용자의 컴퓨터 성능을 향상할 수 있는 구성 요소를 얻으실 수 있습니다.

이 카탈로그를 살펴 보고 원하는 구성 요소를 선택한 다음, 웹에서 직접 다운로드하여 컴퓨터에 설치하십시오. 웹에서 다운로드하는 방법을 모르면 아래의 "사용자가 할 일은?" 섹션을 읽어 보십시오. 원한다면, 컴퓨터에 이미 설치되어 있는 소프트웨어에 따라 여러분의 컴퓨터에 가장 적합한 업데이트와 향상된 기능을 카탈로그가 판단하여 알려 줄 수도 있습니다.

사용 방법
제품 업데이트 페이지를 처음 방문하면 Windows Update는 시스템을 검색하여 기존에 설치되어 있는 구성 요소를 확인하고 설치할 수 있는 구성 요소 목록을 보여 줍니다. 검색 작업은 시스템 정보의 안전과 데이터 보안을 위해 사용자 컴퓨터에서 수행됩니다. 이 정보는 Microsoft나 인터넷으로 전송되지 않습니다. 이러한 시스템 점검을 통해 웹에서 가장 최신의 올바른 버전을 선택하여 다운로드하실 수 있습니다. 또한 이미 설치되어 있는 구성 요소를 또 다시 다운로드하는 것을 막을 수 있습니다.

페이지에 구성 요소 목록이 표시되면, 원하는 구성 요소를 찾아내어 컴퓨터로 다운로드할 수 있습니다.
시스템에 필요한 중요 업데이트가 있으면 모두 다운로드 하십시오. 중요 업데이트는 컴퓨터와 관련된 알려진 문제점(보안 문제 등)을 해결해 줍니다.

사용자가 할 일은?
제품 업데이트에서는 소프트웨어 업데이트와 향상된 기능을 컴퓨터로 쉽게 다운로드할 수 있습니다.
  1. 우선 카탈로그를 살펴 봅니다.
  2. 다운로드할 구성 요소를 결정했으면 구성 요소 옆에 있는 상자를 누릅니다. 상자에 확인 표시가 나타납니다. 구성 요소를 다운로드하기 전에 "업데이트 정보" 링크를 클릭하여 다음과 같은 정보를 반드시 읽어 보십시오.
    • 구성 요소에 대한 자세한 설명
    • 구성 요소를 사용하여 시작하는 방법
    • 제거용 설명서
    • 지원 정책
  3. 선택한 제품을 취소하려면 상자를 다시 클릭하여 확인 표시를 지우면 됩니다.
  4. 원하는 구성 요소를 모두 선택했으면 다운로드 단추를 누릅니다.
  5. 페이지가 나타나면 원하는 구성 요소가 목록에 모두 있는지 확인하십시오. 변경하려면 뒤로 단추를 누르십시오.
  6. 다운로드 시작 단추를 누릅니다. 선택한 구성 요소가 모두 다운로드 되고 설치됩니다. 일부 구성 요소는 설치 중에 추가로 정보를 입력해야 합니다.
설치 완료 메시지가 나타나면 설치가 완료된 것입니다. 구성 요소를 모두 설치한 후에 시스템을 종료하고 다시 시작해야 할 수도 있습니다.

대부분의 구성 요소는 제품 업데이트 페이지에서 제거할 수 있습니다. 구성 요소를 제거하려면

  1. 제품 업데이트 페이지로 이동합니다. 대화 상자가 나타나면 예를 누릅니다
  2. 제품 업데이트 페이지가 나타나면 페이지 위쪽에 있는 설치된 업데이트 보기 단추를 누릅니다.
  3. 제거하려는 구성 요소의 오른쪽에 있는 제거 단추를 누릅니다.
