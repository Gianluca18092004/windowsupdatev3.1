Which version of Internet Explorer do I have?

To determine which version of Internet Explorer you are running, click Help in the Internet Explorer File Menu and then click About Internet Explorer.

  • If you are running Internet Explorer 5.5, the version number will be displayed as 5.50.4134.0600.

  • If you are running Internet Explorer 5.01 SP1, the version number will be displayed as 5.00.3105.0106

  • If you are running Internet Explorer 5.01, the version number will be displayed as 5.0.2919.6307.

  • If you are running Internet Explorer 4.01 SP2, the version number will be displayed as 4.72.3612.1713.
If About Internet Explorer does not display one of these version numbers, your computer may be vulnerable to several known security issues.
