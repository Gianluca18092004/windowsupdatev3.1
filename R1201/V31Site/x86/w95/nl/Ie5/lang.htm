Waarom wordt Windows Update in een andere taal weergegeven?

View this page in English.

Voor Windows Update worden meerdere instellingen gecontroleerd om de taal te bepalen waarin de site moet worden weergegeven en om de taalversies te bepalen van de beschikbare componenten.

Het eerst wordt gekeken naar de taal en de versie van het besturingssysteem en de browser. De taal en versie van de componenten die u kunt downloaden vanaf Windows Update zijn altijd de meest geschikte voor uw besturingssysteem. Dit is onafhankelijk van de taal waarin de site wordt weergegeven.

Ook wordt gekeken naar de landinstellingen van het besturingssysteem. Met deze informatie wordt bepaald in welke taal de site wordt weergegeven. Als u bijvoorbeeld in Nederland woont maar een Engels besturingssysteem hebt, wordt de site in het Nederlands weergegeven maar zijn de componenten die u kunt downloaden Engels.

Internet Explorer bevat ook een functie om talen te accepteren. De ingestelde taal krijgt voorrang op de landinstellingen. Als u de meertalige site Windows Update bezoekt, zoekt Internet Explorer in de lijst met voorkeurtalen om te bepalen in welke taal de site moet worden weergegeven. De talen staan in volgorde van voorkeur, dus als Nederlands de eerste taal in de lijst is en Engels de tweede, wordt Windows Update weergegeven in het Nederlands.

Als u Windows Update in een andere taal wilt weergeven, kunt u de instellingen voor de voorkeurtalen van Internet Explorer wijzigen zoals hieronder wordt beschreven. Let op: deze wijzigingen worden ook van kracht voor andere meertalige sites.

  1. Klik in Internet Explorer op het menu Extra.
  2. Klik op Internet-opties.
  3. Klik op het tabblad Algemeen op de knop Talen.
  4. Selecteer de gewenste taal en plaats deze bovenaan de lijst met de knop Omhoog. Klik op OK.
  5. Klik op OK in het dialoogvenster Internet-opties en start de browser opnieuw.
  6. Laad de pagina Windows Update opnieuw.

Why does the Windows Update site appear in a different language?

The Windows Update site detects several settings to determine the language in which the site will be displayed and the language of the components available for download.

The first detection is the language and version of your operating system and browser. Components available for download from Windows Update will always be the correct language and version for your operating system, regardless of the language in which you are viewing the site.

Windows Update also detects your operating system's Regional Settings to determine the language in which the site will be displayed. As an example, if you live in Greece, but have an English operating system, you would see the site in Greek, but the downloads available would be English.

Internet Explorer also has a feature called Accept Language which supercedes the Regional Settings. If you are viewing Windows Update (which is available in multiple languages), Internet Explorer will look to the list of languages in your Language Preference settings to determine which language to display. This is a prioritized list, so if you have Greek as the first language and English as the second, Windows Update will display in Greek.

To see the Windows Update site in a different language, you can adjust your Internet Explorer Language Preference settings as described below. Please note that changes here will affect other multi-language sites that you view.

  1. In Internet Explorer, click the Tools menu.
  2. Click Internet Options.
  3. On the General tab, click the Languages button.
  4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages. Click "OK."
  5. Click "OK" in the Internet Options dialog and restart your browser.
  6. Reload the Windows Update page.
