Вы попали на эту страницу по одной из следующих причин:
  1. Причиной данного сообщения об ошибке могло быть изменение элементов управления или архитектуры узла Windows Update. Подробнее см. в статье Q243787 базы знаний Microsoft Knowledge Base (KB).
  2. Вы решили не загружать элементы управления или возникли сложности с загрузкой управляющих элементов, поэтому большая часть узла Windows Update для вас недоступна. Если требуется загрузить элементы управления, нажмите находящуюся выше ссылку Повторить.
  3. У вас отсутствуют права администратора на этом компьютере. Чтобы использовать Windows Update для обновления приложений, вам необходимо обратиться к системному администратору.
  4. Возможно, вам следует очистить кеш обозревателя. Более подробные сведения можно найти в статье Q193385 базы знаний Microsoft KB.
  5. Возможно, вам следует удалить папку Windows Update. Более подробные сведения можно найти в статье Q243787 базы знаний Microsoft KB.
  6. Обновления для Windows NT® 4.0 Terminal Server Edition отсутствует на веб-узле Windows Update. Дополнительные сведения можно найти в статье Q323564 базы знаний Майкрософт.
