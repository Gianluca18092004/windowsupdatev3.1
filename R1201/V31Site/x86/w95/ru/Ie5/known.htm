Замеченные неполадки Windows Update
 Ниже перечислены замеченные неполадки в работе Web-узла Windows Update. Если вы обнаружите другие неполадки, сообщите нам об этом. Для этого выберите ссылку Пишите нам на черной инструментальной панели и нажмите Windows Update.
  1. Каждый раз, когда вы просматриваете эту область узла, Windows Update обращается к каталогу обновления приложений. Это позволяет видеть самую последнюю версию состояния вашей системы. Например, если вы загрузили две новых темы и затем вернулись к каталогу обновления приложений, происходит обновление данных, и вы больше не увидите эти темы в качестве доступных обновлений. Темы останутся видимыми в режиме Показать установленное, но будут помечены как установленные.

  2. Во время загрузки левая панель обозревателя не работает. Это обеспечивает успешную загрузку. После завершения загрузки работоспособность панели восстанавливается.

  3. Цвет ссылок на домашней странице Windows Update после использования не меняется.

  4. Не полностью реализована возможность использования клавиатуры при обращении к некоторым функциям узла.

  5. В очень редких случаях при работе с несколькими экземплярами узла, когда управляющий элемент каталога "Обновление приложений" одновременно запущенный различными экземплярами, определяет необходимые обновления, работа обозревателя может быть заблокирована на продолжительное время.

  6. После загрузки компонента из каталога обновления приложений, не открывайте новых окон в обозревателе, пока страница "Загрузка подтверждена" (Download Confirmed) находится на экране. В противном случае появляется сообщение об ошибке сценария.

  7. В редких случаях, когда соединение с Интернетом происходит через модем, может возникнуть ошибка сценария. Если это произойдет, повторите соединение.

  8. Если на вашем компьютере установлен высокий уровень безопасности, доступа в каталог "Обновление приложений" не будет. Чтобы использовать этот раздел узла, необходимо установить средний или низкий уровень безопасности. Чтобы изменить настройки безопасности выполните следующие действия.
    1. В меню Сервис выберите пункт Свойства обозревателя.
    2. Выберите вкладку Безопасность.
    3. Установите значение Средний.

  9. В случае получения сообщения об уведомлении о ключевых обновлениях до входа в сеть или в операционную систему Windows, подтвердите уведомление после входа. Это можно сделать, переместив диалоговое окно выбором команды Повторить уведомление позже. После этого необходимо войти в систему. Такое сообщение может появиться только в том случае, если ваш компьютер подсоединен к локальной сети.

  10. Если в системе Windows 98 загружены исправления неполадок безопасности или компонент IME/SR1, повторная установка Windows 98 стирает эти исправления и код IME. Однако каталог продолжает показывать компоненты IME и исправления неполадок безопасности как установленные. Для повторной установки этих исправлений и компонентов IME выполните следующие действия.
    1. Перейдите в раздел обновления приложений Windows Update.
    2. Выберите команду Показать установленное и затем установите флажки рядом с исправлениями неполадок безопасности и компонентами IME, которые требуется установить.
    3. Нажмите кнопку Загрузить и следуйте указаниям.

  11. В некоторых случаях, после установки компонента "Уведомление о ключевых обновлениях", каталог не показывает, что этот компонент установлен. Щелкните значок Установка и удаление программ панели управления и проверьте правильность установки.

  12. При установке пользовательского пакета обновления для Windows 98 обновление 2000 года для Outlook Express (в составе данного пользовательского пакета обновления) не устанавливается. Компонент отображается с пометкой "не установлено" после завершения установки пользовательского пакета обновления. Чтобы установить обновление 2000 года для Outlook Express, вернитесь в Каталог обновления приложений, выберите Обновление 2000 года для Outlook Express 4.0 в разделе ключевых обновлений и загрузите этот компонент отдельно.

  13. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  14. Веб-узел Windows Update может отображаться на языке, не совпадающем с языком вашего обозревателя или операционной системы. Например, вы можете использовать английский обозреватель и операционную систему, но попадаете на веб-узел Windows Update на французском языке. Узел Windows Update отображается на том языке, который указан как предпочтительный в параметрах обозревателя. Чтобы изменить этот параметр, выполните следующие действия.
    1. В обозревателе Internet Explorer выберите меню Сервис.
    2. Выполните команду Свойства обозревателя.
    3. На вкладке Общие нажмите кнопку Языки.
    4. Выберите нужный язык, затем, используя кнопку Выше, расположите выбранный язык в верхней строке списка.

  15. Пользователям антивирусных программ: обратите внимание на то, что некоторые антивирусные программы могут помешать загрузке и установке компонентов. Чтобы предотвратить это, на время загрузки элементов из каталога "Обновление приложений" отключите антивирусные программы.

  16. Ссылка на узел Windows Update в избранном обозревателя Internet Explorer может работать неправильно. Рекомендуем для доступа к узлу Windows Update использовать ссылку в меню "Пуск" (нажмите кнопку Пуск и затем выберите Windows Update), или введите v31update.vornext.gq в адресной строке обозревателя Internet Explorer.

  17. Сразу после обновления версии обозревателя Internet Explorer при обращении к узлу Windows Update можно увидеть обновления, которые уже были установлены для более ранней версии обозревателя Internet Explorer. Обновления, предлагаемые сейчас, предназначены для обновленной версии обозревателя. Чтобы обеспечить максимальный уровень безопасности обозревателя Internet Explorer, корпорация Майкрософт рекомендует установить эти обновления для новой версии обозревателя.
