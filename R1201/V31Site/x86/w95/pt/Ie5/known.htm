Problemas conhecidos do Windows Update
 Seguidamente serão apresentados os problemas conhecidos com o Web site do Windows Update. Se se deparar com um problema que não esteja referido nesta lista, informe-nos clicando em Write Us (Escreva-nos) na barra de ferramentas a preto e, depois, em Windows Update.
  1. O Windows Update actualiza o catálogo de actualizações de produtos sempre que visitar esta área do site, o que lhe permite ver o estado mais actual do seu computador. Por exemplo, se transferir dois temas musicais novos e, em seguida, regressar ao catálogo de actualizações de produtos, as informações apresentadas estarão actualizadas e deixará de ver os dois temas que transferiu como estando disponíveis. No entanto, continuarão a aparecer na vista Show Installed Updates (Mostrar actualizações instaladas) como instalados.

  2. Não é possível utilizar a barra de navegação à esquerda enquanto a transferência estiver em curso, o que ajuda a garantir o êxito da transferência. Quando esta estiver concluída, a barra de navegação ficará de novo disponível.

  3. A cor das hiperligações da home page do Windows Update não se altera quando são seleccionadas.

  4. O acesso por teclado a determinadas funções do site, para fins de acessibilidade, não está completamente implementado.

  5. Em ocasiões muito raras, se optar por executar várias instâncias deste site e tiver accionado também o controlo das actualizações de produtos para verificar simultaneamente a existência de actualizações, o browser ficará bloqueado durante um longo período de tempo.

  6. Depois de transferir um componente do catálogo de actualizações de produtos, não abra uma nova janela do browser enquanto a página de confirmação de transferência continuar no ecrã. Se o fizer, aparece uma mensagem de erro de scripts.

  7. Em ocasiões muito raras, quando é estabelecida uma ligação à Internet através de um modem, poderá aparecer um erro de scripts. Caso tal aconteça, tente estabelecer novamente a ligação.

  8. Se as definições de segurança estiverem configuradas para Alta, não conseguirá aceder ao catálogo de actualizações de produtos. Para utilizar a área de actualizações de produtos do site, é necessário configurar as definições de segurança como Média ou Baixa. Para alterar as definições de segurança:
    1. No menu Ferramentas, clique em Opções da Internet.
    2. Clique no separador Segurança.
    3. Escolha a definição Média.

  9. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  10. O site do Windows Update poderá aparecer num idioma que não o do seu browser ou sistema operativo. Por exemplo, o browser e o sistema operativo poderão estar em inglês, mas o site do Windows Update em francês. O site do Windows Update aparece no idioma para o qual está configurada a sua preferência de idioma. Para alterar a preferência de idioma do browser:
    1. No Internet Explorer, clique no menu Ferramentas.
    2. Clique em Opções da Internet.
    3. No separador Geral, clique no botão Idiomas.
    4. Seleccione o idioma que deseja e, em seguida, utilize o botão Mover para cima para colocar a sua selecção em primeiro lugar na lista de idiomas.

  11. Utilizadores de software antivírus: Tenha em atenção que alguns programas de software antivírus podem interferir com a transferência e instalação de alguns componentes. Para evitar esta interferência, desactive o software antivírus antes de transferir itens do catálogo de actualizações de produtos.

  12. Uma hiperligação dos Favoritos do Internet Explorer para o Windows Update poderá não funcionar correctamente. Para melhores resultados, aceda ao Windows Update utilizando a hiperligação no menu Iniciar (clique em Iniciar e, em seguida, clique em Windows Update), ou escreva v31update.vornext.gq na barra de endereços do Internet Explorer.

  13. Se actualizou recentemente o browser do Internet Explorer, quando voltar ao Windows Update poderá ver as actualizações que instalou quando estava a executar a versão mais antiga do Internet Explorer. As actualizações agora oferecidas são específicas para o browser actualizado. Para garantir que a nova versão do Internet Explorer apresenta o nível de segurança mais elevado, a Microsoft recomenda a instalação dessas actualizações no novo browser.

  14. A hiperligação Read more (Obter mais informações) já não se encontra disponível no Web site Windows Update. Para mais informações sobre as actualizações de segurança disponíveis no Windows Update, proceda da seguinte forma:
    1. Vá até ao Web site Microsoft Security.
    2. Clique em Search for Bulletins and Patches.
    3. Na caixa de texto Search, escreva o número do artigo da Microsoft Knowledge Base (KB).
    4. Clique em Procurar.

  15. Se desejar remover uma actualização instalada a partir do Web site do Windows Update:
    1. Clique em Iniciar, aponte para Definições e, finalmente, clique em Painel de controlo. (No Windows XP, clique em Iniciar e depois em Painel de controlo.)
    2. Clique em Adicionar ou remover programas.
    3. Seleccione o item que pretende remover e clique em Adicionar/remover. (No Windows XP e Windows 2000, clique em Alterar/remove.)
    Nota: Se o item não estiver na lista de adicionar ou remover programas, é porque não pode ser removido do computador.
