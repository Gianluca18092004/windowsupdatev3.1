Επιστροφή στο Windows Update

Γνωστά θέματα για το Windows Update
 Ακολουθούν τα γνωστά θέματα της τοποθεσίας του Windows Update στο Web. Εάν εντοπίσετε κάποιο πρόβλημα, που δεν περιλαμβάνεται στην ακόλουθη λίστα, ενημερώστε μας, κάνοντας κλικ στο μενού ΕΠΙΚΟΙΝΩΝΙΑ στη μαύρη γραμμή εργαλείων, και, στη συνέχεια, κάνοντας κλικ στην επιλογή Windows Update.
  1. Το Windows Update ανανεώνει τον κατάλογο ενημερωμένων εκδόσεων προϊόντων κάθε φορά που επισκέπτεστε αυτήν την περιοχή της τοποθεσίας. Αυτό σας παρέχει τη δυνατότητα να βλέπετε την πιο πρόσφατη κατάσταση του υπολογιστή σας. Για παράδειγμα, εάν κάνετε λήψη δύο νέων θεμάτων και επιστρέψετε, στη συνέχεια, στον κατάλογο ενημερωμένων εκδόσεων προϊόντων, θα έχει γίνει ενημέρωση της προβολής και δεν θα βλέπετε πλέον μεταξύ των διαθέσιμων ενημερώσεων τα δύο θέματα που λάβατε. Θα εξακολουθούν να εμφανίζονται στην προβολή Εμφάνιση εγκατεστημένων ενημερώσεων, αλλά ως εγκατεστημένα.

  2. Κατά τη διάρκεια της λήψης, δεν είναι δυνατή η χρήση της αριστερής γραμμής περιήγησης. Αυτό συμβαίνει, προκειμένου να ολοκληρώνεται κάθε λήψη με επιτυχία. Μόλις ολοκληρωθεί η λήψη, η γραμμή πλοήγησης είναι και πάλι διαθέσιμη.

  3. Οι συνδέσεις της κεντρικής σελίδας του Windows Update δεν αλλάζουν χρώμα αφού τις επιλέξετε.

  4. Η πρόσβαση μέσω του πληκτρολογίου σε ορισμένες λειτουργίες της τοποθεσίας δεν είναι πάντα εφικτή, για λόγους πρόσβασης.

  5. Σε πολύ σπάνιες περιπτώσεις, εάν επιλέξετε την εκτέλεση πολλών περιόδων λειτουργίας της τοποθεσίας, ενώ εκτελείται ταυτόχρονα ο έλεγχος ενημερωμένων εκδόσεων προϊόντων, μπορεί να κλειδώσει το πρόγραμμα περιήγησης στο Internet για μεγάλο χρονικό διάστημα.

  6. Μετά τη λήψη ενός στοιχείου από τον κατάλογο ενημερωμένων εκδόσεων προϊόντων, μην ανοίγετε νέα παράθυρα του προγράμματος περιήγησης όσο εμφανίζεται στην οθόνη η σελίδα επιβεβαίωσης της λήψης αρχείων. Διαφορετικά, θα εμφανιστεί μήνυμα σφάλματος δέσμης ενεργειών.

  7. Σε σπάνιες περιπτώσεις, μπορεί, κατά τη σύνδεση στο Internet μέσω μόντεμ, να παρουσιαστεί σφάλμα δέσμης ενεργειών. Εάν συμβεί, προσπαθήστε να συνδεθείτε ξανά.

  8. Εάν έχει οριστεί υψηλό επίπεδο ασφαλείας, δεν θα είναι δυνατή η πρόσβαση στον κατάλογο ενημερωμένων εκδόσεων προϊόντων. Για να χρησιμοποιήσετε την περιοχή ενημερωμένων εκδόσεων προϊόντων της τοποθεσίας, θα πρέπει να ορίσετε το μεσαίο ή χαμηλό επίπεδο ασφαλείας. Για να αλλάξετε τις ρυθμίσεις ασφαλείας:
    1. Στο μενού Εργαλεία, κάντε κλικ στην επιλογή Επιλογές Internet.
    2. Κάντε κλικ στην καρτέλα Ασφάλεια.
    3. Επιλέξτε τη ρύθμιση Μεσαίο.

  9. Μόνο σε χρήστες με δικαιώματα διαχειριστή επιτρέπεται η εγκατάσταση ή ενημέρωση λογισμικού σε αυτόν τον υπολογιστή με Windows NT. Κατά συνέπεια, οι χρήστες που δεν έχουν δικαιώματα διαχειριστή δεν θα μπορούν, πιθανότατα, να κάνουν λήψη των στοιχείων ελέγχου για την προβολή ορισμένων σελίδων της τοποθεσίας.

  10. The Windows Update site may appear in a language other than the language of your browser or operating system. For example, your browser and operating system might be in English, but the Windows Update site is in French. The Windows Update site appears in the language for which your browser's language preference is configured. To change the language preference of your browser:
    1. In Internet Explorer, click the Tools menu.
    2. Click Internet Options.
    3. On the General tab, click the Languages button.
    4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages.

  11. Η τοποθεσία του Windows Update μπορεί να εμφανίζεται σε γλώσσα διαφορετική από αυτήν του προγράμματος περιήγησης ή του λειτουργικού συστήματος. Μπορεί, για παράδειγμα, η γλώσσα του προγράμματος περιήγησης και του λειτουργικού συστήματος να είναι η αγγλική, ενώ η τοποθεσία του Windows Update να εμφανίζεται στα γαλλικά. Η γλώσσα στην οποία εμφανίζεται η τοποθεσία του Windows Update εξαρτάται από την προτίμηση γλώσσας που έχει οριστεί για το πρόγραμμα περιήγησης. Για αλλαγή της προτίμησης γλώσσας του προγράμματος περιήγησης:
    1. Στον Internet Explorer, κάντε κλικ στο μενού Εργαλεία.
    2. Κάντε κλικ στην επιλογή Επιλογές Internet.
    3. Στην καρτέλα Γενικά, κάντε κλικ στο κουμπί Γλώσσες.
    4. Επιλέξτε τη γλώσσα που θέλετε και χρησιμοποιήστε το κουμπί Μετακίνηση επάνω για να τοποθετήσετε την επιλογή σας στην κορυφή της λίστας.

  12. Χρήστες προγραμμάτων αντιμετώπισης ιών: Πρέπει να έχετε υπόψη σας ότι ορισμένα προγράμματα αντιμετώπιση ιών μπορεί να προκαλέσουν καθυστερήσεις κατά τη λήψη και εγκατάσταση ορισμένων στοιχείων. Για να αποτρέψετε αυτές τις καθυστερήσεις, απενεργοποιήστε το πρόγραμμα αντιμετώπισης ιών πριν από τη λήψη στοιχείων από τον κατάλογο ενημερωμένων εκδόσεων προϊόντων.

  13. Η σύνδεση Αγαπημένων για το Windows Update στον Internet Explorer μπορεί να μη λειτουργεί σωστά. Για καλύτερα αποτελέσματα κατά την πρόσβασή σας στο Windows Update, χρησιμοποιείτε τη σύνδεση στο μενού "Εργαλεία" του Internet Explorer (στον Internet Explorer, κάντε κλικ στο μενού Εργαλεία και στη συνέχεια κάντε κλικ στην επιλογή Windows Update) ή πληκτρολογήστε v31update.vornext.gq στη γραμμή διευθύνσεων του Internet Explorer.

  14. Εάν αναβαθμίσατε πρόσφατα τον Internet Explorer, μπορεί, επιστρέφοντας στο Windows Update, να δείτε ενημερωμένες εκδόσεις, τις οποίες είχατε εγκαταστήσει στο παρελθόν, όταν χρησιμοποιούσατε παλαιότερη έκδοση του Internet Explorer. Οι ενημερωμένες εκδόσεις που σας προσφέρονται τώρα προορίζονται ειδικά για την αναβαθμισμένη έκδοση του προγράμματος περιήγησης που χρησιμοποιείτε. Για να βεβαιωθείτε ότι το επίπεδο ασφάλειας της νέας έκδοσης του Internet Explorer είναι το υψηλότερο δυνατό, η Microsoft σας συνιστά να εγκαταστήσετε αυτές τις ενημερωμένες εκδόσεις.

  15. Η σύνδεση Περισσότερες πληροφορίες δεν περιλαμβάνεται πλέον στην τοποθεσία του Windows Update στο Web. Για περισσότερες πληροφορίες σχετικά με τις ενημερωμένες εκδόσεις ασφαλείας που διατίθενται από το Windows Update, ακολουθήστε τα παρακάτω βήματα:
    1. Μεταβείτε στην τοποθεσία της Microsoft στο Web για θέματα ασφαλείας.
    2. Κάντε κλικ στη σύνδεση Search for Bulletins and Patches (Αναζήτηση ενημερωτικών δελτίων και διορθωτικών εκδόσεων).
    3. Στο πλαίσιο κειμένου "Search" (Αναζήτηση), πληκτρολογήστε τον αριθμό του άρθρου της γνωσιακής βάσης δεδομένων (KB) της Microsoft.
    4. Κάντε κλικ στο κουμπί Search (Αναζήτηση).

  16. Εάν θέλετε να καταργήσετε μια ενημερωμένη έκδοση, την οποία είχατε εγκαταστήσει από την τοποθεσία του Windows Update στο Web, ακολουθήστε τα παρακάτω βήματα:
    1. Κάντε κλικ στο κουμπί Έναρξη, τοποθετήστε το δείκτη του ποντικιού στην επιλογή "Ρυθμίσεις" και στη συνέχεια κάντε κλικ στην επιλογή Πίνακας Ελέγχου. (Στα Windows XP, κάντε κλικ στο κουμπί Έναρξη και, στη συνέχεια, κάντε κλικ στην επιλογή Πίνακας Ελέγχου.)
    2. Κάντε κλικ στην επιλογή Προσθαφαίρεση προγραμμάτων.
    3. Κάντε κλικ στο στοιχείο που θέλετε να καταργήσετε και στη συνέχεια κάντε κλικ στο κουμπί Προσθαφαίρεση. (Στα Windows XP και στα Windows 2000, κάντε κλικ στο κουμπί Αλλαγή/Κατάργηση.)
    Σημείωση Εάν το στοιχείο δεν εμφανίζεται στην "Προσθαφαίρεση προγραμμάτων", δεν είναι δυνατή η κατάργησή του από τον υπολογιστή.
