Επιστροφή στο Windows Update

Γιατί η τοποθεσία του Windows Update εμφανίζεται σε διαφορετική γλώσσα;

View this page in English.

Η τοποθεσία του Windows Update ανιχνεύει πολλές ρυθμίσεις για τον προσδιορισμό της γλώσσας στην οποία θα εμφανίζεται η τοποθεσία, καθώς και της γλώσσας των στοιχείων που είναι διαθέσιμα για λήψη.

Τα πρώτα στοιχεία που ανιχνεύονται είναι η γλώσσα και η έκδοση του λειτουργικού σας συστήματος και του προγράμματος περιήγησης. Τα στοιχεία που είναι διαθέσιμα για λήψη από το Windows Update θα είναι πάντα στη σωστή γλώσσα και έκδοση για το λειτουργικό σας σύστημα, ανεξάρτητα από τη γλώσσα στην οποία προβάλλετε την τοποθεσία.

Το Windows Update ανιχνεύει επίσης τις τοπικές ρυθμίσεις του λειτουργικού σας συστήματος για τον προσδιορισμό της γλώσσας στην οποία θα εμφανίζεται η τοποθεσία. Για παράδειγμα, εάν ζείτε στην Ελλάδα, αλλά έχετε αγγλικό λειτουργικό σύστημα, θα βλέπετε την τοποθεσία στα ελληνικά, αλλά τα στοιχεία που είναι διαθέσιμα για λήψη θα είναι στα αγγλικά.

Ο Internet Explorer διαθέτει επίσης μια δυνατότητα που ονομάζεται 'Αποδοχή γλώσσας' και η οποία αντικαθιστά τις τοπικές ρυθμίσεις. Εάν προβάλετε την τοποθεσία του Windows Update (η οποία είναι διαθέσιμη σε πολλές γλώσσες), ο Internet Explorer θα κάνει αναζήτηση στη λίστα με τις γλώσσες στις ρυθμίσεις προτίμησης γλώσσας για να προσδιορίσει ποια γλώσσα θα εμφανίζεται. Αυτή είναι μια λίστα προτίμησης, επομένως αν τα ελληνικά είναι η πρώτη γλώσσα και τα αγγλικά η δεύτερη, το Windows Update θα εμφανίζεται στα ελληνικά.

Για να εμφανίσετε την τοποθεσία του Windows Update σε μια άλλη γλώσσα, μπορείτε να προσαρμόσετε τις ρυθμίσεις προτίμησης γλώσσας του Internet Explorer όπως περιγράφεται παρακάτω. Σημειώστε ότι οι αλλαγές στις ρυθμίσεις αυτές θα επηρεάσουν και τις άλλες τοποθεσίες πολλαπλών γλωσσών που προβάλλετε.

  1. Στον Internet Explorer, κάντε κλικ στο μενού Εργαλεία.
  2. Κάντε κλικ στην εντολή Επιλογές Internet.
  3. Στην καρτέλα Γενικά, κάντε κλικ στο κουμπί Γλώσσες.
  4. Επιλέξτε τη γλώσσα που θέλετε και χρησιμοποιήστε το κουμπί Μετακίνηση επάνω για να τοποθετήσετε την επιλογή σας στην αρχή της λίστας με τις γλώσσες. Κάντε κλικ στο κουμπί "OK".
  5. Κάντε κλικ στο κουμπί "OK" στο παράθυρο διαλόγου "Επιλογές Internet" και ξεκινήστε πάλι το πρόγραμμα περιήγησης.
  6. Κάντε πάλι φόρτωση της σελίδας του Windows Update.

Why does the Windows Update site appear in a different language?

The Windows Update site detects several settings to determine the language in which the site will be displayed and the language of the components available for download.

The first detection is the language and version of your operating system and browser. Components available for download from Windows Update will always be the correct language and version for your operating system, regardless of the language in which you are viewing the site.

Windows Update also detects your operating system's Regional Settings to determine the language in which the site will be displayed. As an example, if you live in Greece, but have an English operating system, you would see the site in Greek, but the downloads available would be English.

Internet Explorer also has a feature called Accept Language which supercedes the Regional Settings. If you are viewing Windows Update (which is available in multiple languages), Internet Explorer will look to the list of languages in your Language Preference settings to determine which language to display. This is a prioritized list, so if you have Greek as the first language and English as the second, Windows Update will display in Greek.

To see the Windows Update site in a different language, you can adjust your Internet Explorer Language Preference settings as described below. Please note that changes here will affect other multi-language sites that you view.

  1. In Internet Explorer, click the Tools menu.
  2. Click Internet Options.
  3. On the General tab, click the Languages button.
  4. Select the language you want, then use the Move Up button to place your selection at the top of the list of languages. Click "OK."
  5. Click "OK" in the Internet Options dialog and restart your browser.
  6. Reload the Windows Update page.
