Πληροφορίες σχετικά με το Windows Update
Μόνο για διαχειριστές τεχνολογίας πληροφορικής σε εταιρείες

Εάν αναζητάτε στοιχεία προς λήψη και κρίσιμες ενημερωμένες εκδόσεις με δυνατότητα διανομής για το εταιρικό σας δίκτυο, ανατρέξτε στην τοποθεσία Windows Update Corporate Site σήμερα!

Είναι δυνατό να καταργήσετε τα εικονίδια του Windows Update που εμφανίζονται στο μενού "Έναρξη". Η απενεργοποίηση των εικονιδίων του Windows Update εμποδίζει τους χρήστες να έχουν πρόσβαση στην τοποθεσία του Windows Update (http://windowsupdate.microsoft.com) από οπουδήποτε στα Windows 98.

Για να καταργήσετε τα εικονίδια του Windows Update από το μενού "Έναρξη":

  1. Καταργήστε το εικονίδιο από την ομάδα "Έναρξη/Ρυθμίσεις", δημιουργώντας το κλειδί μητρώου HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer με την τιμή: NoWindowsUpdate = 1 (Αυτή είναι μια τιμή DWORD. Η ρύθμιση με τιμή 0 ενεργοποιεί τα εικονίδια του Windows Update.)
  2. Επανεκκίνηση του υπολογιστή.
  3. Για να καταργήσετε το εικονίδιο από την επάνω ομάδα της γραμμής Έναρξης, εκτελέστε τα εξής από τη γραμμή εντολών: Wupdmgr.exe -shortcut.
Σημείωση Δεν θα πρέπει να επεξεργαστείτε το μητρώο, εκτός εάν είναι απολύτως αναγκαίο. Εάν υπάρχει σφάλμα στο μητρώο σας, ο υπολογιστής σας ίσως να μην λειτουργεί σωστά.
